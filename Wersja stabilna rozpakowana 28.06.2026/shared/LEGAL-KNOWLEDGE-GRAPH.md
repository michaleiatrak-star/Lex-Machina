
# LEGAL-KNOWLEDGE-GRAPH — prawo polskie

Cel: wymusić powiązanie aktu prawnego z procedurą, dowodem, ryzykiem i strategią.

## Relacje obowiązkowe

Dla każdego modułu utwórz w analizie roboczej mapę:

- akt prawny → organ właściwy;
- akt prawny → tryb postępowania;
- akt prawny → termin;
- akt prawny → dowód konieczny;
- akt prawny → typowy zarzut;
- akt prawny → typowy kontrzarzut organu/przeciwnika;
- akt prawny → środek zaskarżenia;
- akt prawny → skutki wykonawcze;
- akt prawny → przepisy przejściowe;
- akt prawny → wymagany status ISAP.

## Konflikt dziedzin

Jeżeli sprawa dotyczy kilku gałęzi prawa, router musi:

1. ustalić procedurę dominującą;
2. wskazać procedury zależne;
3. rozpoznać prejudycjalność;
4. oznaczyć ryzyko kolizji terminów;
5. skierować całość do `prawo-polskie-v2`, a dopiero stamtąd do modułów szczegółowych.
