## J.9 UMOWA O DZIEŁO

**Podstawa: KC art. 627–646**
```
isap.sejm.gov.pl → KC → art. 627–646
UWAGA: ZUS może kwestionować umowy o dzieło i przekwalifikowywać na zlecenie
  → Weryfikuj wyłącznie oficjalnie: zus.pl oraz ISAP; orzeczenia: sn.pl/orzeczenia.ms.gov.pl
```

### Pułapki:

**DZ-1 — Brak definicji "dzieła" = ryzyko braku rezultatu (HIGH RISK)**
```
UMOWA O DZIEŁO = umowa rezultatu (a nie starannego działania jak zlecenie)
PUŁAPKA: "Wykonanie usług doradczych" → nie jest dziełem (brak konkretnego rezultatu)
  → ZUS może zakwestionować i naliczyć składki jak przy zleceniu
PUŁAPKA 2: "Dzieło" musi być obiektywnie weryfikowalne — co konkretnie dostarcza wykonawca?

REKOMENDACJA:
  → Precyzuj rezultat: "projekt graficzny w formacie [X] obejmujący [elementy]"
  → Wskaż kryteria oceny czy dzieło jest wykonane prawidłowo
  → Wskaż protokół odbioru z konkretnymi przesłankami przyjęcia
```

**DZ-2 — Rękojmia za wady dzieła bez limitu (HIGH RISK)**
```
PRAWO: KC art. 637 (uchylony) → nowe: art. 638 KC odsyła do rękojmi przy sprzedaży
  Rękojmia: 2 lata od wydania dzieła (art. 568 KC stosowany odpowiednio)
  Strony w B2B mogą ograniczyć lub wyłączyć rękojmię (art. 558 §1 KC)

REKOMENDACJA DLA WYKONAWCY:
  „§X. Strony ograniczają odpowiedzialność z tytułu rękojmi za wady Dzieła
  do [6/12] miesięcy od daty odbioru. Wykonawca nie odpowiada za wady
  wynikające z użycia materiałów lub wytycznych dostarczonych przez Zamawiającego."
```

**DZ-3 — Wynagrodzenie ryczałtowe a zmiany zakresu (HIGH RISK)**
```
PRAWO: KC art. 632 → wynagrodzenie ryczałtowe = stałe, bez podwyżki za dodatkową pracę
  Wyjątek: "rażąca strata" (art. 632 §2 KC) → sąd może podwyższyć (rzadko stosowane)

PUŁAPKA: Klient zmienia zakres w trakcie ("scope creep") → wykonawca realizuje bez dodatkowej zapłaty

REKOMENDACJA:
  „§X. Wynagrodzenie ryczałtowe obejmuje wyłącznie zakres określony w Specyfikacji
  stanowiącej Załącznik nr [1]. Każda zmiana zakresu wymaga pisemnego aneksu
  określającego dodatkowe wynagrodzenie i zmieniony termin wykonania.
  Realizacja przez Wykonawcę prac nieobjętych Specyfikacją bez pisemnego aneksu
  uprawnia go do wynagrodzenia dodatkowego wg stawek wskazanych w Cenniku."
```

## J.10 UMOWA ZLECENIA

**Podstawa: KC art. 734–751**

### Pułapki:

**ZL-1 — Brak minimalnej stawki godzinowej (CRITICAL)**
```
PRAWO: ustawa o minimalnym wynagrodzeniu za pracę → minimalna stawka godzinowa zlecenia
  → WERYFIKUJ aktualne stawki wyłącznie urzędowo: ISAP + gov.pl/rodzina lub właściwe rozporządzenie w Dzienniku Ustaw
  → Obowiązek ewidencji czasu pracy (w umowie lub oddzielnie)
  → Penalizacja: zleceniodawca odpowiada za wypłatę stawki minimalnej

PUŁAPKA: "Wynagrodzenie ryczałtowe [X] zł miesięcznie" przy pracy 200h/m-c
  → Stawka godzinowa może wyjść poniżej minimum → naruszenie prawa
```

**ZL-2 — Wypowiedzenie bez ważnego powodu (HIGH RISK)**
```
PRAWO: KC art. 746:
  → Dający zlecenie może wypowiedzieć ZAWSZE, nawet bez powodu
  → ALE: musi zwrócić zleceniobiorcy wydatki + zapłacić za wykonaną część
  → Jeśli wypowiada bez ważnego powodu → naprawia szkodę

PUŁAPKA: Umowa "na rok" → zleceniodawca wypowiada po miesiącu (legalne!)
  → Zleceniobiorca traci planowane wynagrodzenie za 11 miesięcy
  → Realna szkoda: utracone kontrakty, nieskute kontrakty z osobami trzecimi

REKOMENDACJA:
  „§X. Zleceniodawca może wypowiedzieć Umowę z ważnego powodu ze skutkiem
  natychmiastowym lub bez ważnego powodu z zachowaniem [3]-miesięcznego
  okresu wypowiedzenia. W przypadku wypowiedzenia bez ważnego powodu
  Zleceniodawca zapłaci Zleceniobiorcy odszkodowanie równe [X]-miesięcznemu
  wynagrodzeniu ryczałtowemu."
```

## J.11 UGODA I POROZUMIENIE

**Podstawa: KC art. 917–919**

### Pułapki:

**UG-1 — Blanketowe zrzeczenie roszczeń (CRITICAL)**
```
PROBLEM: "Strony potwierdzają, że wszelkie roszczenia między nimi zostały
  uregulowane i nie mają do siebie wzajemnych pretensji"
  → Strona może nie wiedzieć, że ma roszczenie (np. roszczenie z RODO / mobbing)
  → Sąd może uznać ugodę za skuteczną nawet dla nieznanych roszczeń

OCHRONA:
  → Wylistuj konkretnie roszczenia objęte ugodą
  → Dodaj klauzulę: "Ugoda nie obejmuje roszczeń związanych z [katalog wyłączeń]"
  → Przy wątpliwościach: ugoda sądowa > ugoda pozasądowa (sąd bada świadomość stron)

PAMIĘTAJ: Zrzeczenie wynagrodzenia pracowniczego → art. 84 KP → nieważne
  Nawet w ugodzie pracownik nie może się zrzec wypłaconego wynagrodzenia
```

**UG-2 — Ugoda zawarta pod wpływem błędu lub przymusu (HIGH RISK)**
```
PRAWO: KC art. 918 → uchylenie się od ugody:
  Możliwe gdy: strona była w błędzie co do okoliczności faktycznych,
  które ugoda miała regulować, i błąd był istotny
  UWAGA: Błąd co do prawa NIE jest podstawą uchylenia (weryfikuj aktualnie: sn.pl)

PUŁAPKA: "Rozwiązanie sporu" gdy jedna strona nie znała kluczowych faktów
  (np. nieznane faktury, ukryte wady, fałszywe dokumenty)
  → Uchylenie możliwe w ciągu roku od wykrycia błędu (art. 88 KC)
```
