## FAZA 0 — INTAKE
```
□ Rola: wykonawca / zamawiający / podwykonawca?
□ Jakiego rodzaju zamówienie? (roboty budowlane / dostawy / usługi / społeczne)
□ Wartość i próg:
  → Poniżej **170 000 zł** → brak PZP (zasady wewnętrzne zamawiającego)
    (próg podniesiony z 130 000 zł do 170 000 zł od 01.01.2026 — art. 2 ust. 1 pkt 1 PZP)
  → 170 000 zł – progi UE → PZP (tryby krajowe)
  → Powyżej progów UE → PZP (tryby unijne + KIO pełne)
  Progi UE 2026–2027 (M.P. 2025.1247 — obwieszczenie UZP):
    Roboty budowlane: 5 404 000 EUR = 23 291 240 zł
    Dostawy/usługi adm. centralna: 140 000 EUR = 603 400 zł
    Dostawy/usługi pozostali: 216 000 EUR = 930 960 zł
    Usługi społeczne: 750 000 EUR = 3 232 500 zł
□ Jaka czynność zamawiającego jest kwestionowana?
  → treść SWZ / wykluczenie / odrzucenie oferty / wybór oferty najkorzystniejszej
□ KIEDY DOKŁADNIE wykonawca otrzymał informację? (data + godzina → wyznacza termin)
□ Czy złożono odwołanie do KIO? Jaki wynik?
□ Czy wpis jest uiszczony?
□ Czy chodzi o zmianę umowy, podwykonawstwo, zabezpieczenie, kary umowne?
```

## MAPA TRYBÓW UDZIELANIA ZAMÓWIEŃ
```
TRYBY PODSTAWOWE:
  Przetarg nieograniczony (art. 132 PZP) — otwarty, wszyscy mogą złożyć ofertę
  Przetarg ograniczony (art. 150 PZP) — tylko zaproszeni (kwalifikacja wstępna)
  Tryb podstawowy (art. 275–296 PZP) — poniżej progów UE, 3 warianty

TRYBY NEGOCJACYJNE:
  Negocjacje z ogłoszeniem (art. 152 PZP)
  Negocjacje bez ogłoszenia (art. 214 ust. 1 pkt 4–5 PZP) — wyjątkowe przesłanki
  Zamówienie z wolnej ręki (art. 214 PZP) — katalog zamknięty: weryfikuj przesłanki!

UPROSZCZONE:
  Tryb podstawowy wariant I (art. 275 pkt 1 PZP) — oferty bez negocjacji
  Tryb podstawowy wariant II (art. 275 pkt 2 PZP) — negocjacje przed wyborem
  Tryb podstawowy wariant III (art. 275 pkt 3 PZP) — negocjacje obligatoryjne
  Partnerstwo innowacyjne (art. 297 PZP)

DYNAMICZNY SYSTEM ZAKUPÓW (DSZ) / UMOWY RAMOWE:
  → Wielokrotne udzielanie zamówień bez pełnego postępowania
```

## WYKLUCZENIE I ODRZUCENIE — OBRONA WYKONAWCY

### Podstawy wykluczenia (art. 108–109 PZP)
```
OBLIGATORYJNE (art. 108 PZP) — zamawiający nie może odstąpić:
  → Skazanie za przestępstwa z listy (korupcja, pranie pieniędzy, terroryzm, etc.)
  → Karalność za przestępstwo podatkowe
  → Naruszenia obowiązków środowiskowych / społecznych / pracowniczych (potwierdzone)
  → Konflikty interesów (art. 108 ust. 1 pkt 5)
  → Porozumienia cenowe / zmowy przetargowe

FAKULTATYWNE (art. 109 PZP) — zamawiający MUSI wskazać w SWZ:
  → Naruszenie obowiązków podatkowych
  → Rażące błędy zawodowe
  → Rozwiązanie umowy z zamówieniem z winy wykonawcy
  → Wprowadzenie zamawiającego w błąd

SELF-CLEANING (art. 110 PZP):
  → Wykonawca może wykazać rzetelność mimo przesłanki wykluczenia
  → Wymaga: naprawienia szkody + współpraca z organami + środki zapobiegawcze
  → Zamawiający ocenia indywidualnie — decyzja uzasadniona
```

### Odrzucenie oferty (art. 226 PZP)
```
NAJCZĘSTSZE PRZESŁANKI:
  → Niezgodność z warunkami zamówienia (art. 226 ust. 1 pkt 5) — najczęstsza
  → Rażąco niska cena (art. 224–225 PZP) — procedura wyjaśnień obowiązkowa
  → Błędy w obliczeniu ceny (pkt 10)
  → Brak RODO-klauzuli (pkt 13) — kontrowersyjne; weryfikuj orzecznictwo KIO
  → Niezgodność z przepisami prawa

RAŻĄCO NISKA CENA — OBRONA:
  → Zamawiający MUSI wezwać do wyjaśnień przed odrzuceniem
  → Wyjaśnienia: oszczędności metody, koszty pracy, pomoc publiczna, innowacje
  → Ciężar dowodu: na wykonawcy (wyjaśnienia muszą być konkretne, nie ogólnikowe)
```

## ŚRODKI OCHRONY PRAWNEJ

### Odwołanie do KIO (art. 513–578 PZP)
```
ZAKRES: każda czynność lub zaniechanie zamawiającego sprzeczne z PZP
  → Powyżej progów UE: pełny zakres
  → Poniżej progów UE: ograniczony katalog (art. 513 PZP) — weryfikuj

FORMA ODWOŁANIA (obligatoryjna treść — art. 516 PZP):
  □ Oznaczenie zamawiającego
  □ Numer ogłoszenia (jeśli dotyczy)
  □ Zarzuty — konkretne, z powołaniem przepisów
  □ Żądanie — precyzyjne (uchylenie, nakazanie czynności)
  □ Uzasadnienie faktyczne i prawne
  □ Dowody — ZŁOŻONE ŁĄCZNIE Z ODWOŁANIEM lub w trakcie

PLATFORM ZAKUPOWA: odwołanie przez platformę e-Zamówienia (ezamowienia.gov.pl)
  → Wymagana kwalifikowana podpis elektroniczny lub EPUAP

ROZPRAWY ZDALNE: od 13.03.2026 możliwe (Dz.U.2025.769) — złóż wniosek
```

### Skarga do Sądu Okręgowego (art. 579–590 PZP)
```
TERMIN: 14 dni od doręczenia orzeczenia KIO (art. 580 §1 PZP)
OPŁATA: 5-krotność wpisu od odwołania — weryfikuj aktualnie
ZAKRES: kontrola orzeczenia KIO — nie ponowne rozpoznanie co do meritum
UWAGA: wąski zakres zaskarżalności — SO bada tylko naruszenia prawa przez KIO
```

## ZMIANA UMOWY — KLUCZOWE REGUŁY
```
ZAKAZ ZMIAN ISTOTNYCH (art. 454 PZP):
  → Zmiana istotna = bezprawna — nowe zamówienie bez przetargu
  → Istotna gdy: zmienia charakter / zakres / wartość / równowagę ekonomiczną

DOPUSZCZALNE ZMIANY (art. 455 PZP) — katalog:
  → Przewidziane w klauzuli przeglądowej w SWZ
  → Nowy wykonawca (sukcesja, restrukturyzacja, gwarancja)
  → Wartość do 15% (dostawy/usługi) lub 15% (roboty) + do 50% w wyjątkowych przypadkach
  → Zmiana nieistotna (test: czy zmiana wpłynęłaby na krąg oferentów lub wynik)
  → Okoliczności nieprzewidywalne (COVID, konflikt zbrojny, gwałtowny wzrost cen)
```
