# ZUS i Ubezpieczenia Społeczne — Szczegółowy Framework

> ⚠️ **WERYFIKACJA ONLINE OBOWIĄZKOWA** — przepisy ZUS zmieniają się często.
> Reforma orzecznictwa rozłożona na etapy 2026–2027 — weryfikuj aktualny stan.

## ZASADA ABSOLUTNA — NIE MA WSA W SPRAWACH ZUS
Decyzje ZUS/KRUS → **Sąd Okręgowy** (wydział pracy i ubezpieczeń społecznych) lub SR.
Tryb wg KPC (art. 477⁸–477¹⁴), NIE tryb PPSA (nie WSA!).

## SCHEMAT POSTĘPOWANIA
```
Decyzja ZUS
    ↓ [autokontrola 30 dni od złożenia odwołania — ZUS może sam zmienić decyzję]
    ↓ [1 miesiąc] ODWOŁANIE DO SĄDU — składane za pośrednictwem ZUS
Emerytura / Renta z FUS       → Sąd OKRĘGOWY (wydział pracy i ubezpieczeń)
Zasiłki / Świadczenie rehab.  → Sąd REJONOWY
Składki < 100 tys. zł         → SR | ≥ 100 tys. → SO
    ↓ [14 dni od doręczenia wyroku z uzasadnieniem] APELACJA (30 zł)
    ↓ [30 dni od wyroku SA] SKARGA KASACYJNA do SN (przymus adwokacki)
```

## SPRZECIW OD ORZECZENIA LEKARSKIEGO — OBOWIĄZKOWY
W sprawach rentowych:
```
Orzeczenie lekarza orzecznika ZUS / pielęgniarki / fizjoterapeuty
    ↓ [14 dni] SPRZECIW DO ZUS (tego samego oddziału)
    ↓ Ponowne rozpatrzenie przez INNEGO lekarza orzecznika (od 13.04.2026 — jednoosobowo)
       lub przez 3 lekarzy (tylko sprawy szczególnie skomplikowane)
    ↓ Decyzja ZUS
    ↓ [1 miesiąc] Odwołanie do sądu
```
⚠️ **POMINIĘCIE SPRZECIWU = sąd ODRZUCI odwołanie (art. 477¹⁴a KPC).**

Pułapka: Orzeczenie pozornie korzystne (stwierdzono niezdolność, ale błędna DATA) →
złóż sprzeciw kwestionując datę powstania niezdolności!

### REFORMA ORZECZNICTWA — AKTUALNY STAN (ustawa z 18.12.2025, Dz.U. 2026 poz. 26)
```
ETAP I  (od 27.01.2026) — JUŻ OBOWIĄZUJE:
  → Nowe definicje pracy zarobkowej i aktywności niezgodnej z celem L4
  → Kontrolerzy ZUS z rozszerzonymi uprawnieniami

ETAP II (od 13.04.2026) — JUŻ OBOWIĄZUJE:
  → Pielęgniarki i fizjoterapeuci mogą wydawać orzeczenia w wybranych sprawach
  → Lekarze bez specjalizacji lub z 5 lat. stażem mogą być orzecznikami
  → Badania orzecznicze możliwe zdalnie (wideorozmowa)
  → Maks. 30 dni na wydanie orzeczenia (przekroczenie → prawo ponaglenia)
  → Sprzeciw od orzeczenia pielęgniarki/fizjoterapeuty → do lekarza orzecznika

ETAP III (od 01.01.2027) — PLANOWANY, JESZCZE NIE OBOWIĄZUJE:
  → Pełne przejście na jednoosobowe orzekanie I i II instancji
  → Komisje lekarskie zastąpione jednym lekarzem (z wyjątkiem spraw szczeg. trudnych)
  → Orzeczenia w formie elektronicznej
  → Maks. 30 dni na orzeczenie — obowiązuje od tego etapu formalnie
```
**Procedura sprzeciwu nadal istnieje — sprzeciw składasz w terminie 14 dni.**
Weryfikuj aktualną procedurę: isap.sejm.gov.pl / zus.pl przed każdą analizą.

## EMERYTURA — WZÓR OBLICZENIA (nowy system, urodzeni od 1949)
```
Emerytura = Podstawa obliczenia / Średnie dalsze trwanie życia (tablice GUS)

Podstawa = Zwaloryzowane składki na koncie ZUS
          + Zwaloryzowany kapitał początkowy (za lata przed 1999 r.)
          + Środki przetransferowane z OFE (po likwidacji / przekształceniu)
```

## TYPOWE PRZYCZYNY ZANIŻONEJ EMERYTURY
```
□ Błędny kapitał początkowy (brak zaświadczeń Rp-7, pominięte okresy)
□ Pominięte okresy składkowe / nieskładkowe (urlopy wychowawcze, wojsko)
□ Niekorzystna tablica GUS — moment złożenia wniosku ma znaczenie
□ Praca w szczególnych warunkach lub o szczególnym charakterze — nieuwzględniona
□ Błąd czerwcowy 2009–2019 (patrz niżej — ustawa naprawcza od 01.01.2026)
□ Wyrok TK SK 140/20 z 4.06.2024 — wcześniejsza emerytura przed 6.06.2012
```

## WYROK TK z 4.06.2024 (SK 140/20) — PRZELICZENIE EMERYTUR
```
Co orzeczono: art. 25 ust. 1b ustawy FUS niezgodny z Konstytucją
              w zakresie, w jakim dotyczy osób, które złożyły wniosek
              o wcześniejszą emeryturę PRZED 6 CZERWCA 2012 r.

Kogo dotyczy: kobiety ur. 1949–1959 (bez rocznika 1953*) i
              mężczyźni ur. 1949–1954 (bez rocznika 1953*)
              którzy pobierali wcześniejszą emeryturę, a następnie
              przeszli na emeryturę powszechną po 01.01.2013
              (* rocznik 1953 objęty odrębną ustawą z 2020 r.)

Problem: ZUS odmawiał przeliczenia, bo wyrok TK NIE ZOSTAŁ OPUBLIKOWANY
         w Dz.U. (wyroki TK od lat nie są publikowane przez MJ)

Stan na maj 2026: sądy masowo nakazują ZUS przeliczyć emerytury
  → jeśli ZUS odmówi → odwołanie do sądu okręgowego → sąd nakazuje przeliczenie
  → brak specustawy wykonawczej do wyroku TK (rząd jej nie uchwalił)
  → tryb: wniosek o wznowienie postępowania (art. 190 ust. 4 Konstytucji)
           lub odwołanie od decyzji odmownej ZUS do SO
```

## EMERYTURY CZERWCOWE 2009–2019 — PRZELICZONE Z URZĘDU
```
Ustawa z 5.08.2025 (Dz.U. 2025 poz. 1169) — weszła w życie 01.01.2026
Zakres: emerytury ustalone w czerwcu w latach 2009–2019 (ok. 219 tys.)
        + powiązane renty rodzinne

ZUS przeliczył z URZĘDU — nie trzeba składać wniosku
Termin przeliczenia: do 31.03.2026 (ZUS zakończył w marcu 2026)
Skutek: emerytura obliczona jak w maju z kwartalną waloryzacją — korzystniejszy wariant
Nowe świadczenie przysługuje od 01.01.2026 + wyrównanie za 3 miesiące
Średnia podwyżka: ok. 163 zł/mies.; nie można było obniżyć świadczenia

⚠️ Jeśli sprawa jest w sądzie: postępowanie zostało zawieszone do czasu decyzji ZUS
   (art. 4 ustawy z 5.08.2025)
```

## RENTA Z TYTUŁU NIEZDOLNOŚCI DO PRACY (art. 57 ustawy FUS) — TRZY PRZESŁANKI
```
1. Całkowita lub częściowa niezdolność do pracy (orzeczenie)
2. Wymagany staż ubezpieczeniowy:
   - do 20 r.ż.: 1 rok
   - 20–22 r.ż.: 2 lata
   - 22–25 r.ż.: 3 lata
   - 25–30 r.ż.: 4 lata
   - powyżej 30 r.ż.: 5 lat (w ostatnim 10-leciu przed wnioskiem lub niezdolnością)
3. Niezdolność powstała w OKRESIE SKŁADKOWYM / NIESKŁADKOWYM lub ≤ 18 m-cy po nim
   ⚠️ Wyjątek: jeśli ubezpieczony ma 25 lat stażu — warunek pkt 3 odpada
```

## FAZA 0 — INTAKE
```
□ Rodzaj decyzji ZUS: emerytura / renta / zasiłek / składki / kapitał początkowy?
□ Sprawa rentowa: czy złożono sprzeciw od orzeczenia w 14 dni?
   → Pominięcie = sąd ODRZUCI odwołanie!
□ Termin 1 miesiąca na odwołanie — upłynął?
□ Sąd właściwy: SO (emerytura/renta) czy SR (zasiłki)?
□ Czy decyzja ZUS zawiera pouczenie? → brak = termin odwołania nie biegnie!
□ Czy sprawa dotyczy emerytury czerwcowej (2009–2019)? → ustawa z 5.08.2025
□ Czy sprawa dotyczy wcześniejszej emerytury przed 6.06.2012? → wyrok TK SK 140/20
□ Data złożenia orzeczenia: przed czy po 13.04.2026?
   (sprzeciw od orzeczeń pielęgniarek/fizjoterapeutów dopiero od 13.04.2026)
□ Reforma 2027: od 01.01.2027 komisje lekarskie zastąpione jednoosobowym orzekaniem
```
