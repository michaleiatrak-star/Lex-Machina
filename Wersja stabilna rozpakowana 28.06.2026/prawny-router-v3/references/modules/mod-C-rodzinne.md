## ZASADA ABSOLUTNA — TRYBY POSTĘPOWANIA

```
PROCESOWY (powód vs pozwany):
  Rozwód / Separacja     → Sąd OKRĘGOWY (ostatnie wspólne miejsce zamieszkania)
  Alimenty               → Sąd Rejonowy (wybór: miejsce zamieszkania pozwanego
                           LUB uprawnionego — art. 32 KPC)
  Ustalenie ojcostwa     → Sąd Rejonowy
  Zaprzeczenie ojcostwa  → Sąd Rejonowy

NIEPROCESOWY (wnioskodawca + uczestnicy):
  Władza rodzicielska    → Sąd Rejonowy (sąd opiekuńczy)
  Kontakty z dzieckiem   → Sąd Rejonowy (sąd opiekuńczy)
  Podział majątku        → Sąd Rejonowy (właściwy wg miejsca położenia majątku)
  Ubezwłasnowolnienie    → Sąd OKRĘGOWY
```

**Błędny tryb = odrzucenie pisma bez merytorycznego rozpoznania.**
**Zasada nadrzędna: dobro dziecka** (art. 56 §2 KRO, art. 95 KRO).
