# Prawo Ubezpieczeniowe — Szczegółowy Framework

## TERMINY — SPRAWDŹ JAKO PIERWSZE
```
Przedawnienie roszczeń z umowy ubezp.:  3 lata (art. 819 §1 KC)
  → Bieg przerywa: zgłoszenie szkody do ubezpieczyciela
  → Po przerwaniu: biegnie na nowo od daty odmowy / zakończenia postępowania likwidacyjnego
Termin na wypłatę odszkodowania:       30 dni od zgłoszenia szkody (art. 14 ustawy ub. obow.)
  → Gdy wyjaśnienie niemożliwe w 30 dniach: 14 dni od wyjaśnienia, max 90 dni
Odwołanie do Rzecznika Finansowego:    brak terminu zawitego, ale im szybciej tym lepiej
Pozew cywilny:                         3 lata przedawnienia (art. 819 KC)
```

## FAZA 0 — INTAKE
```
□ Jaki rodzaj ubezpieczenia? (OC / AC / życiowe / majątkowe / NNW / odpow. cyw.)
□ Co ubezpieczyciel zrobił?
  → odmowa / zaniżenie odszkodowania / opóźnienie / wyłączenie z OWU
□ Kiedy nastąpiła szkoda i kiedy zgłoszono? (ocena przedawnienia)
□ Jaka kwota sporu?
□ Czy było odwołanie wewnętrzne i jaka odpowiedź? (termin odpowiedzi: 30 dni)
□ Czy kontaktowano się z Rzecznikiem Finansowym?
□ Czy polisa / OWU zawiera klauzule wyłączające (alkohol, umyślność, rażące niedbalstwo)?
□ Czy sprawca wypadku / szkody jest znany i czy jest ubezpieczony?
```

## MAPA POSTĘPOWAŃ
```
1. Zgłoszenie szkody do ubezpieczyciela (art. 14 ustawy ub. obow.)
   → Obowiązek likwidacji: 30 dni od zgłoszenia
   → Upływ 30 dni bez odpowiedzi = opóźnienie → odsetki ustawowe za opóźnienie

2. Odwołanie wewnętrzne do ubezpieczyciela
   → Obowiązkowe przed wnioskiem do RF w sporach z grupą I (konsumenci)
   → Termin odpowiedzi zakładu: 30 dni (złożone sprawy: 60 dni)
     (art. 6 ustawy o rozpatrywaniu reklamacji przez podmioty rynku finansowego — weryfikuj t.j. w isap)

3. Interwencja Rzecznika Finansowego (rzecznikfinansowy.gov.pl)
   → Bezpłatna — postępowanie interwencyjne lub pozasądowe rozwiązywanie sporów
   → Rekomendacja RF nie wiąże sądu, ale ma walor dowodowy
   → Podmiot rynku finansowego jest ZWIĄZANY rekomendacją RF
     (może odmówić — ale musi uzasadnić: ryzyko dodatkowych kosztów)

4. Pozew cywilny (SO lub SR zależnie od WPS)
   → Roszczenie bezpośrednie poszkodowanego do ubezpieczyciela sprawcy (OC)
   → Podstawa: art. 822 §4 KC (actio directa)
   → Możliwe łączenie: ubezpieczyciel + sprawca jako pozwani solidarni
```

## OC KOMUNIKACYJNE — SPECYFIKA
```
SUMA GWARANCYJNA (weryfikuj aktualnie — zależy od dyrektyw UE):
  → OC komunikacyjne: min. kwota za szkodę na osobie i na mieniu
  → Weryfikuj aktualną wysokość: gov.pl / art. 36 ustawy ub. obow.

ACTIO DIRECTA (art. 822 §4 KC):
  → Poszkodowany może pozwać ubezpieczyciela sprawcy bezpośrednio
  → Nawet bez pozywania sprawcy

REGRES (art. 43 ustawy ub. obow.):
  → Ubezpieczyciel który wypłacił może żądać zwrotu od sprawcy gdy:
    □ Prowadził pod wpływem alkoholu/narkotyków
    □ Wszedł w posiadanie pojazdu wskutek przestępstwa
    □ Zbiegł z miejsca zdarzenia
    □ Nie miał wymaganego uprawnienia do prowadzenia pojazdu

UFG (ubezpieczyciel nieznany lub bez OC):
  → Ubezpieczeniowy Fundusz Gwarancyjny zastępuje ubezpieczyciela sprawcy
  → Wniosek do UFG: ufg.pl
```

## NAJCZĘSTSZE WYŁĄCZENIA W OWU — WERYFIKACJA
```
⚠️ ZAWSZE czytaj OWU łącznie z polisą — wyłączenia mogą być abuzywne

Wyłączenia typowe (AC):
  □ Kradzież bez śladów włamania — kwestionuj: SN dopuszcza odszkodowanie
  □ Rażące niedbalstwo — wąska interpretacja; zwykłe niedbalstwo ≠ rażące
  □ Stan techniczny pojazdu — ciężar dowodu na ubezpieczycielu
  □ Brak aktualnego przeglądu — weryfikuj: związek przyczynowy ze szkodą?

Klauzule potencjalnie abuzywne (sprawdź: rejestr.uokik.gov.pl):
  → Postanowienia ograniczające zakres ochrony w sposób nieoczekiwany
  → Definicje szkody całkowitej dyskryminujące wobec standardów rynkowych
```

## SZKODA CAŁKOWITA — OBLICZANIE
```
Szkoda całkowita gdy: koszty naprawy > wartość pojazdu przed szkodą

METODA RÓŻNICOWA:
  Odszkodowanie = wartość rynkowa pojazdu PRZED szkodą − wartość pozostałości (wraku)

UWAGA: zakład może zaniżać:
  □ Wartość pojazdu przed szkodą — weryfikuj: info-ekspert.pl / eurotax / inne
  □ Odejmować koszty naprawy części nie uszkodzonych
  → Opinia rzeczoznawcy lub biegłego sądowego — kluczowy dowód
```

## ODPOWIEDZIALNOŚĆ CYWILNA — ZBIEG Z UBEZPIECZENIEM
```
ZASADA ZBIEGÓW (art. 824¹ KC):
  → Ubezpieczenie OC nie może prowadzić do bezpodstawnego wzbogacenia
  → Suma odszkodowań z różnych ubezpieczeń ≤ szkoda rzeczywista
  → Wyjątek: ubezpieczenie na życie / NNW — bez zasady zbiegów
```

## PREDYKCJA WYNIKU — SZABLON
```

         rażące zaniżenie wartości pojazdu (>20% vs wycena)
```
