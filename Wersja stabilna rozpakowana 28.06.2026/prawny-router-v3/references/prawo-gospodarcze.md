# Prawo Gospodarcze — Szczegółowy Framework

## TERMINY KRYTYCZNE — SPRAWDŹ JAKO PIERWSZE
```
Wniosek o upadłość:     30 dni od zaistnienia podstawy (art. 21 PrUp)
  → Brak = odpowiedzialność osobista członka zarządu (art. 299 KSH)
Zaskarżenie uchwały ZW: 1 miesiąc od powzięcia lub 6 m-cy od powzięcia
  (art. 252 KSH dla sp. z o.o. / art. 422 KSH dla S.A.)
Odpowiedzialność zarządu (art. 299): 3 lata od dnia, gdy wierzyciel dowiedział się
  albo przy zachowaniu należytej staranności mógł się dowiedzieć o bezskuteczności
  egzekucji wobec spółki (art. 442¹ §1 KC — delikt) — nie wcześniej niż od daty
  umorzenia egzekucji; max. 10 lat od zdarzenia wywołującego szkodę; WERYFIKUJ ONLINE
```

## ODPOWIEDZIALNOŚĆ ZARZĄDU (art. 299 KSH) — MECHANIZM
```
Wierzyciel bezskutecznie egzekwował od spółki
    ↓
Pozew przeciw członkowi zarządu (każdemu solidarnie)
    ↓
Członek zarządu może się uwolnić TYLKO gdy wykaże:
  □ Złożył wniosek o upadłość we właściwym czasie (30 dni)
  LUB □ Wierzyciel nie poniósł szkody mimo niezłożenia wniosku
  LUB □ Niezłożenie wniosku nastąpiło bez jego winy
```

## UMOWY B2B — NAJWAŻNIEJSZE ELEMENTY
```
□ Strony: NIP, REGON, KRS / CEIDG — zawsze weryfikuj w rejestrze
□ Przedmiot: precyzyjny opis + parametry jakościowe
□ Wynagrodzenie: stałe / zmienne + VAT / netto
□ Kara umowna: tylko za zobowiązania NIEPIENIĘŻNE (art. 483 KC)
□ Zakaz konkurencji: ograniczony terytorialnie + zakres + ekwiwalent
□ Poufność (NDA): czas trwania + zakres + sankcja
□ Prawo właściwe + sąd właściwy (klauzula jurysdykcyjna)
□ Rozwiązanie: tryb + wypowiedzenie + skutki
```

## POSTĘPOWANIE RESTRUKTURYZACYJNE — 4 TRYBY
```
1. Postępowanie o zatwierdzenie układu (PZU):
   → Dłużnik > 15% wierzytelności głosuje → najszybsze, bez sądu
2. Przyspieszone postępowanie układowe (PPU):
   → Sporne wierzytelności < 15% → sąd zatwierdza układ
3. Postępowanie układowe:
   → Sporne wierzytelności > 15% → pełna ochrona sądowa
4. Postępowanie sanacyjne:
   → Głęboka restrukturyzacja, zarządca przejmuje majątek
```

## UPADŁOŚĆ LIKWIDACYJNA — ETAPY
```
Wniosek o upadłość (dłużnik lub wierzyciel)
    ↓ Postanowienie sądu o ogłoszeniu upadłości
    ↓ Syndyk przejmuje zarząd masą
    ↓ Spis wierzytelności → lista wierzytelności
    ↓ Likwidacja majątku → plan podziału
    ↓ Zakończenie postępowania → umorzenie długów (FUZ)
```

## PROKURA
Art. 109¹ KC: prokura = pełnomocnictwo handlowe, udzielana przez zarząd.\
Prokura samoistna vs łączna. Rejestr: KRS (obowiązkowy wpis).\
Prokura NIE obejmuje: zbycia przedsiębiorstwa, nieruchomości, obciążenia — bez pełnomocnictwa.
