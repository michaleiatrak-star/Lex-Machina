# BRAKI-FORMALNE — wykrywanie i kwalifikacja braków pisma

## 1. Klasyfikacja braków

| Poziom | Znaczenie | Reakcja |
|---|---|---|
| KRYTYCZNY | może skutkować zwrotem, odrzuceniem lub bezskutecznością | nie oddawaj pisma jako finalnego |
| ISTOTNY | osłabia skuteczność, ale zwykle da się uzupełnić | oznacz i popraw |
| TECHNICZNY | format, numeracja, załączniki, redakcja | popraw przed finalizacją |

## 2. Braki krytyczne

- brak oznaczenia sądu / organu,
- brak oznaczenia strony wnoszącej,
- brak żądania,
- brak podpisu,
- brak opłaty, jeżeli jest warunkiem nadania biegu,
- brak pełnomocnictwa przy działaniu przez pełnomocnika,
- brak odpisów, jeżeli są wymagane,
- brak wartości przedmiotu sporu / zaskarżenia, jeżeli wymagana,
- wniesienie do niewłaściwego organu bez mechanizmu przekazania,
- uchybienie terminowi zawitemu,
- brak wskazania zaskarżonego orzeczenia w środku zaskarżenia,
- brak zakresu zaskarżenia albo wniosków odwoławczych, jeżeli wymagane.

## 3. Braki istotne

- nieprecyzyjne żądanie,
- brak powiązania dowodu z faktem,
- brak chronologii,
- brak wskazania interesu prawnego,
- brak legitymacji czynnej albo biernej w uzasadnieniu,
- sprzeczności między żądaniem, uzasadnieniem i dowodami,
- powołanie niezweryfikowanego przepisu,
- powołanie niezweryfikowanego orzeczenia.

## 4. Format raportu

```text
BRAKI FORMALNE
Krytyczne: ...
Istotne: ...
Techniczne: ...
Czy pismo można złożyć: TAK / NIE / TYLKO PO UZUPEŁNIENIU
Rekomendowana poprawka: ...
```
