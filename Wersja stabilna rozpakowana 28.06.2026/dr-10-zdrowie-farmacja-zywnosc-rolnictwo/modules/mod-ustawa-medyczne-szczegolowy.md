# Prawo Medyczne — Szczegółowy Framework

## TERMINY — SPRAWDŹ JAKO PIERWSZE
```
Przedawnienie roszczenia o błąd medyczny (art. 4421 KC):
  → 3 lata od dowiedzenia się o szkodzie i osobie zobowiązanej
  → MAX 20 lat od zdarzenia (dla szkód na osobie — art. 4421 §3 KC)
  → Małoletni: 3 lata od uzyskania pełnoletności (ostrożnie — weryfikuj)
Wniosek do FKZM (Fundusz Kompensacyjny Zdarzeń Medycznych):
  → 1 rok od dowiedzenia się o zdarzeniu (art. 67t ust. 8 u.p.p.)
  → MAX 3 lata od zdarzenia medycznego
Odwołanie od decyzji NFZ (odmowa świadczenia):
  → 14 dni (KPA) lub 30 dni (PPSA) — zależnie od trybu
Skarga do RPP:                   brak terminu zawitego
Skarga do sądu lekarskiego:      brak terminu zawitego (ale wpływa na dowody)
```

## FAZA 0 — INTAKE
```
□ Co konkretnie nastąpiło?
  → błąd diagnostyczny / terapeutyczny / operacyjny / pielęgnacyjny / anestezjologiczny
  → zakażenie szpitalne / upadek / pominięcie leku / złe dawkowanie
  → brak zgody na zabieg / naruszenie praw pacjenta
  → odmowa leczenia / odmowa wydania dokumentacji
□ Gdzie nastąpiło zdarzenie?
  → szpital publiczny (NFZ) / prywatna klinika / NZOZ / lekarz indywidualnie
□ Kiedy nastąpiło zdarzenie? (ocena terminów)
□ Jaka jest dokumentacja medyczna?
  → historia choroby / wyniki badań / epikryzy / karty zleceń
□ Jaka jest szkoda?
  → trwały uszczerbek na zdrowiu / śmierć / dodatkowe leczenie / ból i cierpienie
□ Jaki efekt oczekuje klient?
  → odszkodowanie / zadośćuczynienie / stwierdzenie błędu / skargi na lekarza
□ Czy złożono już jakiekolwiek pisma lub skargi?
```

## ŚCIEŻKI DOCHODZENIA ROSZCZEŃ

### A. FUNDUSZ KOMPENSACYJNY ZDARZEŃ MEDYCZNYCH (FKZM)
```
ZAKRES: Zdarzenia medyczne w szpitalach (hospitalizations) od 06.09.2023
  → Zakażenie pacjenta biologicznym czynnikiem chorobotwórczym
  → Uszkodzenie ciała lub rozstrój zdrowia pacjenta
  → Śmierć pacjenta — roszczenia bliskich

PROGI KWOTOWE (weryfikuj aktualnie w RPP — waloryzacja coroczna):
  → Uszkodzenie ciała / rozstrój zdrowia: do 200 000 zł
  → Śmierć: do 100 000 zł (na rzecz każdego uprawnionego)

PROCEDURA:
  Wniosek do Prezesa RPP → Komisja RPP (90 dni) → propozycja świadczenia
  → AKCEPTACJA: wypłata (zrzeczenie się dalszych roszczeń!)
  → ODMOWA lub ODRZUCENIE: wniosek o ponowne rozpatrzenie (30 dni) → WSA

⚠️ PRZYJĘCIE ŚWIADCZENIA Z FKZM = ZAMKNIĘCIE ŚCIEŻKI SĄDOWEJ
  Rozważ dokładnie przed akceptacją — nie łącz z pozwem cywilnym
```

### B. POZEW CYWILNY
```
PODSTAWY PRAWNE:
  → Art. 415 KC (czyn niedozwolony — odpow. lekarza/personelu)
  → Art. 430 KC (odpow. szpitala za pracownika — zwierzchnik)
  → Art. 444 KC (odszkodowanie za szkodę na osobie)
  → Art. 445 KC (zadośćuczynienie za doznaną krzywdę)
  → Art. 446 KC (roszczenia po śmierci pacjenta — bliscy)
  → Art. 471 KC (nienależyte wykonanie umowy — odpow. kontraktowa)

CIĘŻAR DOWODU:
  → Poszkodowany: szkoda + zdarzenie + związek przyczynowy
  → Domniemanie winy w organizacji (szpital): można zastosować w pewnych przypadkach
  → KLUCZOWY DOWÓD: opinia biegłego sądowego (obligatoryjny w sprawach med.)

SĄD WŁAŚCIWY:
  → SR: do 100 000 zł (próg od 01.07.2023 na mocy Dz.U. 2023 poz. 1114 zm. KPC art. 17 pkt 4)
     → SO: powyżej 100 000 zł — weryfikuj aktualnie art. 17 KPC
  → SO: powyżej + sprawy o zadośćuczynienie dot. naruszenia dóbr osobistych
  → Właściwość: siedziba pozwanego LUB miejsce zdarzenia (wybór powoda)
```

### C. SKARGA DO RZECZNIKA PRAW PACJENTA (RPP)
```
ZAKRES RPP:
  → Naruszenie prawa do świadczeń zdrowotnych
  → Naruszenie prawa do informacji (art. 9 u.p.p.)
  → Naruszenie prawa do dokumentacji medycznej (art. 23 u.p.p.)
  → Naruszenie prawa do poszanowania intymności i godności
  → Naruszenie tajemnicy lekarskiej

PROCEDURA: bezpłatna; pisemna skarga → RPP bada → wystąpienie do kierownika placówki
  → RPP może wydać stanowisko; nie przyznaje odszkodowania
  → Kara finansowa (art. 108 u.p.p.) do 500 000 zł dla placówki przy ciężkim naruszeniu
```

### D. SĄD LEKARSKI (odpowiedzialność zawodowa)
```
IZBY LEKARSKIE: Okręgowy Sąd Lekarski → Naczelny Sąd Lekarski
KARY: upomnienie / nagana / zawieszenie prawa wykonywania zawodu / pozbawienie
PODSTAWA: naruszenie zasad etyki lekarskiej lub przepisów o wykonywaniu zawodu
NIE: nie przyznaje odszkodowania; niezależne od postępowania cywilnego/karnego
```

## PRAWA PACJENTA — KATALOG (u.p.p. Dz.U. 2024.581 t.j.)
| Prawo | Podstawa | Jak egzekwować |
|---|---|---|
| Do świadczeń zdrowotnych | art. 6–8 u.p.p. | NFZ / RPP / skarga do NFZ |
| Do informacji o stanie zdrowia | art. 9 u.p.p. | Żądanie pisemne; jeśli odmowa → RPP |
| Do dokumentacji medycznej | art. 23–29 u.p.p. | 7 dni na wydanie kopii; brak → RPP |
| Do tajemnicy lekarskiej | art. 13 u.p.p. | Skarga do RPP / sąd cywilny |
| Do poszanowania godności i intymności | art. 20 u.p.p. | Skarga do RPP |
| Do wyrażenia zgody na zabieg | art. 15–19 u.p.p. | Brak zgody = bezprawność zabiegu |
| Do leczenia bólu | art. 20a u.p.p. | Odmowa → skarga RPP |

## UZYSKANIE DOKUMENTACJI MEDYCZNEJ
```
PRAWO DO KOPII (art. 27 u.p.p.):
  → Szpital/klinika: obowiązek wydania w ciągu 7 dni od wniosku
  → Forma: kopia papierowa (opłata do 0,002% MWPS) / elektroniczna (bezpłatna)
  → Odmowa lub opóźnienie → skarga do RPP

PRAWO DO WGLĄDU (art. 23 u.p.p.):
  → Na miejscu, bez kopiowania — obowiązek niezwłoczny
  → Reprezentacja przez pełnomocnika (pisemne pełnomocnictwo)
```

## OPINIA BIEGŁEGO — STRATEGIA
```
BIEGŁY SĄDOWY (art. 278–291 KPC):
  → Obowiązkowy w sprawach błędów medycznych
  → Sąd powołuje — ale strona może wnosić o konkretnego eksperta
  → Biegły z listy: sądy okręgowe (biegły-lekarz.pl — nieoficjalne, weryfikuj)

PRYWATNA OPINIA MEDYCZNA:
  → Dokument prywatny (art. 245 KPC) — nie zastępuje biegłego sądowego
  → Wartość: jako podstawa oceny szans przed złożeniem pozwu
  → Koszt: 2 000–10 000 zł zależnie od specjalizacji

INSTYTUT EKSPERTYZ SĄDOWYCH (Kraków):
  → Opinie na zlecenie sądu i stron — wyspecjalizowany
```

## PREDYKCJA WYNIKU — SZABLON
```
Szanse: [0–100%]
IN PLUS: udokumentowana przyczyna szkody w dokumentacji (adnotacje operacyjne),
         pozytywna opinia prywatna, zakażenie szpitalne (domniemanie winy placówki),
         brak zgody na zabieg (bezprawność → odpow. niezależna od błędu),
         trwały uszczerbek > 20%, śmierć pacjenta
IN MINUS: wieloczynnikowa etiologia (trudny związek przyczynowy), brak dokumentacji
          operacyjnej, upływ 3 lat od dowiedzenia się o szkodzie, sprzeczne opinie

BENCHMARKING:
  → Wywołaj orzeczenia-sadowe-v2 z frazą:
    "błąd medyczny art. 445 KC zadośćuczynienie SN"
    "zgoda na zabieg art. 19 ustawa o prawach pacjenta SN"
  ⚠️ NIEZWERYFIKOWANE — wyszukaj URL: sn.pl / orzeczenia.ms.gov.pl

REKOMENDACJA: □ FKZM (do 200 000 zł, szybciej)  □ Pozew cywilny (bez limitu)
              □ Oba — UWAGA: FKZM zamyka ścieżkę sądową
```
