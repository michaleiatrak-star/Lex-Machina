# MIN8 UPGRADE — chronologia źródłowa i wykrywanie sprzeczności

## Cel modernizacji
Każde zdarzenie w osi czasu musi być powiązane ze źródłem, statusem pewności i skutkiem procesowym.

## Format zdarzenia
- data/godzina,
- opis faktu,
- źródło,
- status: potwierdzone / prawdopodobne / sporne / brak źródła,
- znaczenie prawne,
- powiązane dowody,
- sprzeczności,
- ryzyko procesowe.

## Wykrywanie sprzeczności
Skill musi oznaczać:
- sprzeczne daty,
- luki czasowe,
- zdarzenia bez źródła,
- nielogiczne następstwo przyczynowo-skutkowe,
- brak dowodu dla zdarzenia kluczowego.

## Wynik ekspercki
Chronologia nie jest tylko listą dat. Musi wskazywać, które zdarzenia są prawnie decydujące.
