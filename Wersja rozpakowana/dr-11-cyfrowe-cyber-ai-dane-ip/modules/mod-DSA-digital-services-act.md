# Modul - DSA - Digital Services Act (Akt o Uslugach Cyfrowych)

## Akt prawny
**Rozporządzenie (UE) 2022/2065 z 19.10.2022 r. o jednolitym rynku uslug cyfrowych (DSA)**
- Dz.Urz. UE L 277/1 z 27.10.2022
  VER: eur-lex.europa.eu/legal-content/PL/TXT/?uri=CELEX:32022R2065 [2026-05-31]
- Stosowanie: od **17.02.2024** (dla wszystkich podmiotow)
- Koordynator uslug cyfrowych (DSC): Polska - wyznaczenie w toku (UKE lub nowy organ)
  web_search: "DSA Polska koordynator uslug cyfrowych DSC wyznaczenie 2025"

**Weryfikacja eur-lex przed kazdy cytowaniem.**

## Adresaci i kategorie
```
Posrednicy (hosting, cache, access): Podstawowe zasady "safe harbour"
Platformy internetowe:               Powiadomienie o nielegalnych tresciach + usuniecie
Duze platformy (VLOP):               > 45 mln uzytkownikow/mies. w UE - dodatkowe obowiazki
  - Ocena ryzyka systemowego (co roku)
  - Audyt niezalezny
  - Dostep do danych dla badaczy
Bardzo duze wyszukiwarki (VLOSE):    Analogicznie

Nadzor nad VLOP/VLOSE: KE (Komisja Europejska) - bezposrednio
```

## Kluczowe obowiazki
```
Notice-and-action: System zglaszan nielegalnych tresci + szybka reakcja
Przejrzystosc reklam: Oznaczenie reklam + baza reklam (VLOP)
Zakaz targetowania: Dzieci, dane wrazliwe - zakaz targetowanej reklamy
Mechanizm skargi:   Wewnetrzny + zewnetrzny do DSC
Srodki przymusu:    Do 6% globalnego obrotu (VLOP niezastosowanie sie do KE)
```
