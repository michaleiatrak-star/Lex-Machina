# Modul - MiCA - Kryptoaktywa (Rozp. UE 2023/1114)

## Akt prawny
**Rozporządzenie Parlamentu Europejskiego i Rady (UE) 2023/1114 z 31.05.2023 r. w sprawie rynkow kryptoaktywow (MiCA)**
- Opublikowane: Dz.Urz. UE L 2023/1114 (09.06.2023)
  VER: eur-lex.europa.eu/legal-content/PL/TXT/?uri=CELEX:32023R1114 [2026-05-31]
- Stosowanie: w pelni od **30.12.2024** (ART, EMT od 30.06.2024)
- Organ nadzoru: KNF (Komisja Nadzoru Finansowego)

**Weryfikacja eur-lex i KNF przed kazdy cytowaniem.**

## Zakres
MiCA = pierwsze kompleksowe prawo UE regulujace rynek kryptoaktywow.
Obejmuje: emitentow kryptoaktywow, dostawcow uslug kryptoaktywow (CASP).

## Typy kryptoaktywow wedlug MiCA
```
ARTs (Asset-Referenced Tokens):  Powiazane z koszykiem aktywow (np. walut, towarow)
EMTs (E-Money Tokens):           Powiazane z jedna waluta fiat (np. EUR stablecoin)
Inne kryptoaktywa:               Bitcoin, Ether, altcoiny (nie ART/EMT)
Wylaczenia:                      NFT jako unikalne (oceniac indywidualnie), DeFi (monitoring)
```

## CASP (Crypto-Asset Service Provider)
```
Wymagane zezwolenie: Od KNF (lub innego organu w UE - paszport regulacyjny)
Uslugi:              Handel, przechowywanie, wymiana, doradztwo, transfer
Kapital minimalny:   Zalezny od uslugi (12 500 - 150 000 EUR)
Ochrona klientow:    Segregacja aktywow, polityka refundacji, whitepaper
```

## Whitepaper (Biala Ksiega)
```
Obowiazkowa:     Przy publicznej ofercie kryptoaktywow (poza wylaczeniami)
Zawartosc:       Opis projektu, ryzyko, prawa inwestorow
Organ:           Powiadomienie organu (nie zatwierdzenie dla innych niz ART/EMT)
Odpowiedzialnosc: Emitent za dokladnosc whitepaper
```
