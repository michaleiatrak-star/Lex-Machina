# Modul - Otwarte dane i ponowne wykorzystywanie informacji sektora publicznego

## Akt prawny
**Ustawa z 11.08.2021 r. o otwartych danych i ponownym wykorzystywaniu informacji sektora publicznego**
- Dz.U. 2023 poz. 1524 t.j. (tekst jednolity aktu z Dz.U. 2021 poz. 1641)
  VER: isap.sejm.gov.pl [2026-06-14, TRYB DZU]
- Implementacja Dyrektywy UE 2019/1024 (Open Data Directive)

**Weryfikacja ISAP przed kazdy cytowaniem.**

## Zakres
Obowiazek udostepniania danych publicznych do ponownego wykorzystywania.
Regula domyslna: dane publiczne = otwarte (nie zamkniete).

## Katalog danych o wysokiej wartosci
```
Dyrektywa UE + Rozp. Wykonawcze KE 2023/138 okreslaja 6 tematycznych kategorii:
  1. Dane geoprzestrzenne
  2. Dane o obserwacji Ziemi i srodowisku
  3. Dane meteorologiczne
  4. Dane statystyczne
  5. Dane o spolkach i ich wlascicielach
  6. Dane o mobilnosci
Obowiazek: Udostepnienie bezplatnie przez API przez organy publiczne
```

## Ponowne wykorzystywanie informacji sektora publicznego
```
Definicja:   Uzywanie przez podmioty prywatne danych zebranych przez podmioty publ.
Warunki:     Podmiot publiczny nie moze ograniczac (wyj.: prywatnosc, tajemnica, IP)
Oplaty:      Dozwolone w wyjatkowych wypadkach (koszt marginalny) lub wg rozporzadzenia
Skargi:      Do organu wydajacego decyzje odmowna + skarga do WSA
```

## Portal danych
```
dane.gov.pl - portal danych publicznych Polska
Obowiazek:  Organy publiczne maja obowiazek rejestracji zbiorow danych
Licencja:   CC0 lub CC-BY 4.0 jako standard dla danych publicznych
```
