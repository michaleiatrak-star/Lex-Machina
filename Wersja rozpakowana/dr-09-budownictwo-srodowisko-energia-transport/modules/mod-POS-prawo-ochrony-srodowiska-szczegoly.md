# Moduł — Prawo Ochrony Środowiska: Szczegółowy Framework

**Zakres:** Szczegółowy workflow kanceryjski dla spraw środowiskowych: intake, OOŚ
obligatoryjna/screening, Natura 2000, kary administracyjne WIOŚ, odpowiedzialność
za szkody w środowisku, predykcja wyniku. Uzupełnia `mod-POS-prawo-ochrony-srodowiska.md`.

## FAZA 0 — INTAKE

```
□ Kto jest stroną?
  → inwestor / właściciel gruntu / sąsiad / organizacja ekologiczna / organ
□ Jaki problem / co się dzieje?
  → uzyskanie DŚU / odwołanie od DŚU / naruszenie warunków decyzji środowiskowej
  → kara za naruszenie przepisów środowiskowych (WIOŚ)
  → odpowiedzialność karna za przestępstwo przeciwko środowisku (KK 181–188)
  → odpady / emisje do powietrza / hałas / wyciek substancji
  → obszar Natura 2000 / park narodowy / rezerwat
□ Jaki organ prowadzi / prowadził postępowanie?
  → Wójt/Burmistrz/Prezydent (DŚU dla mniej skomplikowanych)
  → RDOŚ (DŚU złożone / Natura 2000)
  → Starosta (pozwolenia na emisje, pozwolenia wodnoprawne)
  → GDOŚ (II inst. od RDOŚ)
  → WIOŚ (kontrola i kary administracyjne)
□ Czy jest już decyzja i jaka data doręczenia? (termin odwołania!)
□ Czy inwestycja jest już w toku / zakończona?
```

## DECYZJA O ŚRODOWISKOWYCH UWARUNKOWANIACH (DŚU) — SZCZEGÓŁY

### Kiedy wymagana

```
OBLIGATORYJNA (art. 71 u.u.i.ś.) dla:
  → Przedsięwzięcia mogące zawsze znacząco oddziaływać na środowisko (I grupa)
     — lista w rozporządzeniu RM: elektrownie, autostrady, duże instalacje
  → Przedsięwzięcia mogące potencjalnie znacząco oddziaływać (II grupa)
     — lista w rozporządzeniu RM: mniejsze instalacje, inwestycje komunalne

ORGAN WŁAŚCIWY:
  → Wójt / Burmistrz / Prezydent (I i II grupa — zasada)
  → RDOŚ (I grupa + obszary Natura 2000 + wymagające OOŚ — wspólne orzekanie)
  → Starosta (wybrane inwestycje gospodarcze — weryfikuj rozporządzenie)
```

### Ocena oddziaływania na środowisko (OOŚ)

```
OBLIGATORYJNA OOŚ (I grupa):
  → Raport o oddziaływaniu na środowisko (zawartość: art. 66 u.u.i.ś.)
  → Udział społeczeństwa: ogłoszenie + 21 dni na uwagi (art. 33–36 u.u.i.ś.)
  → Uzgodnienia: RDOŚ + PPIS (sanepid) + inne organy

SCREENING (II grupa — organ decyduje czy OOŚ konieczna):
  → Wniosek + karta informacyjna przedsięwzięcia (KIP)
  → Postanowienie stwierdzające obowiązek / brak obowiązku OOŚ
  → Zaskarżalne zażaleniem (14 dni → organ wyższego stopnia)

ORGANIZACJA EKOLOGICZNA (art. 44 u.u.i.ś.):
  → Może być stroną postępowania jeśli zgłosiła udział w terminie
  → Może wnosić odwołanie, skargę do WSA (interes w ochronie środowiska)
```

## POZWOLENIA ŚRODOWISKOWE — MAPA

| Rodzaj | Organ | Podstawa | Uwagi |
|---|---|---|---|
| Pozwolenie zintegrowane (IPPC) | Marszałek / RDOŚ | Prawo ochrony środowiska | Dla instalacji IPPC — weryfikuj listę |
| Pozwolenie na emisje do powietrza | Starosta | POŚ art. 180 | Wymagane powyżej progów emisji |
| Pozwolenie wodnoprawne | Wody Polskie (PGW) | Prawo wodne | Pobór wody, odprowadzanie ścieków |
| Zezwolenie na zbieranie odpadów | Marszałek / Starosta | Ustawa o odpadach | Prowadzenie działalności odpadowej |
| Zezwolenie na przetwarzanie odpadów | Marszałek / Starosta | Ustawa o odpadach | Instalacje przetwarzające |

⚠️ WERYFIKUJ WŁAŚCIWOŚĆ ORGANU dla każdego rodzaju decyzji — zmiany 2024/2025.

## KARY ADMINISTRACYJNE — WIOŚ

### Administracyjne kary pieniężne (POŚ art. 298–318 — Dz.U.2025.647)

```
NARUSZENIA KARALNE ADMINISTRACYJNIE (m.in.):
  → Przekroczenie dopuszczalnych emisji (hałas, powietrze)
  → Naruszenie warunków pozwoleń środowiskowych
  → Działalność bez wymaganego pozwolenia

WŁAŚCIWOŚĆ: WIOŚ — wydaje decyzję o karze
ODWOŁANIE: SKO → WSA → NSA (tryb KPA)

WYMIAR KARY: proporcjonalny do:
  → Rodzaju naruszenia, czasu trwania, zasięgu szkody
  → Korzyści osiągniętej przez naruszającego
  → Działań naprawczych podjętych dobrowolnie

⚠️ Weryfikuj aktualne stawki kar w ISAP przed powołaniem.
```

## NATURA 2000 — SPECYFIKA

```
OBSZARY SPECJALNEJ OCHRONY PTAKÓW (OSO) i SPECJALNEJ OCHRONY SIEDLISK (SOO):
  → DŚU dla inwestycji na obszarach Natura 2000 lub w sąsiedztwie: zawsze wymagana OOŚ
  → Organ: RDOŚ (dla I grupy) lub uzgodnienie RDOŚ (dla II grupy)
  → Zakaz pogarszania stanu siedlisk i gatunków chronionych (art. 33 ustawy o ochr. przyrody
    Dz.U. 2026 poz. 13 t.j.)

DEROGACJE (wyjątkowe zezwolenia na odstępstwo):
  → RDOŚ może zezwolić gdy: nadrzędny interes publiczny + brak alternatyw + kompensacja
  → Decyzja RDOŚ → podlega kontroli WSA

⚠️ Weryfikuj ustawa o ochronie przyrody: Dz.U. 2026 poz. 13 t.j. z 08.01.2026 w ISAP.
```

## ODPOWIEDZIALNOŚĆ ZA SZKODĘ W ŚRODOWISKU

```
USTAWA z 13.04.2007 r. o zapobieganiu szkodom w środowisku:
  → Odpowiedzialność prowadzącego działalność za szkodę w środowisku
  → ORGAN: RDOŚ (działania naprawcze) / Starosta (zabezpieczenie)
  → Podmiot może być zobowiązany do przywrócenia stanu poprzedniego
  → Jeśli nie: organ działania naprawcze na koszt sprawcy
  → Zasada "zanieczyszczający płaci" — odpowiedzialność niezależna od winy

⚠️ Brak nowszego t.j. ustawy o zapobieganiu szkodom — weryfikuj aktualny stan w ISAP.
```
