## SKRÓT OPERACYJNY (najczęstsze terminy — ZAWSZE zweryfikuj w źródle)

| Czynność | Termin | Liczony od |
|---|---|---|
| Sprzeciw od nakazu (zwykły/EPU) | **14 dni** | doręczenia nakazu |
| Zarzuty od nakazu nakazowego | **7 dni** | doręczenia nakazu |
| Wniosek o uzasadnienie wyroku | **7 dni** | ogłoszenia wyroku |
| Sprzeciw od orzeczenia referendarza | **7 dni** | doręczenia orzeczenia |
| Przywrócenie terminu | **7 dni** | ustania przeszkody |
| Zażalenie | **7 dni** | doręczenia postanowienia |

## REGUŁY LICZENIA (art. 165 KPC)

- Jeśli ostatni dzień = sobota, niedziela lub dzień ustawowo wolny → termin przesuwa się na najbliższy dzień roboczy.
- Data złożenia = data nadania listem poleconym lub złożenia w biurze podawczym sądu.

## ALERT TERMINOWY

- Jeśli ≤ 3 dni: 🚨 **PILNE — działaj natychmiast**
- Jeśli termin minął: ⛔ **TERMIN UPŁYNĄŁ** — zaproponuj wniosek o przywrócenie (art. 168 KPC)
