# MOD-DOKTRYNA — polityka cytowania doktryny i komentarzy w pismach procesowych

> Wersja: 1.0.0 | Typ: moduł redakcyjny | shared/
> Wywoływany z: pisma-procesowe-v3 W2 (sekcja uzasadnienia prawnego)
>               MOD-ORZE (W3.2) gdy obok orzeczenia potrzebna doktryna
> Podstawa ekspercka: Garner *Legal Writing in Plain English* (substantive footnotes);
> Scalia/Garner *Making Your Case* §37 (footnotes debate);
> polska tradycja: Gniewek, Gudowski, Jaśkowska — standard cytowania komentarzy SN

---

## 1. Zasada naczelna (Garner vs Scalia)

Garner: **doktryna w przypisie**, nie w tekście — przerywanie argumentacji
wtrąconą opinią autora osłabia tempo narracji procesowej.

Scalia: **doktryna w tekście dopuszczalna** dla nieprymarnych autorytetów —
gdy opinia autora jest jedynym lub najsilniejszym źródłem dla tezy.

**Przyjęty standard dla polskiego pisma procesowego:**

```
REGUŁA D-1 (orzecznictwo ma pierwszeństwo):
  Gdy tezę potwierdza orzeczenie SN/SA → cytuj orzeczenie w tekście.
  Doktrynę wtedy pomiń lub wstaw w przypis jako "zob. też."

REGUŁA D-2 (doktryna jako wsparcie):
  Gdy brak orzecznictwa lub linia jest rozbieżna → doktrynę w tekście,
  ale po orzeczeniu. Format: "Pogląd ten podziela doktryna: [autor, tytuł, str.]"

REGUŁA D-3 (doktryna w przypisie):
  Gdy doktryna jest dodatkowym wzmocnieniem, nie głównym argumentem →
  przypis numerowany na końcu zdania lub akapitu.

REGUŁA D-4 (zakaz doktryny zamiast prawa):
  NIGDY nie cytuj komentarza zamiast przepisu. Komentarz interpretuje prawo
  — przepis jest źródłem. Format błędny: "Zgodnie z Gniewkiem art. X KC..."
  Format poprawny: "Zgodnie z art. X KC (Dz.U. ... poz. ...) [...]; zob. Gniewek..."
```

## 2. Hierarchia autorytetów doktrynalnych w Polsce

```
POZIOM 1 — Komentarze systemowe do KPC / KC / KP (najwyższy autorytet):
  KPC:  Gudowski (red.), Komentarz do KPC, C.H. Beck / LexisNexis
        Manowska (red.), Komentarz do KPC, Wolters Kluwer
        Feliga (red.), Komentarz do KPC, C.H. Beck
  KC:   Gniewek/Machnikowski (red.), Komentarz do KC, C.H. Beck
        Osajda (red.), Komentarz KC, C.H. Beck (Legalis)
  KP:   Jaśkowska/Sobczyk (red.), Komentarz do KP, Wolters Kluwer
        Góral, Komentarz do KP, LexisNexis

POZIOM 2 — Monografie (szczegółowa analiza instytucji):
  Cytuj gdy brak komentarza lub komentarz nie omawia kwestii szczegółowej.
  Wskazany rok wydania i strony: [Autor, *Tytuł*, wyd. X, Wydawnictwo rok, s. XX]

POZIOM 3 — Artykuły doktrynalne (PiP, KPP, PS, Palestra, MoP):
  Stosuj ostrożnie — niższy autorytet niż komentarze systemowe.
  Zawsze podaj: [Autor, "Tytuł artykułu", Palestra/PiP nr X/rok, s. XX]

POZIOM 4 — Materiały konferencyjne / opracowania niepublikowane:
  ⛔ ZAKAZ cytowania w piśmie procesowym składanym do sądu.
```

## 3. Format cytowania w tekście pisma

### 3a. Komentarz w tekście (D-2)

```
"Pogląd ten jest utrwalony w doktrynie. Jak wskazuje Gniewek [red.], Kodeks
cywilny. Komentarz, wyd. 10, C.H. Beck 2023, art. [X], Nb [Y]: »[cytat
dosłowny lub parafraza]«."
```

### 3b. Komentarz w przypisie (D-3)

```
Tekst główny:
"Roszczenie odszkodowawcze oparte na art. [X] KC jest wymagalne z chwilą
wyrządzenia szkody.¹"

Przypis 1:
"¹ Tak Gniewek/Machnikowski (red.), KC. Komentarz, wyd. 10, C.H. Beck 2023,
art. [X], Nb [Y]; zob. też wyrok SN z [data], sygn. [nr]."
```

### 3c. Monografia w przypisie

```
"¹ T. Ereciński, Apelacja w postępowaniu cywilnym, wyd. 4, Wolters Kluwer
2019, s. 145–148."
```

## 4. Polityka przypisów rzeczowych (substantive footnotes)

```
DOZWOLONE w przypisach rzeczowych:
  ✓ cytaty doktrynalne (D-3)
  ✓ orzeczenia pomocnicze (nie główna linia — te w tekście)
  ✓ odesłanie do powiązanej regulacji UE / prawa obcego
  ✓ zastrzeżenie intertemporalne ("w brzmieniu na datę zdarzenia...")

ZAKAZANE w przypisach rzeczowych (Garner — interrupt flow):
  ⛔ rozbudowana analiza prawna (powinna być w tekście lub usunięta)
  ⛔ polemika z doktryną (o ile nie jest to pismo do SN/SA)
  ⛔ cytaty z pism procesowych z innych spraw
  ⛔ prywatne opinie i nieformalne ekspertyzy
```

## 5. Zasady liczby cytowań

```
POZEW / odpowiedź na pozew (I instancja):
  Komentarze: ≤3 cytowania (pismo nie powinno wyglądać jak artykuł naukowy)
  Monografie: ≤1 cytowanie
  Artykuły:   ≤1 cytowanie (tylko gdy brak komentarza)

APELACJA / zażalenie (II instancja):
  Komentarze: ≤5 cytowań
  Monografie: ≤2 cytowania

SKARGA KASACYJNA (SN):
  Komentarze: bez limitu, ale każde musi być precyzyjne
  Monografie: bez limitu
  Artykuły:   ≤3 cytowania z wiodących periodyków (PiP, KPP)

Zasada Garnera: jeden celny cytat lepszy niż pięć marginalnych.
```

---

## 6. HARDGATE doktryny

```
⛔ HARDGATE DOK-1: ZAKAZ cytowania doktryny z pamięci.
  Każdy cytat = podaj: autor, tytuł, wydawnictwo, rok, artykuł/strony.
  Brak danych bibliograficznych → ⬛ [UZUPEŁNIJ: dane źródła] — nigdy parafrazuj
  "jak mówi doktryna" bez wskazania konkretnego źródła.

⛔ HARDGATE DOK-2: ZAKAZ cytowania komentarza zamiast przepisu.
  Komentarz wspiera przepis, nie zastępuje go.

⛔ HARDGATE DOK-3: ZAKAZ cytowania doktryny przeciwko obowiązującemu przepisowi.
  "Doktryna krytykuje art. X KC" → NIEDOPUSZCZALNE jako argument procesowy.
  Podstawą jest przepis — doktryna może być przywołana wyłącznie interpretacyjnie.
```
