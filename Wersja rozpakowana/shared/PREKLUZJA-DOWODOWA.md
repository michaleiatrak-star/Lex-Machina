# PREKLUZJA-DOWODOWA — kontrola spóźnionych twierdzeń i dowodów

## 1. Kiedy używać

Wczytaj zawsze przy:

- odpowiedzi na pozew,
- piśmie przygotowawczym,
- sprawie gospodarczej,
- postępowaniu z planem rozprawy,
- apelacji z nowymi dowodami,
- wniosku dowodowym po wcześniejszym etapie sprawy.

## 2. Kontrola podstawowa

Dla każdego dowodu ustal:

| Pytanie | Odpowiedź |
|---|---|
| Jaki fakt ma wykazać? | ... |
| Czy fakt jest istotny dla rozstrzygnięcia? | ... |
| Kiedy strona dowiedziała się o dowodzie? | ... |
| Dlaczego nie powołano go wcześniej? | ... |
| Czy sąd może uznać go za spóźniony? | ... |
| Czy zachodzi potrzeba uprawdopodobnienia? | ... |

## 3. Typowe ryzyka

- dowód powołany dopiero po odpowiedzi na pozew,
- dowód bez tezy dowodowej,
- dowód na fakt bez znaczenia prawnego,
- dowód możliwy do powołania wcześniej,
- dowód z dokumentu znanego stronie od początku,
- próba obejścia koncentracji materiału procesowego.

## 4. Raport

```text
PREKLUZJA
Dowód: ...
Fakt: ...
Moment powołania: ...
Ryzyko pominięcia: niskie / średnie / wysokie
Uzasadnienie dopuszczenia: ...
Formuła wniosku: ...
```
