## 2. INTAKE

```
□ Jaki problem: ochrona działacza ZZ / układ zbiorowy / spór zbiorowy / strajk?
□ Czy ZOZ jest reprezentatywna (≥ 8% ogółu zatrudnionych — art. 24125d KP)?
□ Czy doszło do zwolnienia działacza ZZ bez zgody zarządu ZOZ (art. 32 ustawy)?
□ Czy pracodawca przeprowadził konsultację ZZ przy zwolnieniu?
□ Czy spór dotyczy zbiorowych interesów pracowniczych (nie indywidualnych)?
□ Na jakim etapie jest spór zbiorowy: rokowania / mediacja / arbitraż / strajk?
```

## 3. PROCEDURA

### Ochrona działacza związkowego

```
Zakaz wypowiedzenia / rozwiązania umowy z chronionym działaczem (art. 32 ustawy o ZZ):
  → Wymagana UPRZEDNIA ZGODA zarządu ZOZ
  → Brak zgody → zwolnienie bezskuteczne → roszczenie o przywrócenie
  → Ochrona: przez czas pełnienia funkcji + rok po zakończeniu

Ochrona przeciwko dyskryminacji (art. 35 §1 KP):
  → Zakaz nierównego traktowania za przynależność lub działalność związkową
```

### Kto może być członkiem ZZ (po nowelizacji 2019)

```
Nie tylko pracownicy:
  □ Pracownicy (art. 2 ustawy o ZZ)
  □ Osoby wykonujące pracę zarobkową (zleceniobiorcy, samozatrudnieni)
    gdy mają prawo koalicji (od nowelizacji 2019)
  ⚠️ Weryfikuj aktualne przepisy o przynależności w ISAP.
```

### Układy zbiorowe pracy (nowa ustawa Dz.U. 2025 poz. 1661)

```
Strony: Pracodawca / organizacja pracodawców + ZOZ / federacja ZZ
Forma: Pisemna
Rejestracja: Okręgowy Inspektor Pracy (zakładowe) / MRiPS (ponadzakładowe)
Zasada uprzywilejowania: UZP stosuje się zamiast KP — korzystniej niż KP

⚠️ Dla szczegółów zawierania, rejestracji i wypowiadania UZP:
   → zawsze weryfikuj aktualny tekst Dz.U. 2025 poz. 1661 w ISAP
```

### Spory zbiorowe i strajk (nowa ustawa Dz.U. 2025 poz. 1661)

```
Procedura:
1. ROKOWANIA (obowiązkowe, min. kilka spotkań — weryfikuj w ustawie)
2. MEDIACJA (mediator ze wspólnej listy lub wyznaczony przez MRiPS)
3. ARBITRAŻ (Kolegium Arbitrażu Społecznego przy SN — dobrowolny)
4. STRAJK (ostateczność)
   → Wymaga referendum: frekwencja ≥ 50% + co najmniej 50% głosujących ZA
   → Zakaz strajku w służbach kluczowych — weryfikuj art. ustawy
   → Pracodawca NIE może wypowiadać umów w trakcie legalnego strajku

⚠️ Szczegółowe wymogi proceduralne — ZAWSZE weryfikuj aktualną ustawę Dz.U. 2025 poz. 1661
```

## 5. STRATEGIA

### Perspektywa działacza ZZ

1. Sprawdź czy ZOZ wyrażała/odmawiała zgody na zwolnienie na piśmie.
2. Jeśli pracodawca zwolnił bez zgody → roszczenie o przywrócenie (art. 45 KP + ochrona z ustawy o ZZ).
3. Dokumentuj każde przejawy dyskryminacji za działalność związkową.

### Perspektywa pracodawcy

1. Zawsze uzyskaj pisemną zgodę zarządu ZOZ lub pisemną odmowę przed zwolnieniem działacza.
2. Zachowaj procedurę konsultacyjną przy zwolnieniach grupowych.
3. W sporze zbiorowym — prowadź pisemne protokoły z każdego etapu.
