# MD2 — Scoring dowodów 0–10

## Wzór scoringu

```
Autentyczność    × 0.8   (1–5)
Kompletność      × 0.6   (1–5)
Bezpośredniość   × 1.0   (1–5)
Bonus poziomu:   A=+3.0  B=+1.5  C=0  D=-2.0
Legalność:       legalna=0  wątpliwa=-1.5  nielegalna=-4.0
Weryfikowalność: łatwa=+1.0  trudna=-0.5  niemożliwa=-1.5
```

## Skala ocen

| Score | Ocena | Rekomendacja |
|-------|-------|-------------|
| 8–10 | Silny | Priorytet w piśmie procesowym |
| 6–7 | Umiarkowany | Powołaj, wzmocnij jeśli możliwe |
| 4–5 | Słaby | Powołaj ostrożnie, wskaż ograniczenia |
| 0–3 | Bezużyteczny | Rozważ rezygnację lub uzupełnienie |

## Format scoringu

```
SCORING — [nazwa]
  Autentyczność [n]×0.8=[x] | Kompletność [n]×0.6=[x] | Bezpośredniość [n]×1.0=[x]
  Bonus [poz.]=[x] | Legalność=[x] | Weryfikowalność=[x]
  ─────────────────────────────────────────────────
  WYNIK: [X.X]/10 → [Silny / Umiarkowany / Słaby / Bezużyteczny]
  Uzasadnienie: [1 zdanie kluczowego czynnika]
```
