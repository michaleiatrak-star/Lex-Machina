# M5 — Plan uderzenia przeciwnika i przeciwplan

## Cel
Zbudować realną strategię strony przeciwnej oraz natychmiastową kontrstrategię. Ten moduł ma być brutalnie praktyczny.

## 5.1 Cele przeciwnika

```text
Cel główny:
Cele poboczne:
Co musi podważyć:
Co może przyznać bez szkody:
Co będzie próbował przemilczeć:
```

## 5.2 Typy uderzeń

- formalne: właściwość, termin, prekluzja, braki formalne;
- materialnoprawne: brak przesłanki roszczenia;
- **dowodowe** (patrz: MOD-ATAK-NA-DOWOD.md §AD-1..AD-12):
  - AD-1 autentyczność: metadane, podpis, fałszerstwo
  - AD-2 custody: łańcuch przechowywania, przerwy
  - AD-3 relewantność: fakt bez znaczenia (art. 227 KPC / art. 170 §1 pkt 2 KPK)
  - AD-4 forma: kopia bez oryginału / poświadczenia (art. 129 §1 KPC)
  - AD-5 zakaz ustawowy: nagrania, tajemnica, RODO
  - AD-6 wiarygodność treści: retrospektywne, interes autora
  - AD-7 zakres wniosku: nieokreślony, nieprzydatny (art. 235¹ KPC)
  - AD-8 prekluzja: spóźniony (art. 235² KPC / art. 170 §1 pkt 5-6 KPK)
  - AD-9 kontrdowód aktywny: KD-1 dokument → KD-5 chronologia negatywna
  - AD-10 elektroniczne: brak metadanych, hash, AI/deepfake
  - AD-11 jednostronny ex parte: wytworzony przez stronę dla sporu
  - AD-12 systemowy: sprzeczność, cherry-picking, koordynacja, luki
- kauzalne: brak związku przyczynowego;
- kwotowe: niewykazana wysokość;
- reputacyjne: atak na konsekwencję i wiarygodność;
- narracyjne: odwrócenie osi sporu;
- proceduralne: wnioski o pominięcie, oddalenie, zobowiązanie, dopuszczenie dowodu przeciwnego.

```text

```

```text
Faza 4 — atak na kwoty i związek przyczynowy:

```

## 5.6 Strategia własna

```text
Co podnieść proaktywnie:
Co uprzedzić:
Co zostawić jako odpowiedź:
Co usunąć z narracji:
Co wzmocnić dowodowo:
Najlepsza kolejność argumentów:
```

## 5.7 Reguły

- Nie buduj ataku bez faktu.
- Atak słabszy niż 4/10 oznacz jako poboczny.
- Atak na wiarygodność wymaga konkretnej kolizji.
- Jeżeli przeciwnik ma argument skuteczny, nie udawaj, że go nie ma.
- Każdy plan ataku musi mieć przeciwplan.
