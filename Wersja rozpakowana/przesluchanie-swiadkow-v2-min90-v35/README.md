# przesluchanie-swiadkow-v2-min90

Scalona, najdojrzalsza wersja modułu przesłuchania świadków.

## Źródła scalania

- stabilność i self-contained JSX z `jsxfix2`,
- typologie świadków i sędziów z `jsxfix1`,
- routing/testy z wersji pośredniej,
- text-first i YAML frontmatter z wersji kompletnej.

## Status

- `SKILL.md` zaczyna się od YAML `---`.
- Domyślny tryb: tekst.
- JSX: tylko na żądanie.
- JSX: samowystarczalny, bez importów `local component subfolder imports`.
- Dodano typy świadków, typy sędziów i macierz strategii.
