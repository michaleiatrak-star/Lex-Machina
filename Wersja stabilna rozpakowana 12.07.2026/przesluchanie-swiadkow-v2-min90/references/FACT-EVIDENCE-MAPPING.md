# FACT-EVIDENCE-MAPPING — v3.4

> Plik jest INDEX-em do preparation chartów z WITNESS-INTELLIGENCE KROK IV.
> Każde pytanie W3 musi mieć wpis tutaj z pełną ścieżką źródłową.

## Zasada

Mapa obowiązkowa: pytanie → cel procesowy → fakt → dowód kontrolny → ryzyko

Rozszerzona forma (v3.4):

```
PYTANIE_ID: [P-XX]
Pytanie:    [treść skrócona]
Teza:       [T_X]
Fakt:       [FAKT-XX z preparation chart]
Źródło:     dok_id=[X], strona=[Y]
Dowód kon.: [dok_id + strona] lub [SAFE-Q — brak dowodu, oba warianty bezpieczne]
Ryzyko:     [BEZPIECZNE / RYZYKO-KONTROLA / RYZYKO-ODPOWIEDŹ / RYZYKO-KUMULACJA]
Blok W3:    [A / B / C / D / E]
```

## Zasilenie

Wpisy budowane automatycznie z KROK IV (preparation chart) podczas W3.
Nie twórz wpisów ręcznie — każdy wpis musi mieć dok_id z KROKU III.
