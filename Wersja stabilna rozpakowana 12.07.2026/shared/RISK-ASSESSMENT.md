# RISK-ASSESSMENT — procesowa ocena ryzyka

## 1. Skala

| Poziom | Znaczenie |
|---|---|
| Niskie | brak widocznej przeszkody, ryzyko zwykłe |
| Średnie | istnieje problem wymagający wyjaśnienia lub dowodu |
| Wysokie | realne ryzyko zwrotu, odrzucenia, pominięcia albo oddalenia |
| Krytyczne | pismo w obecnej postaci nie powinno zostać złożone |

## 2. Kategorie

- formalne,
- terminowe,
- opłatowe,
- właściwości sądu / organu,
- legitymacji,
- ciężaru dowodu,
- prekluzji,
- kosztów,
- wiarygodności,
- związku przyczynowego,
- wysokości szkody,
- dopuszczalności środka.

```text

```

## 4. Zasada uczciwej oceny

Nie wzmacniaj sztucznie argumentacji strony zlecającej. Wskaż najgorszy realistyczny scenariusz i najbardziej prawdopodobną reakcję sądu.
