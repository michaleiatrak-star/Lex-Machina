# Modul - Swiadczenia opieki zdrowotnej finansowane ze srodkow publicznych (NFZ)

## Akt prawny
**Ustawa z 27.08.2004 r. o swiadczeniach opieki zdrowotnej finansowanych ze srodkow publicznych**
- Tekst jednolity: **Dz.U. 2025 poz. 1461** (t.j. z 26.09.2025)
  VER: isap.sejm.gov.pl [2026-05-31]

**Weryfikacja ISAP przed kazdy cytowaniem.**

## Zakres
Reguluje: zasady finansowania swiadczen przez NFZ, ubezipieczenie zdrowotne,
koszyk swiadczen gwarantowanych, listy oczekujacych, kontrakty NFZ z placowkami.

## Ubezpieczenie zdrowotne
```
Ubezpieczeni:     Pracownicy, zleceniobiorcy, przedsiebiorcy, bezrobotni (z UP)
Skladka:          9% od wynagrodzenia brutto (pracownik) - via ZUS
Nieubezpieczeni:  Moga dobrowolnie przystapic za oplaata lub korzystac z pomocy spolecznej
Dzieci do 18 lat: Z mocy prawa objete opiek
```

## Swiadczenia gwarantowane (koszyk)
```
Podstawowa opieka zdrowotna (POZ):  Lekarz rodzinny + pielegniarnia + polozna - bez skierowania
Ambulatoryjna opieka specjalistyczna (AOS): Ze skierowaniem od POZ (wyjatki: gin, onkol., psych)
Leczenie szpitalne:                 Hospitalizacja, SOR, ambulatoria szpitalne
Rehab. lecznicza, stomatol.:        W ramach kontraktu z NFZ
```

## Kolejki i listy oczekujacych
```
Czas oczekiwania: Publikowane na portalu pacjenta.gov.pl
Priorytetowe:     Przypadki pilne (rozszerzony koszyk, priorytet w kolejce)
Onkologia:        Szybka sciezka diagnostyczno-lecznicza (DILO) - bez kolejek
Garantowane terminy: AOS specjalistyczne - terminy max (co kwartale aktualizowane)
```
