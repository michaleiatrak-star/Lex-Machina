# WITNESS-SCORING

> ⛔ Przy ocenie scoringu opartego na przepisach prawa (np. zakazy dowodowe,
> ciężar dowodu art. 6 KC / art. 232 KPC):
> `view /mnt/skills/user/shared/PRAWO-HARDGATE.md`

## Skala 0–10

| Zakres | Ocena jakościowa |
|---|---|
| 0–3 | słaby — niska moc dowodowa lub wysokie ryzyko wiarygodności |
| 4–6 | umiarkowany — istotny, ale z ograniczeniami |
| 7–8 | istotny — mocny punkt sprawy |
| 9–10 | kluczowy — może przesądzić o wyniku |

## Kryteria oceny

| Kryterium | Waga |
|---|---|
| Bezpośredniość wiedzy o zdarzeniu | wysoka |
| Interes w wyniku sprawy | wysoka |
| Spójność z dokumentami i innymi dowodami | wysoka |
| Odporność na kontrpytania (prognoza) | średnia |
| Znaczenie dla ciężaru dowodu | wysoka |
| Stabilność relacji w czasie | średnia |
| Warunki obserwacji (tylko typ 2 — świadek naoczny) | wysoka |

## Wpływ zakazów dowodowych na scoring

Jeśli zeznania świadka mogą podlegać zakazowi dowodowemu lub ograniczeniu
(prawo do odmowy, tajemnica zawodowa, zakaz impeachmentu):

```
→ HARDGATE → view /mnt/skills/user/shared/PRAWO-HARDGATE.md
→ Zweryfikuj przepis online
→ Zmniejsz scoring o 1–3 punkty jeśli zakaz potwierdzony
→ Odnotuj w uzasadnieniu: "Ryzyko: [przepis] ✅ [VER: ISAP, data]"
```

## Scoring finalny — format

```
SCORING FINALNY: [X]/10 — [słaby / umiarkowany / istotny / kluczowy]

Uzasadnienie:
- Bezpośredniość wiedzy: [ocena]
- Interes w wyniku: [ocena]
- Spójność z dokumentami: [ocena]
- Odporność na kontrpytania: [ocena]
- Znaczenie dla ciężaru dowodu: [ocena]
- Warunki obserwacji (jeśli typ 2): [ocena]
- Ograniczenia prawne: [ewentualny zakaz dowodowy ✅ lub brak]

Zmiana względem W2: [bez zmiany / podniesiony / obniżony + powód]
```
