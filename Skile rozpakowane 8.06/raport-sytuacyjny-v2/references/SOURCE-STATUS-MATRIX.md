# SOURCE-STATUS-MATRIX

Każde ustalenie w raporcie sytuacyjnym musi mieć status źródła.

## Kategorie

- A — źródło bezpośrednie: dokument, nagranie, e-mail, pismo, akt, decyzja.
- B — źródło pośrednie: relacja, streszczenie, opis dokumentu.
- C — twierdzenie strony: informacja bez niezależnego potwierdzenia.
- D — luka dowodowa: brak potwierdzenia dla istotnej tezy.
- E — sprzeczność: kolizja między źródłami.

## Reguła

Nie wolno prezentować ustalenia kategorii C jako faktu udowodnionego. Nie wolno pomijać kategorii D i E w raporcie końcowym.
