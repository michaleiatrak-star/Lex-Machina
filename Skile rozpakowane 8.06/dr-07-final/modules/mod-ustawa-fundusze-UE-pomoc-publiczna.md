## 3. FUNDUSZE UE 2021–2027 — PROCEDURA

### Architektura systemu

```
Komisja Europejska (KE)
  → Umowa Partnerstwa z Polską (UPP 2021–2027)
    → Programy Operacyjne (krajowe i regionalne):
       FENG (innowacje), FERC (kolej), FEPW (wschód), FEnIKS (infrastruktura),
       FEW (woj. odpowiedniki — 16 programów regionalnych)
       KPO (Krajowy Plan Odbudowy — odrębny instrument)

Instytucje w systemie:
  IZ (Zarządzająca) — ministerstwo / marszałek
  IP (Pośrednicząca) — delegowane zadania
  IW (Wdrażająca) — nabory wniosków, umowy z beneficjentami
```

### Ścieżka odwoławcza beneficjenta

```
Negatywna ocena wniosku / odmowa dofinansowania:
  → Protest do IZ (termin: 14 dni od doręczenia informacji)
  → Wynik protestu → skarga do WSA (termin: 14 dni od wyniku protestu)
  → NSA (skarga kasacyjna: 30 dni od wyroku WSA)
  ⚠️ Terminy — weryfikuj aktualne w ustawie o realizacji funduszy UE w ISAP

Decyzja o zwrocie dotacji:
  → KPA → odwołanie (14 dni) → WSA → NSA
```

### Zwrot dotacji — typowe przyczyny

```
□ Naruszenie zasad PZP przy zamówieniach w projekcie
□ Kwalifikowalność wydatków (wykazanie poniesienia i związku z projektem)
□ Nieosiągnięcie wskaźników rezultatu
□ Podwójne finansowanie (ten sam wydatek z dwóch źródeł)
□ Nieprawidłowości w dokumentacji (faktury, zestawienia, raporty)
□ Zmiana przeznaczenia dofinansowanego majątku (trwałość projektu)

KOREKTY FINANSOWE:
  → Taryfikator korekt (MFiPR) — weryfikuj aktualny
  → Naruszenie PZP: korekta 5% / 10% / 25% / 100% w zależności od wagi naruszenia
```

## 4. POMOC PUBLICZNA

### Zasady absolutne

```
ZAKAZ (art. 107 ust. 1 TFUE):
  Selektywna korzyść dla przedsiębiorstwa finansowana z zasobów państwowych,
  zakłócająca konkurencję i wpływająca na wymianę handlową w UE

WYJĄTKI:
  → De minimis: do 300 000 EUR / 3 lata obrotowe (Rozp. UE 2023/2831, od 01.01.2024)
    ⚠️ NOWE ROZPORZĄDZENIE od 01.01.2024 — poprzednie: 200 000 EUR
    Rolnictwo: 20 000 EUR / 3 lata (Rozp. UE 2023/2832)
    Transport: 300 000 EUR / 3 lata
  → GBER (Rozp. UE 2022/2472 — nowy od 2023):
    Wyłączenia blokowe dla wybranych kategorii (B+R, MŚP, inwestycje, szkolenia)
  → Pomoc indywidualna: notyfikacja KE i zatwierdzenie przed udzieleniem

KUMULACJA POMOCY:
  → Zakaz przekraczania intensywności z różnych programów / instrumentów
  → Rejestracja w SUDOP (System Udostępniania Danych o Pomocy Publicznej)

ZWROT POMOCY NIEZGODNEJ:
  → Decyzja KE + odsetki (rynkowe od dnia udzielenia)
  → Zawieszenie de minimis gdy przekroczony limit
```

## 7. STRATEGIA, QUALITY GATE, OUTPUT

**Strategia:** Przy zwrocie dotacji — zaskarż decyzję i kwestionuj taryfikator korekt (nieproporcjonalne). Przy de minimis — zawsze weryfikuj limit w SUDOP przed udzieleniem. Przy kontroli — dostarcz pełną dokumentację zanim wynik kontroli zostanie zatwierdzony.

**Quality gate:** Termin protestu (14 dni) / odwołania (14 dni) obliczony? Nowe de minimis 300 000 EUR (od 01.01.2024) zastosowane? Ustawa polityki rozwoju — aktualny t.j. Dz.U. 2025 poz. 1733?

**Output:** Kwalifikacja (fundusz / instrument) → etap → termin → dowody → korekty → strategia → rekomendacja.
