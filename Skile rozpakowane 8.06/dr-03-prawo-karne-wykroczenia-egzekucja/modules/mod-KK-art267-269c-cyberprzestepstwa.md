# Moduł [T] — Cyberprzestępczość

**Zakres:** Art. 267–269c KK (hacking, DDoS, sabotaż), art. 287 KK (oszustwo komputerowe,
phishing), art. 190a §2 KK (podszywanie w sieci), art. 212/216 KK (zniesławienie/zniewaga
online), naruszenie wizerunku, fałszywe profile, procedury dowodowe dla cyberprzestępczości.

**Weryfikacja:** 22.05.2026
**Podstawa:** KK Dz.U.2025.383 t.j. (wersja od 26.03.2025)

---

## ZASADY ABSOLUTNE

1. **DOWODY CYFROWE NISZCZONE SZYBKO** → zabezpiecz natychmiast: screenshoty z URL i datą,
   eksport danych, hash pliku. Platformy usuwają treści nawet po zgłoszeniu.
2. **LOGI OPERATORA / IP SPRAWCY** → tylko prokuratura / policja może je uzyskać (art. 218 KPK).
   Osoba prywatna nie ma do nich dostępu. Dlatego zawiadomienie karne jest konieczne nawet
   gdy celem jest wyłącznie sprawa cywilna.
3. **TRYBY ŚCIGANIA — KLUCZOWA RÓŻNICA (szczegóły w tabeli poniżej):**
   - Art. 267 §1–4 KK (hacking, dostęp, podsłuch) → **NA WNIOSEK** pokrzywdzonego (§5)
   - Art. 268 §1–3 KK (niszczenie informacji) → **NA WNIOSEK** (§4)
   - Art. 268a §1–2 KK (niszczenie danych informatycznych) → **NA WNIOSEK** (§3)
   - Art. 269 §1–2 KK (dane o szczególnym znaczeniu — infrastruktura krytyczna) → **Z URZĘDU**
   - Art. 269a KK (zakłócenie systemu — DDoS) → **Z URZĘDU**
   - Art. 269b KK (narzędzia hakerskie) → **Z URZĘDU**
   - Art. 287 KK (oszustwo komputerowe) → **Z URZĘDU**; wyjątek: osoba najbliższa → **na wniosek** (§3)
4. **JURYSDYKCJA** → cyberprzestępstwo może być popełnione z zagranicy → MLAT (pomoc prawna
   międzynarodowa) przez prokuraturę. Efekty bardzo powolne.
5. **PLATFORMY ZAGRANICZNE** → zgłoszenie do platformy (Facebook, Google, X) nie zastępuje
   zawiadomienia o przestępstwie. Obie ścieżki mogą i powinny być równoległe.
6. **KLAUZULA WHITE-HAT** (art. 269b §1a + art. 269c KK — Dz.U.2025.383):
   - Art. 269b §1a: nie popełnia przestępstwa (narzędzia hakerskie) ten, kto działa wyłącznie
     w celu zabezpieczenia systemu lub opracowania metody zabezpieczenia
   - Art. 269c: nie podlega karze za art. 267 §2 (nieautoryzowany dostęp) lub art. 269a (DDoS)
     ten, kto: (a) działa wyłącznie w celu zabezpieczenia/opracowania metody + (b) niezwłocznie
     powiadomił dysponenta systemu o ujawnionych zagrożeniach + (c) nie naruszył interesu
     publicznego/prywatnego i nie wyrządził szkody

## ZABEZPIECZENIE DOWODÓW — PROTOKÓŁ

```
NATYCHMIAST (zanim platforma usunie):
  1. Screenshot z widocznym URL, datą i godziną systemową (nie kadruj — pokaż pasek przeglądarki)
  2. Eksport/archiwum korespondencji:
     Gmail → Ustawienia → Zarządzaj kontem → Dane i prywatność → Pobierz dane
     Facebook → Ustawienia → Twoje informacje na Facebooku → Pobierz dane
     WhatsApp → Eksport rozmowy (w ustawieniach czatu)
  3. Zrzut ekranu z datą i URL profilu / strony / wpisu sprawcy
  4. Notarialne poświadczenie treści strony internetowej (najsilniejszy dowód)
  5. Hash MD5/SHA256 pliku screenshota — opcjonalnie, wzmacnia wiarygodność

ADRES IP SPRAWCY:
  → Tylko policja/prokuratura uzyska go od platformy/operatora (art. 218 KPK)
  → Dlatego zawiadomienie karne = warunek ustalenia tożsamości anonimowego sprawcy

ZGŁOSZENIE DO PLATFORMY:
  → Równolegle z zawiadomieniem karnym (nie zamiast)
  → Facebook: facebook.com/help/reportlinks
  → Google/YouTube: support.google.com/youtube/answer/2802027
  → X (Twitter): help.twitter.com/en/safety-and-security/report-abusive-behavior
```

## TRYB ZAWIADOMIENIA KARNEGO

```
Przestępstwo z urzędu (269 §1–2 = infrastruktura krytyczna, 269a = DDoS, 269b §1 = narzędzia hakerskie, 287 = phishing):
  → Zawiadomienie o podejrzeniu popełnienia przestępstwa do Policji/Prokuratury
  → Organy mają obowiązek wszcząć postępowanie — wola pokrzywdzonego irrelewantna

Przestępstwo wnioskowe (267 §1–4 = hacking, 268, 268a, 190a §2):
  → Zawiadomienie MUSI zawierać wyraźny WNIOSEK O ŚCIGANIE
  → Bez wniosku — organy nie mogą wszcząć postępowania (art. 17 §1 pkt 10 KPK)
  → Wniosek można cofnąć tylko za zgodą prokuratora (w pp) lub sądu (w ps)

Prywatno-skargowe (212, 216 — zniesławienie/zniewaga online):
  → Prywatny akt oskarżenia bezpośrednio do sądu rejonowego
  → Wyjątek: prokurator może wstąpić do sprawy (art. 60 §1 KPK — interes społeczny)
```

## INTAKE — PYTANIA WSTĘPNE

```
□ Co konkretnie nastąpiło? (włamanie / fałszywy profil / groźby / zniesławienie / DDoS / phishing)
□ Kiedy odkryto i kiedy to nastąpiło?
□ Czy wiadomo kto jest sprawcą (nick / IP / profil / znajomy)?
□ Jakie dowody już zabezpieczono?
□ Jaki efekt oczekuje klient? (usunięcie treści / odszkodowanie / skazanie sprawcy)
□ Czy sprawa ma wymiar transgraniczny (platforma lub sprawca za granicą)?
□ Czy to zdarzenie jednorazowe czy trwające (nękanie, stalking online)?
□ Czy jest szkoda majątkowa? (phishing, ransomware, wyłudzenie — art. 287 KK)
□ Czy sprawca to osoba bliska? (tryb wnioskowyn dla art. 268, 268a, 287 §3)
```
