# DOWODY-METODOLOGIA — metodologia oceny materiału dowodowego

## 1. Cel

Moduł porządkuje dowody według faktów istotnych prawnie, a nie według kolejności załączników.

## 2. Klasyfikacja dowodu

| Kategoria | Opis |
|---|---|
| Bezpośredni | wprost wykazuje fakt istotny |
| Pośredni | wymaga wnioskowania |
| Wspierający | wzmacnia inny dowód |
| Kontekstowy | pokazuje tło, ale nie przesądza znamienia / przesłanki |
| Ryzykowny | może zaszkodzić albo zostać zakwestionowany |

## 3. Matryca dowodowa

```text
FAKT ISTOTNY: ...
Przesłanka prawna: ...
Dowód główny: ...
Dowody wspierające: ...
Luka dowodowa: ...
Kontrargument przeciwnika: ...
Ocena sądu: prawdopodobne / niepewne / słabe
Działanie: uzupełnić / ograniczyć / wykorzystać / nie eksponować
```

## 4. Test wiarygodności

Sprawdź:

- pochodzenie dokumentu,
- datę i integralność,
- zgodność z chronologią,
- sprzeczności z innymi dowodami,
- interes autora,
- możliwość autentyfikacji,
- związek z tezą dowodową.
