# LOCAL-SOURCE-EQUIVALENCE-PROTOCOL

## Cel
Dla prawa miejscowego i lokalnego źródłem równoważnym wobec ISAP są właściwe lokalne publikatory urzędowe.

## Źródła
- Dzienniki Urzędowe Województw.
- BIP wojewodów.
- BIP powiatów.
- BIP gmin.
- BIP jednostek organizacyjnych.
- Akty nadzorcze wojewodów.
- Uchwały i rozstrzygnięcia RIO.
- TERYT/GUS dla struktury jednostek.

## Statusy
- `LOCAL_DUW_OK`
- `LOCAL_BIP_OK`
- `TERYT_OK`
- `RIO_REQUIRED`
- `VOIVODE_SUPERVISION_REQUIRED`
- `REQUIRES_SOURCE_VERIFICATION`
