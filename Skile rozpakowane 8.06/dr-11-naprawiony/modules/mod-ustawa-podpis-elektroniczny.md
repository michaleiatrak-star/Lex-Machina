## Rodzaje podpisow elektronicznych (eIDAS)
```
Elektroniczny (zwykly): Jakiekolwiek dane elektroniczne; najslabsza moc prawna
Zaawansowany:          Powiazany z podpisujacym, identyfikowalny, wykrywa zmiany
Kwalifikowany (QES):   Najsilniejszy - rownowaznie z podpisem wlasnorecz. (art. 25 eIDAS)
                       Wymagany: kwalifikowane urzadzenie + kwalifikowany certyfikat
```

## Dostawcy uslug zaufania (TSP)
```
Kwalifikowani TSP:  Wpisani do listy nadzorowanej przez Ministra Cyfryzacji (MC)
Polska lista:       mc.gov.pl -> lista dostawcow uslug zaufania
Uslugi kwalif.:     Kwalifikowany podpis, piecze, znacznik czasu, doreczenia
```

## eIDAS 2.0 - EUDIW (European Union Digital Identity Wallet)
```
Cel:     Europejski Portfel Tozsamosci Cyfrowej - dla wszystkich obywateli UE
Funkcja: Przechowywanie: dowod osobisty, prawo jazdy, atrybuty zdrowotne
         Uzywanie: logowanie do uslug online (nie tylko rządowe)
Termin:  Panstwa UE musza zaoferowac EUDIW do 31.12.2026 r.
web_search: "EUDIW eIDAS 2.0 portfel cyfrowy Polska 2025 2026 termin"
```
