# Moduł A — Błędy pełnomocnika strony przeciwnej
**v6:** Uruchamiany w Przejściu III (sekcja 3B). Wyniki wchodzą do V10-2 (self-destructive-admissions).
Przed klasyfikacją każdej szkody: obowiązkowa lista kontrolna W3 z WERYFIKACJA-DOWODOW.md.

**Uruchom:** gdy dysponujesz ≥2 pismami procesowymi jednej strony i sprawa jest na etapie aktywnego postępowania.

## 4 kategorie szkód

**KATEGORIA I — Przyznania niekorzystne**
Fragment pisma, który potwierdza fakt, którego strona nie musiała przyznawać i który dostarcza przeciwnikowi gotowego argumentu.

Test:
```
„Czy to zdanie, gdyby znalazło się w piśmie PRZECIWNIKA, szkodzi mojemu klientowi?"
TAK → przyznanie niekorzystne. Oceń wagę: Krytyczna / Wysoka / Średnia / Niska.
```

**KATEGORIA II — Argumenty o skutku odwrotnym**
Fragment o uzasadnionym celu obronnym, lecz faktycznym efekcie odwrotnym.

Mechanizmy:
```
A: Wyjaśnienie kontekstu ujawnia niekorzystny fakt chronologiczny
B: Argument osłabiający jedną tezę podważa inną tezę tej samej strony
C: Złożony dowód potwierdza więcej niż zamierzała strona
D: Powołanie przepisu bez jego użycia ujawnia słabość dowodową
   (np. groźba art. 52 KP + brak dyscyplinarki = wątpliwe podstawy)
```

**KATEGORIA III — Ujawnienie zbędnych faktów**
Fakty wprowadzone do akt, które nie były wymagane przez Sąd ani niezbędne do odparcia konkretnego zarzutu, a dają przeciwnikowi nową linię ataku.

Test konieczności:
```
„Czy Sąd tego wymagał? Czy Powód tego żądał?
 Czy jest to niezbędne do odparcia konkretnego zarzutu?"
NIE na wszystkie trzy → ujawnienie zbędne.
```

**KATEGORIA IV — Błędy strategiczne i procesowe**
- Powołanie świadka bez weryfikacji dostępności, a następnie przyznanie braku jego adresu (rozbija własny wniosek dowodowy)
- Informowanie strony przeciwnej o progach właściwości sądu bez potrzeby
- Rozbudowana narracja o faktach spoza przedmiotu sporu — dostarcza przeciwnikowi materiału do rozszerzenia powództwa

## Format wyników

```
SZKODA [nr] — [nazwa skrótowa]
KATEGORIA: I / II / III / IV
FRAGMENT: [cytat z datą pisma]
ZAMIERZONY EFEKT: [co chciał osiągnąć pełnomocnik]
RZECZYWISTY EFEKT: [co faktycznie osiągnął]
MOŻLIWE WYKORZYSTANIE: [konkretny argument / roszczenie przeciwnika]
SKALA: Krytyczna / Wysoka / Średnia / Niska
```

```

Czy ustalenie faktyczne leżące u podstaw oceny jest zweryfikowane
we WSZYSTKICH pismach tej samej strony?
TAK → klasyfikuj wstępnie | NIE → wróć do Przejścia I

WERYFIKACJA 2 (podczas Przejścia IV — protokół W3 z WERYFIKACJA-DOWODOW.md):
Czy cytat przeczytany w pełnym akapicie (min. 3 zdania kontekstu)?
Czy strona nie wyjaśniła lub nie odwołała twierdzenia w tym samym piśmie?
Czy twierdzenie nie jest sformułowaniem ostrożnościowym?
TAK do wszystkich → klasyfikuj ostatecznie | NIE → zrewiduj ocenę wagi
```
