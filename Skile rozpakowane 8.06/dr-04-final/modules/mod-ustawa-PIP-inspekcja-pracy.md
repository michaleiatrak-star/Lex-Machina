## 1. CORE

PIP = organ nadzoru i kontroli przestrzegania prawa pracy (BHP, czas pracy, wynagrodzenia, umowy, BHP). Podlega Sejmowi RP. Główny Inspektor Pracy powoływany przez Marszałka Sejmu.

**Akt:** Ustawa o Państwowej Inspekcji Pracy — Dz.U. 2024 poz. 1712 t.j.

---

## 2. INTAKE

```
□ Kto składa skargę/wniosek: pracownik / pracodawca / osoba trzecia?
□ Czego dotyczy: wynagrodzenie / BHP / czas pracy / umowa / nielegalny stosunek pracy?
□ Czy skarga jest poufna? (art. 44 ustawy o PIP — pracodawca NIE może się dowiedzieć kto złożył)
□ Jakie działania PIP już podjęła: nakaz / mandat / wniosek o ukaranie?
□ Czy PIP zakwestionowała umowę B2B jako stosunek pracy?
```

---

## 3. PROCEDURA

### Kompetencje PIP

```
□ Kontrola pracodawców (BHP, czas pracy, wynagrodzenia, umowy, ZZ)
□ Nakaz: wstrzymanie pracy (zagrożenie życia/zdrowia), usunięcie uchybień
□ Mandat karny do 2 000 zł / wniosek o ukaranie do sądu grodzkiego
   (art. 108–112 ustawy o PIP — weryfikuj aktualne kwoty w ISAP)
□ Skarga pracownika: przyjmowana w trybie art. 44 — POUFNOŚĆ DANYCH SKARŻĄCEGO
□ Pseudosamozatrudnienie: PIP może kwalifikować umowę B2B jako stosunek pracy
   i nakazać zawarcie umowy o pracę
```

### Kontrola PIP — prawa i obowiązki

```
Pracodawca:
  □ OBOWIĄZEK udostępnienia dokumentacji na żądanie inspektora (art. 23 ustawy)
  □ Umożliwienie przeprowadzenia kontroli

Godziny kontroli:
  → BHP: o każdej porze dnia i nocy
  → Inne kontrole: w godzinach 6:00–22:00 (weryfikuj aktualny art. w ISAP)

Prawa pracodawcy:
  → Sprzeciw od nakazów inspektora do okręgowego inspektora pracy
  → Odwołanie od decyzji inspektora do sądu (gdy kwestionuje stosunek pracy)
```

---

## 4. STRATEGIA

**Perspektywa pracownika:** Skarga do PIP jest bezpłatna i poufna — dobry pierwszy krok przed sporem sądowym. Zgłoś PIP naruszenia BHP, wynagrodzenia lub nielegalny stosunek zatrudnienia.

**Perspektywa pracodawcy:** Przygotuj dokumentację przed kontrolą. Prowadź ewidencję czasu pracy i listy płac zgodnie z prawem. Kwestionuj przekroczenie kompetencji inspektora przez odwołanie.

---

## 5. QUALITY GATE / OUTPUT

Quality gate: Aktualny tekst ustawy o PIP w ISAP? Poufność skargi zachowana? Właściwy inspektor okręgowy?

Output: stan faktyczny → naruszenie → tryb PIP → dowody → strategia → rekomendacja.
