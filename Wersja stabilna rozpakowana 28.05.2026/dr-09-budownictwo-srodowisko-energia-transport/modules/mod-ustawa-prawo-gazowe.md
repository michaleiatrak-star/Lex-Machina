# Modul - Prawo gazowe i regulacja rynku gazu

## Akt prawny
**Ustawa z 10.04.1997 r. - Prawo energetyczne (w zakresie gazu)**
- Tekst jednolity: **Dz.U. 2024 poz. 1538** (ze zm.)
  VER: isap.sejm.gov.pl [2026-05-31]
  UWAGA: Ta sama ustawa reguluje ene. elektryczna, cieplo i gaz (patrz tez PrEnergetyczne-URE-OZE)

**Weryfikacja ISAP przed kazdy cytowaniem.**

## Regulacja rynku gazu
```
Organ regulacyjny: Prezes Urzedu Regulacji Energetyki (URE)
Sprzedaz gazu:     Liberalizowany rynek - wiele podmiotow, wybor sprzedawcy
Taryfy:            Operatorzy systemu gazowego zatwierdza taryfy URE
                   Sprzedaz do odbiorow: taryfy lub oferty rynkowe
Magazynowanie:     GAZ-SYSTEM - operator systemu przesylowego
```

## Obowiazek sprzedazy na gieldzie (TGE)
```
Obowiazek gieldowy: Sprzedawcy gazu musza oferowac na TGE (Towarowa Gielda Energii)
                    okreslony % wolumenu
Cele: Transparentnosc cen, rynek spot i terminowy
```

## Ochrona odbiorcy wrazliwego (art. 3 pkt 13c PrEn.)
```
Odbiorca wrazliwy gazu:    Osoba fizyczna, kobieta w ciazy lub grzejnik glowny
Ochrona: Zasilanie nieprzerwane w sezonie grzewczym nawet przy dlugach
Zmiana sprzedawcy:         Wniosek do sprzedawcy z 1-miesiac wypowiedzeniem
```

## Gaz LNG i biogas
```
Maly skroplony gaz (LNG): Stacje dla transportu ciezkiego
Biometan:                 Wtlaczanie do sieci (regulacje doprecyzowywane)
Wodor:                    Przepisy w toku - blending H2 do sieci i pure H2
```
