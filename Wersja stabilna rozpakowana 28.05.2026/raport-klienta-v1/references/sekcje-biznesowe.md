# Sekcje Raportu dla Klienta Biznesowego

## ZASADA OGÓLNA

Raport BIZ to dokument decyzyjny dla zarządu / rady nadzorczej / właścicieli.
Styl: board memo — zwięzły, strukturalny, bez retoryki. Każda sekcja odpowiada
na konkretne pytanie menedżerskie.

---

## SEKCJA 1 — EXECUTIVE SUMMARY

```
Pytanie menedżerskie: "Co się dzieje i jaka jest rekomendacja?"

Format (max 5 zdań):
• Status sprawy: [etap postępowania — 1 zdanie]
• Ocena ryzyka: [korzystna / umiarkowana / niekorzystna] — [1 zdanie uzasadnienia]
• Prawdopodobieństwo wyniku korzystnego: XX%
• Kluczowy termin: [data + co się dzieje]
• Rekomendacja kancelarii: [1 zdanie — kontynuować / negocjować / rozważyć ugodę / inne]
```

---

## SEKCJA 2 — EKSPOZYCJA FINANSOWA

```
Pytanie menedżerskie: "Ile możemy stracić / zyskać?"

Struktura:
┌─────────────────────────────────────────────────────────┐
│ WARIANT                    │ KWOTA (szacunek)  │ PRAWDOP. │
├─────────────────────────────────────────────────────────┤
│ Wynik w pełni korzystny    │ +/- XX zł         │  XX%     │
│ Wynik częściowo korzystny  │ +/- XX zł         │  XX%     │
│ Wynik niekorzystny         │ +/- XX zł         │  XX%     │
│ Ugoda (szacunek)           │ +/- XX zł         │  —       │
└─────────────────────────────────────────────────────────┘

Zawiera:
• Wartość przedmiotu sporu
• Koszty procesowe (łączne — sądowe + obsługa prawna)
• Potencjalne odsetki ustawowe / kary umowne
• Koszty egzekucji jeśli dotyczy

Jeśli kancelaria nie ma danych do wyceny → wstaw [do ustalenia] i wyjaśnij co jest potrzebne.
NIE podawaj fikcyjnych kwot — tylko dane z akt lub szacunki z wyraźną adnotacją "[szacunek]"
```

---

## SEKCJA 3 — WPŁYW NA DZIAŁALNOŚĆ

```
Pytanie menedżerskie: "Co ta sprawa robi naszej firmie poza pieniędzmi?"

Podsekcje (wypełnij tylko te, które dotyczą sprawy):

3A. REPUTACJA
   • Czy sprawa jest / może być publiczna? (KRS, rejestry, media)
   • Ryzyko medialne: niskie / średnie / wysokie
   • Rekomendacja komunikacyjna: [czy cokolwiek komunikować]

3B. OPERACJE
   • Czy toczące się postępowanie blokuje działalność? (np. zabezpieczenie, zakaz)
   • Wpływ na kontrakty z kontrahentami: [tak/nie + opis]
   • Wpływ na pracowników / strukturę: [tak/nie + opis]

3C. KONTRAKTY I COMPLIANCE
   • Czy wynik sprawy wpłynie na istniejące umowy? (klauzule MAC, change of control)
   • Czy istnieje obowiązek notyfikacji kontrahentów lub regulatora? [tak/nie]
   • Ryzyko naruszenia compliance: [tak/nie + opis]

3D. UBEZPIECZENIA
   • Czy sprawa objęta polisą OC / D&O / inne? [tak/nie/weryfikować]
   • Czy kancelaria otrzymała instrukcje od ubezpieczyciela? [tak/nie]
```

## SEKCJA 4 — HARMONOGRAM ETAPÓW

```
Pytanie menedżerskie: "Co i kiedy się wydarzy?"

Format tabelaryczny:

| # | ETAP                        | PLANOWANA DATA | STATUS      | UWAGI               |
|---|-----------------------------|----------------|-------------|---------------------|
| 1 | [nazwa etapu]               | dd.mm.rrrr     | ✓ Zakończony | —                  |
| 2 | [nazwa etapu]               | dd.mm.rrrr     | ▶ W toku    | —                   |
| 3 | [nazwa etapu]               | dd.mm.rrrr     | ○ Planowany | [zależy od X]       |
| 4 | [nazwa etapu]               | dd.mm.rrrr     | ○ Planowany | —                   |

IKONY STATUSU:
✓ Zakończony  ▶ W toku  ○ Planowany  ⚠ Zagrożony  ✗ Przekroczony termin

SEKCJA "DZIAŁANIA WYMAGANE PO STRONIE KLIENTA":
Wyróżniona osobno — co, do kiedy, kto po stronie klienta jest odpowiedzialny.
```

---

## SEKCJA 5 — REKOMENDACJE DLA ZARZĄDU

```
Pytanie menedżerskie: "Co powinniśmy zdecydować?"

Format:
REKOMENDACJA GŁÓWNA: [kontynuować postępowanie / negocjować ugodę / rozważyć
                      wycofanie / zmienić strategię / podjąć działania prewencyjne]

UZASADNIENIE (max 3 punkty):
• [punkt 1]
• [punkt 2]
• [punkt 3]

DECYZJA WYMAGANA OD ZARZĄDU: [co konkretnie zarząd musi zatwierdzić / zdecydować]
TERMIN DECYZJI: [data — kiedy decyzja jest potrzebna żeby nie stracić możliwości]

OPCJE:
A) [opcja] — skutki: [opis] — rekomendacja: [tak/nie]
B) [opcja] — skutki: [opis] — rekomendacja: [tak/nie]
```

---

## SEKCJA 6 — KLAUZULA POUFNOŚCI

```
Obowiązkowa w każdym raporcie BIZ. Umieszczona na początku i końcu dokumentu.

WZORZEC NAGŁÓWKA:
DOKUMENT POUFNY — TAJEMNICA ZAWODOWA ADWOKATA / RADCY PRAWNEGO
Chroniony tajemnicą zawodową (art. 6 Prawa o adwokaturze /
art. 3 ustawy o radcach prawnych). Przeznaczony wyłącznie dla: [imię nazwisko / stanowisko].
Zakazane: kopiowanie, udostępnianie, cytowanie bez pisemnej zgody kancelarii.

WZORZEC STOPKI:
© [Nazwa Kancelarii] [rok] · Dokument poufny · Tajemnica zawodowa
Wygenerowano: [data] · Raport nr: [numer jeśli jest system numeracji]

UWAGA: Podstawy prawne weryfikuj w ISAP przed każdym użyciem.
```

## RÓŻNICE IND vs BIZ — TABELA SEKCJI

| Sekcja | IND | BIZ |
|---|---|---|
| Status sprawy (co się dzieje) | ✓ język prosty, rzetelny | ✓ formalny |
| Ocena sytuacji | ✓ opisowa (bez owijania w bawełnę) | ✓ procentowa |
| Co dalej / następny krok | ✓ max 3 kroki | ✓ tabela etapów |
| Ekspozycja finansowa | — | ✓ |
| Wpływ na działalność | — | ✓ |
| Rekomendacje dla zarządu | — | ✓ |
| Klauzula poufności / NDA | — | ✓ |
| Executive summary | — | ✓ |
| Działania wymagane od klienta | ✓ prosta lista | ✓ tabela z terminami |
