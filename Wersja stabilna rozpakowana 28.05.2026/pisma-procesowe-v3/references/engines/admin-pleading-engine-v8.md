# ADMIN PLEADING ENGINE V8

## Zakres

- odwołanie od decyzji,
- zażalenie na postanowienie,
- ponaglenie,
- skarga na bezczynność,
- skarga do WSA,
- skarga kasacyjna do NSA,
- wniosek o wznowienie,
- wniosek o stwierdzenie nieważności,
- wniosek o wstrzymanie wykonania decyzji.

## Obowiązkowa analiza

1. właściwość organu,
2. termin,
3. doręczenie,
4. strona/interes prawny,
5. podstawa prawna,
6. ustalenia faktyczne,
7. materiał dowodowy,
8. uzasadnienie,
9. pouczenie,
10. tryb zaskarżenia.

## Typowe zarzuty

- brak wyjaśnienia stanu faktycznego,
- dowolna ocena dowodów,
- naruszenie czynnego udziału strony,
- brak uzasadnienia,
- błędna wykładnia przepisu,
- niewłaściwa subsumcja,
- pominięcie wniosków dowodowych,
- bezczynność lub przewlekłość.
