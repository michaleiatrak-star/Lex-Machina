# STRATEGIA-PROCESOWA — decyzje taktyczne w sprawie

## 1. Cel

Moduł nie tworzy pisma. Służy do decyzji: co złożyć, kiedy, w jakiej kolejności i czego nie ujawniać przedwcześnie.

## 2. Analiza strategiczna

```text
CEL PROCESOWY: ...
Etap sprawy: ...
Najważniejszy fakt do udowodnienia: ...
Najsilniejszy dowód: ...
Największa luka: ...
Najsilniejszy argument przeciwnika: ...
Najbardziej prawdopodobne stanowisko sądu: ...
Rekomendowany następny ruch: ...
Ruchy niewskazane: ...
```

## 3. Typowe decyzje

- czy wnosić pismo teraz czy czekać na zarządzenie,
- czy składać wniosek dowodowy od razu,
- czy żądać zabezpieczenia,
- czy wnosić o zobowiązanie przeciwnika do przedłożenia dokumentów,
- czy składać zażalenie,
- czy wniosek o mediację pomaga czy osłabia pozycję,
- czy eskalować do organu dyscyplinarnego / prokuratury / UODO / PIP,
- czy ujawnienie określonego dowodu nie pozwoli przeciwnikowi przygotować obrony.

## 4. Timing — kiedy złożyć pismo (MOD-TIMING)

Gdy pytanie dotyczy terminu złożenia pisma lub kolejności ruchów:

```
view /mnt/skills/user/shared/MOD-TIMING.md
```

MOD-TIMING realizuje macierz T1–T5 i sześć modeli timing:
- T-EARLY (kształtuj perspektywę sądu przed rozprawą),
- T-REACTIVE (złóż po ruchu przeciwnika),
- T-INERTIA (wykorzystaj efekt korzystnego orzeczenia),
- T-SERIAL (sekwencja wniosków częściowych zamiast omnibus),
- T-ORAL-AMBUSH (zachowaj argument na rozprawę),
- T-ADVANCE-NOTICE (edukacja sądu w specjalistycznej dziedzinie).

Wczytaj MOD-TIMING gdy użytkownik pyta: "kiedy złożyć", "czy teraz czy później",
"czy nie poczekać z tym wnioskiem", "w jakiej kolejności składać pisma".
