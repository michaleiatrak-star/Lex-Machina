# Prawo Spadkowe — Szczegółowy Framework

> ⚠️ **WERYFIKACJA ONLINE OBOWIĄZKOWA** — każdy przepis weryfikuj w isap.sejm.gov.pl
> Prawo spadkowe przeszło istotną nowelizację 15 listopada 2023 r. (Dz.U. 2023 poz. 1615).

## TERMIN 6 MIESIĘCY — ABSOLUTNY PRIORYTET
Oświadczenie o przyjęciu / odrzuceniu: **6 miesięcy od dowiedzenia się o powołaniu**.
Brak oświadczenia = przyjęcie **Z DOBRODZIEJSTWEM INWENTARZA** (od nowelizacji 2015).
Sprawdź termin JAKO PIERWSZY ELEMENT każdej analizy.

### NOWELIZACJA 2023 — kluczowe zmiany art. 1015 KC (od 15.11.2023):
```
§ 1¹ KC: Do zachowania 6-miesięcznego terminu WYSTARCZY złożenie przed jego
         upływem WNIOSKU do sądu o odebranie oświadczenia (nie samo oświadczenie!)
         → praktyczne znaczenie: termin zachowany nawet jeśli sąd odbierze
           oświadczenie po jego upływie
§ 1² KC: Jeśli złożenie oświadczenia wymaga ZEZWOLENIA SĄDU (np. w imieniu
         małoletniego) → bieg terminu ULEGA ZAWIESZENIU na czas postępowania
         sądowego w tym przedmiocie
Stosuje się do terminów, które nie upłynęły przed 15.11.2023.
```

## SCHEMAT POSTĘPOWANIA SPADKOWEGO
```
Śmierć → Sprawdź NRSD u notariusza (Notarialny Rejestr Spadków i Dokumentów)
       → Testament? → ważność formalna (patrz niżej)
       → Długi? (BIK, ekw.ms.gov.pl — KW, komornik, ZUS, US)
       → Oświadczenie w 6 m-cy (złóż WNIOSEK do sądu lub oświadczenie
           u notariusza / przed sądem — w terminie 6 m-cy)
       → SNS: notariusz (APD — gdy wszyscy zgodni) LUB sąd rejonowy
       → Dział spadku (opcjonalnie przy kilku spadkobiercach)
       → Zgłoszenie do US (SD-Z2) w 6 m-cy od uprawomocnienia SNS / APD
       → Wpis w KW nieruchomości
```

## NOTARIUSZ vs SĄD — KIEDY CO
| Notariusz (APD) | Sąd (SNS) |
|---|---|
| Wszyscy spadkobiercy obecni i zgodni | Spór / ktoś nieobecny |
| Szybko i drożej | Wolniej (3–12 m-cy), opłata 100 zł |
| Brak testamentów szczególnych | Testamenty szczególne, kwestionowanie |

## DZIEDZICZENIE USTAWOWE — GRUPY (art. 931–935 KC)
```
KRĄG I:   Małżonek + dzieci (małżonek min. 1/4)
          Nie dożyło dziecko → dziedziczą jego zstępni

KRĄG II:  Małżonek + rodzice — gdy brak zstępnych (art. 932 KC)
          Małżonek: 1/2
          Każde z rodziców: 1/4
          ⚠️ Jeśli jedno z rodziców nie żyje: jego 1/4 → rodzeństwo
             spadkodawcy (po równo); nie dożyło rodzeństwo → jego zstępni
          Jeśli brak małżonka: rodzice + rodzeństwo + zstępni rodzeństwa
             dziedziczą w częściach równych

KRĄG III: Dziadkowie + ich dzieci + wnuki dziadków (art. 934 KC)
          ⚠️ NOWELIZACJA 2023 (od 15.11.2023):
          Jeśli dziadek nie dożył → jego udział → jego DZIECI (po równo)
          Jeśli dziecko dziadka nie dożyło → udział → jego DZIECI (po równo)
          WYŁĄCZENI: dalsi zstępni (cioteczne/stryjeczne wnuki, dalsze pokolenia)
          Brak dziadka + jego dzieci + wnuków → udział → POZOSTAŁYM DZIADKOM
          (gmina/Skarb Państwa zamiast dalszych krewnych)

KRĄG IV:  Pasierbowie — gdy oboje rodzice biologiczni spadkodawcy nie żyją

OSTATECZNIE: Gmina (miejsce zam.) / Skarb Państwa (gdy gmina nieznana)
```

## TESTAMENT — WAŻNOŚĆ FORMALNA
```
Własnoręczny (art. 949 KC):
  ✓ CAŁY odręcznie napisany + podpis + data
  ✗ Wydruk z podpisem = NIEWAŻNY
  ✗ Brak daty → nieważny (wyjątek: sąd może uznać za ważny gdy brak daty
    nie wywołuje wątpliwości co do zdolności / kolejności testamentów)
Notarialny (art. 950 KC):
  ✓ Najtrudniejszy do podważenia
  ✓ Rejestrowany w NRSD
Allograficzny (art. 951 KC):
  ✓ Przed wójtem / starostą / prezydentem / marszałkiem + 2 świadkowie
  ✓ Spisany w protokole, odczytany, podpisany
  ✗ Nieważny gdy spadkodawca głuchoniemy
Ustny (art. 952 KC):
  ✓ Wyłącznie szczególne okoliczności (obawa rychłej śmierci / nadzwyczajne utrudnienie)
  ✓ 3 świadkowie (lub 2 gdy pisany protokół)
  ✗ WYGASA po 6 m-cy od ustania okoliczności
  ✗ Wymaga sądowego stwierdzenia nabycia (nie APD)
```

## NIEGODNOŚĆ DZIEDZICZENIA (art. 928 KC) — AKTUALNY KATALOG 5 PRZESŁANEK
```
Nowelizacja 2023 (Dz.U. 2023 poz. 1615) — obowiązuje od 15.11.2023:

Pkt 1: dopuścił się umyślnie ciężkiego przestępstwa przeciwko spadkodawcy
Pkt 2: podstępem / groźbą nakłonił / przeszkodził w sporządzeniu testamentu
Pkt 3: umyślnie ukrył / zniszczył / podrobił / przerobił testament
Pkt 4 (NOWY): uporczywie uchylał się od wykonywania wobec spadkodawcy
               OBOWIĄZKU ALIMENTACYJNEGO określonego CO DO WYSOKOŚCI:
               orzeczeniem sądu, ugodą lub inną umową
               ⚠️ Samo moralne zobowiązanie do pomocy NIE wystarczy
Pkt 5 (NOWY): uporczywe uchylanie od OBOWIĄZKU PIECZY nad spadkodawcą,
               w szczególności wynikającego z:
               → władzy rodzicielskiej, opieki, funkcji rodzica zastępczego
               → małżeńskiego obowiązku wzajemnej pomocy
               → obowiązku wzajemnego szacunku rodzica i dziecka

Skutek uznania za niegodnego: traktowany jakby nie dożył otwarcia spadku.
Przebaczenie przez spadkodawcę: uchyla niegodność.
Stosuje się do zdarzeń zaistniałych po 15.11.2023.
```

## ODRZUCENIE SPADKU W IMIENIU MAŁOLETNIEGO (nowelizacja 2023)
```
Od 15.11.2023 — uproszczona procedura:
□ Wniosek o zezwolenie na odrzucenie → sąd SPADKU (nie opiekuńczy!)
□ Jeśli sprawa SNS już się toczy → sąd spadku rozpoznaje obie sprawy łącznie
□ Termin 6 m-cy ulega ZAWIESZENIU na czas postępowania o zezwolenie (art. 1015 §1² KC)
□ Wystarczy złożyć wniosek do sądu w terminie (§1¹ KC)
Podstawa: art. 640¹ KPC (dodany nowelizacją 2023)
```

## ZACHOWEK (art. 991–1011 KC)
**Uprawnieni:** Zstępni + małżonek + rodzice (RODZEŃSTWO NIE JEST uprawnione do zachowku!).
**Wysokość:**
- ½ udziału ustawowego → zasada ogólna
- ⅔ udziału ustawowego → małoletni LUB trwale niezdolny do pracy

**Przedawnienie: 5 lat** od ogłoszenia testamentu / otwarcia spadku (art. 1007 KC).

### Obliczenie — 4 kroki:
```
1. Udział ustawowy uprawnionego (jakby nie było testamentu)
2. Substrat = czyste aktywa + darowizny doliczane − długi − koszty pogrzebu
3. Dolicz darowizny:
   - Na rzecz spadkobierców lub uprawnionych do zachowku: BEZ LIMITU czasowego
   - Na rzecz osób trzecich: z ostatnich 10 lat przed śmiercią
   - NIE dolicz: drobnych darowizn, starszych niż 10 lat (od osób trzecich)
   - NIE dolicz: darowizn na cele użyteczności publicznej (art. 994 KC)
4. Zachowek = ½ (lub ⅔) × udział ustawowy × substrat zachowku
```

## DZIAŁ SPADKU
```
Notarialny: przy pełnej zgodzie spadkobierców (szybszy, droższy)
Sądowy: przy sporze → sąd rejonowy
  Opłata: 500 zł (gdy brak zgodnego projektu) / 300 zł (z projektem)
Sąd rozlicza: nakłady na spadek, pożytki, darowizny zaliczone na schedę
```

## PODATEK — TERMIN SD-Z2
```
Grupa 0 (małżonek, dzieci, rodzice, dziadkowie, wnuki, rodzeństwo):
  Zwolnienie od podatku — ALE WYMAGANE ZŁOŻENIE SD-Z2 w 6 m-cy
  od uprawomocnienia postanowienia SNS / sporządzenia APD
⚠️ Brak zgłoszenia w terminie = podatek naliczony wg GRUPY I (bez zwolnienia)
Grupy I, II, III: progresywne stawki podatkowe — weryfikuj progi w aktualnej ustawie
```

## FAZA 0 — INTAKE
```
□ Data śmierci spadkodawcy: [...]
□ 6 miesięcy na oświadczenie upływa: [oblicz natychmiast]
  ⚠️ Wystarczy wniosek do sądu złożony w terminie — nie trzeba samego oświadczenia!
□ Czy jest testament? → sprawdź NRSD u dowolnego notariusza
□ Czy są długi spadkowe? → BIK, ekw.ms.gov.pl (KW), komornik, ZUS, US
□ Czy wśród spadkobierców są małoletni? → procedura sądowa (zawieszenie terminu)
□ Ilu spadkobierców? → zgodnie (APD) czy spór (SNS)?
□ Czy uprawniony do zachowku złożył żądanie? → termin 5 lat
□ Kiedy nastąpiło otwarcie spadku: przed czy po 15.11.2023?
  (III krąg, niegodność, art. 1015 — inne zasady dla sprzed nowelizacji)
```

## PREDYKCJA WYNIKU — SZABLON
```
Szanse na wygraną: [0–100%]
IN PLUS: testament ważny formalnie, brak długów, wszyscy zgodni
IN MINUS: testament kwestionowany, ukryte darowizny, długi > majątek,
          upływ terminu 6 m-cy (nieuzasadniony), brak SD-Z2
BENCHMARKING: → wywołaj orzeczenia-sadowe-v2
  (⚠️ sygnatury tylko po weryfikacji online — nigdy z pamięci)
REKOMENDACJA: □ Notariusz (APD)  □ SNS — sąd  □ Ugoda w sprawie zachowku
```
