# Prawo Karne — Szczegółowy Framework

## PRAWA ZATRZYMANEGO — ABSOLUTNY PRIORYTET
```
OD CHWILI ZATRZYMANIA — ŻĄDAJ NATYCHMIAST:
  □ Prawo do kontaktu z adwokatem (art. 245 §1 KPK) — ZANIM zeznasz
  □ Prawo do informacji o przyczynie zatrzymania (art. 244 §2 KPK)
  □ Prawo do milczenia (art. 175 §1 KPK) — na każde pytanie
  □ Prawo do powiadomienia osoby bliskiej lub pracodawcy (art. 245 §2 KPK)
  □ Prawo do tłumacza (art. 72 KPK) — gdy nie władasz językiem polskim
  □ Prawo do lekarza (gdy stan zdrowia wymaga)

CZEGO NIE ROBIĆ:
  → NIE składaj wyjaśnień bez adwokata — nawet „niewinne" słowa szkodzą
  → NIE podpisuj protokołów, których treści nie rozumiesz
  → NIE wyrażaj zgody na przeszukanie bez nakazu (chyba że i tak nastąpi)
```

## FAZA 0 — INTAKE
```
□ Etap postępowania: zatrzymanie / podejrzany / oskarżony / sąd / wyrok?
□ Zarzut: jaki przepis KK? → sprawdź zagrożenie karą + tryb ścigania
□ Tryb ścigania: z urzędu / na wniosek / z oskarżenia prywatnego?
□ Czy złożono już wyjaśnienia? → co powiedziano? → możliwość odwołania
□ Czy są dowody rzeczowe / nagrania / świadkowie?
□ Czy możliwe warunkowe umorzenie / mediacja karna?
□ Termin przedawnienia: sprawdź art. 101 KK natychmiast
```

## ETAPY POSTĘPOWANIA KARNEGO
```
POSTĘPOWANIE PRZYGOTOWAWCZE:
  Dochodzenie (Policja) lub Śledztwo (Prokuratura)
      ↓
  Akt oskarżenia → Sąd Rejonowy / Okręgowy
      ↓
POSTĘPOWANIE SĄDOWE:
  Rozprawa główna
      ↓ [14 dni od wyroku] APELACJA → Sąd Okręgowy / Apelacyjny
      ↓ [30 dni od SA] KASACJA do SN (przymus adwokacki)
```

## TRYBY ŚCIGANIA
| Tryb | Przykłady | Jak wszcząć |
|---|---|---|
| Z urzędu | Zabójstwo, rozbój, przemoc domowa | Policja/Prokuratura działa sama |
| Na wniosek | Stalking, gwałt, zniszczenie mienia | Wniosek pokrzywdzonego |
| Prywatnoskargowy | Zniesławienie, naruszenie NTN | Prywatny akt oskarżenia |

## WARUNKOWE UMORZENIE (art. 66–68 KK) — STRATEGIA OBRONY
```
WARUNKI ŁĄCZNIE (wszystkie muszą być spełnione):
  □ Wina i społeczna szkodliwość czynu nieznaczne
  □ Sprawca niekarany za przestępstwo umyślne
  □ Postawa, właściwości i warunki osobiste sprawcy rokują przestrzeganie prawa
  □ Zagrożenie do 5 lat pozbawienia wolności

OKRES PRÓBY: 1–3 lata
OBOWIĄZKI MOGĄ OBEJMOWAĆ: naprawienie szkody, zakaz prowadzenia pojazdów,
  zakaz kontaktu z pokrzywdzonym, świadczenie pieniężne na cel społeczny

STRATEGIA:
  → Napraw szkodę przed wnioskiem (zwiększa szanse)
  → Wyraź skruchę (protokół) + zaświadczenia o niekaralności
  → Wniosek do prokuratora (etap przygotowawczy) lub do sądu
```

## MEDIACJA KARNA (art. 23a KPK)
```
KIEDY: Na każdym etapie postępowania (prokuratura + sąd)
KTO: Mediator wpisany na listę Prezesa SO
SKUTKI:
  → Porozumienie z pokrzywdzonym → podstawa warunkowego umorzenia
  → Naprawienie szkody → nadzwyczajne złagodzenie kary
  → Możliwe umorzenie postępowania (za zgodą prokuratora)
WNIOSEK: pisemny do prokuratora / sądu — złóż jak najwcześniej
```

## PRZEDAWNIENIE KARALNOŚCI (art. 101 KK) — Dz.U.2025.383
```
Zbrodnia zabójstwa:                40 lat od popełnienia  ⚠️ (nie 30!)
Inna zbrodnia (≥3 lat PW):        20 lat
Występek zagroż. > 5 lat PW:      15 lat                 ⚠️ (dodany próg)
Występek zagroż. > 3 lat PW:      10 lat
Pozostałe występki:                 5 lat
Prywatnoskargowe (§2):             1 rok od wiedzy / max 3 lata od czynu
Wykroczenia:                        1 rok (2 lata gdy wszczęto)

Przedłużenie po wszczęciu (art. 102 KK): +10 lat dla przestępstw z §1 / +5 lat dla pozostałych
```

## ART. 178a KK — JAZDA PO ALKOHOLU / NARKOTYKACH — SZCZEGÓŁY (nowelizacja 14.03.2024)
⚠️ Weryfikuj każdorazowo w isap.sejm.gov.pl (Dz.U.2025.383 t.j.)

### §1 — Typ podstawowy (pierwsza jazda w stanie nietrzeźwości >0,5‰ lub pod wpływem narkotyków)
- Kara: grzywna, ograniczenie wolności albo PW do 3 lat
- Zakaz prowadzenia pojazdów: obowiązkowy, min. 3 lata
- Świadczenie na Fundusz Pomocy Pokrzywdzonym: min. 5 000 zł (sąd może do 60 000 zł)
- Przepadek pojazdu: OBOWIĄZKOWY gdy stężenie ≥ 1,5‰ (0,75 mg/dm³ powietrza)
  → Wyjątek: sąd może odstąpić od przepadku w szczególnie uzasadnionych okolicznościach

### §4 — Typ kwalifikowany (RECYDYWA — skazany uprzednio za art. 178a lub art. 173/174/177 KK w stanie nietrz.)
- Kara: 3 miesiące – 5 lat PW (OBLIGATORYJNIE pozbawienie wolności)
- Świadczenie na Fundusz: min. 10 000 zł
- Przepadek pojazdu i zakaz prowadzenia — jak w §1 lub surowiej

### STRATEGIA OBRONY art. 178a
```
□ Weryfikuj stężenie alkoholu — błąd pomiaru (urządzenie, kalibracja, choroba)
□ Sprawdź czy to droga publiczna / strefa ruchu / teren zabudowany
□ Warunkowe umorzenie: MOŻLIWE przy §1, niekaralność, niska szkodliwość
  → wymaga: wina i szkodliwość nieznaczne + zagrożenie ≤ 5 lat ✓ + brak skazania za umyślne
□ Blokada alkoholowa (art. 182a KKW) — wniosek po ½ zakazu prowadzenia
□ Przepadek pojazdu: skarżyć wartość, żądać wyceny rynkowej
```

### GRANICA WYKROCZENIE / PRZESTĘPSTWO (art. 87 KW vs art. 178a KK)
```
0,2–0,5‰ we krwi (lub 0,1–0,25 mg/dm³ w powietrzu) → art. 87 §1 KW (WYKROCZENIE)
  Kara: do 30 000 zł grzywny + zakaz prowadzenia 6 m-cy – 3 lata
> 0,5‰ we krwi (lub > 0,25 mg/dm³ w powietrzu) → art. 178a §1 KK (PRZESTĘPSTWO)
  Kara: grzywna / ograniczenie wolności / do 3 lat PW + min. 3 lata zakazu
```

## ART. 190 KK — GROŹBA KARALNA — SZCZEGÓŁY
⚠️ Weryfikuj w isap.sejm.gov.pl (Dz.U.2025.383 t.j.)

### §1 — Znamiona
- Grożenie popełnieniem PRZESTĘPSTWA (nie wykroczenia!) na szkodę pokrzywdzonego lub osoby mu najbliższej
- Groźba WZBUDZA uzasadnioną obawę spełnienia (ocena obiektywna — test przeciętnej osoby)
- Forma dowolna: słowna, pisemna, przez gest, przez osobę trzecią
- Kara: do 3 lat PW (po nowelizacji z 07.07.2022, obowiązuje od 01.10.2023)
  ⚠️ UWAGA: do 01.10.2023 zagrożenie wynosiło 2 lata — stosuj prawo względniejsze (art. 4 §1 KK)

### §2 — Tryb ścigania: NA WNIOSEK pokrzywdzonego
- Brak wniosku = Policja nie wszczyna postępowania z urzędu
- Wyjątek: prokurator może wszcząć bez wniosku gdy: duże prawdopodobieństwo że brak wniosku wynika z obawy przed odwetem LUB przemawia interes społeczny
- Wycofanie wniosku: za zgodą prokuratora (etap przygotowawczy) lub sądu (do zamknięcia przewodu)

## 5 LINII OBRONY OSKARŻONEGO
```
1. Brak znamion: czyn nie wyczerpuje definicji przestępstwa
2. Kontratypy: obrona konieczna (art. 25 KK), stan wyższej konieczności (art. 26)
3. Niepoczytalność / poczytalność ograniczona (art. 31 KK) → biegły psychiatra
4. Błąd co do faktu (art. 28 KK) lub bezprawności (art. 30 KK)
5. Atypowy przebieg przyczynowości → brak związku z wynikiem
```

## ZABEZPIECZENIE DOWODÓW — STRONA POKRZYWDZONA
```
□ Zawiadomienie na piśmie (zalecane) — z wnioskiem o ściganie gdy wymagany
□ Zdjęcia obrażeń ciała niezwłocznie + dokumentacja medyczna
□ Zabezpieczenie nagrań z kamer (72h — potem nadpisane!)
□ Screenshoty wiadomości / maili z datą i nadawcą
□ Lista świadków z danymi kontaktowymi
□ Dziennik zdarzeń (data, opis, reakcja)
```
