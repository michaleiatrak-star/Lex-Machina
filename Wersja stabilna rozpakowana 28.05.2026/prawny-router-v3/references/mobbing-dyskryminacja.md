# Mobbing i Dyskryminacja — Szczegółowy Framework

## ROZRÓŻNIENIE CZYNÓW — KLUCZOWE

| Cecha | Mobbing (94³ KP) | Dyskryminacja (18³a KP) | Molestowanie |
|---|---|---|---|
| Cecha chroniona | NIE | TAK | TAK |
| Powtarzalność | TAK (obowiązkowa) | NIE | NIE |
| Ciężar dowodu | Na pracowniku | Przesuwa się na pracodawcę | Przesuwa się |
| Min. odszkodowanie | Min. wynagrodzenie | Min. wynagrodzenie | Min. wynagrodzenie |

## PIĘĆ PRZESŁANEK MOBBINGU — WSZYSTKIE ŁĄCZNIE
```
1. Działania/zachowania dotyczące/skierowane PRZECIWKO pracownikowi
2. UPORCZYWE I DŁUGOTRWAŁE nękanie lub zastraszanie
3. Wywołujące ZANIŻONĄ OCENĘ PRZYDATNOŚCI ZAWODOWEJ
4. LUB mające na celu PONIŻENIE / OŚMIESZENIE
5. LUB IZOLOWANIE / ELIMINOWANIE Z ZESPOŁU
```
Brak choćby jednej przesłanki = brak mobbingu.

## CIĘŻAR DOWODU — KLUCZOWA ASYMETRIA
**Mobbing:** Pracownik musi UDOWODNIĆ wszystkie przesłanki (przez cały proces).
**Dyskryminacja:** Pracownik UPRAWDOPODABNIA → ciężar PRZESUWA SIĘ na pracodawcę.
**Molestowanie seksualne:** Pracodawca NIE MOŻE uzasadnić obiektywnie.

## STRATEGIA DOWODOWA PRACOWNIKA
```
□ Dziennik mobbingu (data, godzina, opis, świadkowie, reakcja)
□ Screenshoty e-maili / SMS / komunikatorów
□ Nagrania własnych rozmów (LEGALNE w Polsce)
□ Dokumentacja medyczna (kiedy objawy, powiązanie z pracą)
□ Skarga złożona do pracodawcy + odpowiedź
□ Protokół PIP z kontroli
□ Zeznania świadków (byli pracownicy — często zeznają swobodniej)
```

## STRATEGIA OBRONY PRACODAWCY
```
□ Linia 1: Brak przesłanek (jednostkowe incydenty, uzasadniona krytyka)
□ Linia 2: Wdrożona WPA (Wewnętrzna Polityka Antymobbingowa)
□ Linia 3: Obiektywne przyczyny decyzji (przy dyskryminacji)
□ Linia 4: Brak związku przyczynowego z rozstrojem zdrowia
```

## ROSZCZENIA — TABELA
| Roszczenie | Podstawa | Przesłanki | Min. kwota |
|---|---|---|---|
| Zadośćuczynienie | art. 94³ §3 KP | Mobbing + rozstrój zdrowia | brak |
| Odszkodowanie po rozwiązaniu | art. 94³ §4 KP | Mobbing + rozwiązanie umowy | min. wynagrodzenie |
| Odszkodowanie dyskrym. | art. 18³d KP | Dyskryminacja | min. wynagrodzenie |

## FAZA 0 — INTAKE
```
□ Czy czyn jest powtarzalny i długotrwały? → przesłanka mobbingu
□ Czy związany z cechą chronioną (płeć, wiek, religia...)? → dyskryminacja
□ Czy pracownik poniósł rozstrój zdrowia? → zadośćuczynienie art. 94³ §3
□ Czy pracownik rozwiązał umowę z powodu mobbingu? → odszkodowanie §4
□ Czy pracodawca ma WPA (Wewnętrzną Politykę Antymobbingową)?
□ Data pierwszego zdarzenia: [...] → termin przedawnienia 3 lata (art. 291 KP)
```
