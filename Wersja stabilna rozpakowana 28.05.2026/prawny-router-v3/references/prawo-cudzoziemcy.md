# Prawo Cudzoziemców — Szczegółowy Framework

## TERMINY — SPRAWDŹ JAKO PIERWSZE
```
Odwołanie od decyzji o odmowie zezwolenia na pobyt: 14 dni (art. 129 §2 KPA)
Odwołanie od decyzji o wydaleniu (zobowiązanie do powrotu): 14 dni
Skarga do WSA:                                        30 dni od decyzji II inst.
  Wyjątek: od decyzji konsula / MSZ → WSA BEZ etapu odwoławczego
    (art. 52 §4 PPSA — weryfikuj aktualny stan w isap.sejm.gov.pl Dz.U. 2026 poz. 143)
Wniosek o przedłużenie wizy:                         przed wygaśnięciem wizy
Wniosek o pobyt czasowy (zapobieganie nielegalnemu pobytowi):
  → złożenie w terminie ważności poprzedniego zezwolenia = legalizuje pobyt do decyzji
Azyl / ochrona uzupełniająca: brak terminu (złóż jak najszybciej po przybyciu do PL)
```

## FAZA 0 — INTAKE
```
□ Jaki jest aktualny status pobytu?
  → wiza (jakie: Schengen / krajowa D) / karta pobytu / stempel z granicy
  → data ważności — CZY TERMIN JUŻ UPŁYNĄŁ?
□ Jaki jest problem / cel?
  → odmowa / wydalenie / praca / nauka / łączenie rodzin / obywatelstwo / azyl
□ Jaka jest podstawa pobytu?
  → praca / nauka / małżeństwo z obywatelem PL/UE / repatriacja / ochrona
□ Czy jest decyzja i jaka data doręczenia? (termin odwołania!)
□ Czy złożono już wniosek i jaki jest etap?
□ Pilność: czy grozi wydalenie lub termin wydalenia?
□ Jaka narodowość? (UE → inne zasady; kraje umów dwustronnych)
□ Czy cudzoziemiec był już karany w PL lub za granicą?
```

## MAPA TYTUŁÓW POBYTOWYCH
```
A. WIZA SCHENGEN (C): do 90 dni w ciągu 180 dni
   → Nie uprawnia do pracy (z wyjątkiem określonych wiz)
   → Nie można przedłużyć w kraju — tylko wyjazd + nowa wiza

B. WIZA KRAJOWA (D): pobyty długoterminowe (cel: praca, nauka, leczenie, etc.)
   → Do 1 roku; przedłużalna jednokrotnie o ten sam okres
   → Uprawnia do pracy jeśli wskazano cel: praca

C. ZEZWOLENIE NA POBYT CZASOWY (do 3 lat):
   → Praca (art. 114 u.c.) — wymagane zezwolenie na pracę LUB oświadczenie pracodawcy
   → Nauka (art. 144 u.c.) — zaświadczenie z uczelni
   → Łączenie rodzin (art. 159 u.c.) — wymagane potwierdzenie pobytu/obywatelstwa
   → Blue Card (wysoko wykwalifikowani: art. 127a u.c.)
   → Składanie w WUW — termin rozpatrzenia: 30–60 dni (często dłużej w praktyce)
   ⚠️ ZŁOŻENIE WNIOSKU W TERMINIE WAŻNOŚCI POPRZEDNIEGO TYTUŁU = LEGALNY POBYT DO DECYZJI

D. ZEZWOLENIE NA POBYT STAŁY (bezterminowe):
   → 5 lat legalnego pobytu (art. 195 u.c.) — w tym co najmniej 3 lata zezwolenia czas.
   → Małżeństwo z obywatelem PL (art. 195 ust. 1 pkt 2) — wymagane 2–3 lata małżeństwa
   → Karta Polaka (art. 195 ust. 1 pkt 9)
   → Status uchodźcy / ochrona uzupełniająca

E. ZEZWOLENIE NA POBYT REZYDENTA DŁUGOTERMINOWEGO UE:
   → 5 lat legalnego, nieprzerwanego pobytu w PL
   → Uprawnia do pobytu w innych krajach UE
   → Art. 211 i n. u.c.
```

## PRACA CUDZOZIEMCÓW — QUICK CHECK
```
ZEZWOLENIE NA PRACĘ (typ A–E, S):
  → Typ A: ogólne — pracodawca składa wniosek do WUW
  → Test rynku pracy: czy obowiązkowy? (zależy od zawodu — weryfikuj rozporządzenie)
  → Zmiana pracodawcy: wymagane nowe zezwolenie lub aneks (typ A)

OŚWIADCZENIE O POWIERZENIU PRACY (zwolnione z obowiązku zezwolenia):
  → Obywatele: UA, BY, GE, MD, AM, RU (z wyjątkami — weryfikuj aktualnie)
  → Max 24 miesiące w ciągu 36 miesięcy
  → Rejestracja w PUP (nie WUW)
  → Praca wyłącznie u pracodawcy z oświadczenia

JEDNOLITE ZEZWOLENIE (pobyt + praca):
  → Wniosek cudzoziemca do WUW — łączy pobyt i pracę w jednej procedurze
  → Wymagane gdy cudzoziemiec przebywa legalnie w PL

⚠️ WERYFIKUJ AKTUALNY WYKAZ KRAJÓW I ZAWODÓW: gov.pl / praca.gov.pl
```

## WYDALENIE — OBRONA
```
PODSTAWY WYDALENIA (zobowiązanie do powrotu — art. 302 u.c.):
  → Nielegalny pobyt (wygaśnięcie tytułu pobytowego)
  → Stwarzanie zagrożenia bezpieczeństwa / porządku publicznego
  → Skazanie za przestępstwo
  → Fałszywe dokumenty lub dane we wniosku

OCHRONA PRZED WYDALENIEM:
  → Małoletnie dzieci z obywatelstwem PL (art. 304 ust. 1 pkt 3 u.c.)
  → Osoba objęta ochroną uzupełniającą / azylem
  → Długoletni legalny pobyt i silne więzi z PL (orzecznictwo ETPC art. 8 EKPC)
  → Wiek — małoletni cudzoziemiec bez opieki: SZCZEGÓLNA OCHRONA

ŚRODKI OCHRONY TYMCZASOWEJ:
  → Wniosek o wstrzymanie wykonania decyzji (art. 61 §3 PPSA — do WSA)
  → Wniosek do sądu o zastosowanie środka zapobiegawczego (zatrzymanie zamiast wydalenia)
  → ETPC — środek tymczasowy Rule 39 (wyjątkowe: ryzyko naruszenia art. 3 EKPC)
```

## PROCEDURA AZYLOWA — PODSTAWY
```
PODSTAWY OCHRONY (weryfikuj w ustawa o udzielaniu cudzoziemcom ochrony):
  A. NADANIE STATUSU UCHODŹCY (Konwencja Genewska 1951):
     → Prześladowanie ze względu na: rasę, religię, narodowość, poglądy polityczne,
       przynależność do grupy społecznej
     → Wniosek do Szefa UdSC (Urząd do Spraw Cudzoziemców)

  B. OCHRONA UZUPEŁNIAJĄCA:
     → Ryzyko doznania poważnej krzywdy (kara śmierci, tortury, konflikty zbrojne)
     → Brak przesłanek do statusu uchodźcy, ale powrót niemożliwy

  C. OCHRONA CZASOWA (UE):
     → Dyrektywa 2001/55/WE — aktywowana decyzją Rady UE (np. dla Ukrainy 2022–)
     → W PL: ustawa z 12.03.2022 o pomocy obywatelom UA (Dz.U. 2022.583)
     → Weryfikuj aktualny status przedłużeń: gov.pl / UdSC

PRZEBIEG POSTĘPOWANIA:
  Wniosek (SG / Policja lub UdSC) → przesłuchanie → decyzja UdSC (I inst.)
  → [14 dni] odwołanie do Rady do Spraw Uchodźców (organ II inst., tryb pozakodeksowy — NIE KPA!)
  → [30 dni] skarga do WSA → NSA
```

## OBYWATELSTWO POLSKIE — ŚCIEŻKI
```
A. NADANIE PRZEZ PREZYDENTA RP (art. 18 u.o.p.):
   → Uznaniowe — brak przesłanek materialnych poza legalnością pobytu
   → Wniosek przez WUW

B. UZNANIE ZA OBYWATELA POLSKIEGO (art. 30 u.o.p.):
   → Legalny nieprzerwany pobyt 3 lata (stały/rezydent UE) + znajomość języka polskiego
   → 2 lata po ślubie z obywatelem PL (+ pobyt stały lub rezydent)
   → Decyzja Wojewody — nie Prezydenta

C. REPATRIACJA (u.o. repatriacji):
   → Polskie pochodzenie + urodzenie na terenach historycznych (Azja Środkowa)
   → Specjalna procedura; ośrodki repatriacyjne
```

## PREDYKCJA WYNIKU — SZABLON
```
Szanse: [0–100%]
IN PLUS: długoletni legalny pobyt, więzi rodzinne z obywatelami PL, dzieci w PL,
         brak karalności, terminowo złożone wnioski, silna dokumentacja
IN MINUS: nielegalny pobyt, uprzednie wydalenie lub zakaz wjazdu, karalność,
          fałszywe dokumenty, upływ terminów odwołania

BENCHMARKING:
  → Wywołaj orzeczenia-sadowe-v2 z frazą:
    "cudzoziemiec wydalenie art. 8 EKPC więzi rodzinne WSA"
    "odmowa zezwolenia pobyt praca WSA"
  ⚠️ NIEZWERYFIKOWANE — wyszukaj URL: nsa.gov.pl / orzeczenia.ms.gov.pl / etpc.pl

REKOMENDACJA: □ Odwołanie  □ Skarga do WSA  □ Wstrzymanie wydalenia  □ Azyl
```
