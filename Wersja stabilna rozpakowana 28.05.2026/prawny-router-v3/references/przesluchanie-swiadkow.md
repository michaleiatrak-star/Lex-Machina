# Przesłuchanie Świadków — Szczegółowy Framework

## TRZY ŻELAZNE ZASADY
> **Nie pytaj, jeśli nie znasz odpowiedzi** — dotyczy każdego wrogiego świadka.
> **Nie kłóć się ze świadkiem** — kłótnia sygnalizuje słabość, nie siłę.
> **Nie demaskuj od razu** — sprzeczności zostawiaj do mowy końcowej.

## TYPY ŚWIADKÓW — SZYBKA IDENTYFIKACJA
| Typ | Cel | Styl pytań | Ryzyko |
|---|---|---|---|
| Własny – naoczny | Narracja, wiarygodność | Otwarte | Niespójność pod cross |
| Własny – ekspert | Metodologia i pewność | Fokusujące | Atak na metodę |
| Wrogi – naoczny | Ograniczyć szkody, uzyskać przyznanie faktów | Zamknięte (tak/nie) | Dygresje |
| Wrogi – zainteresowany | Ujawnić motyw, stronniczość | Sekwencja + motyw | Kłótnia |
| Biegły przeciwnika | Podważyć metodologię | Fokusujące + ograniczenie | Technikalia |

## 10 TECHNIK PROCESOWYCH
1. **Mały krok** — seria małych przyznań przed faktem kluczowym
2. **Milczenie strategiczne** — pauza 3–5 sek po odpowiedzi
3. **Powrót do punktu wyjścia** — reset narracji bez konfliktu
4. **Lustro (mirroring)** — powtórzenie ostatnich słów jako pytanie (FBI)
5. **Kotwica czasowa** — precyzja daty/godziny destabilizuje kłamstwo
6. **Mapa** — wizualizacja miejsca zdarzenia odsłania luki
7. **Niewinne podsumowanie** — świadek potwierdza Twoje fakty
8. **Ograniczenie możliwości zaprzeczenia** — dokument przed pytaniem
9. **Pytanie o motyw** — ujawnia stronniczość bez ataku
10. **Peak-end rule** — zakończ na szczycie, sąd to zapamięta

## PRZYKŁADOWE SEKWENCJE PYTAŃ

### Świadek własny (ustalenie faktów):
```
→ "Proszę opisać zdarzenie z dnia [data]."
→ "Kto był obecny?"
→ "Co Pan/Pani widział/a i słyszał/a dokładnie?"
→ "Jak Pan/Pani to zrozumiał/a?"
```

### Cross-examination (świadek wrogi):
```
→ "Czy był/a Pan/Pani obecny/a w dniu [data]?" (tak/nie)
→ "Czy widział/a Pan/Pani dokładnie [fakt]?" (tak/nie)
→ "Ile razy rozmawiał/a Pan/Pani z powodem przed dzisiejszą rozprawą?"
→ "Czy uzgadniał/a Pan/Pani treść zeznań z [powodem]?"
```

### Biegły strony przeciwnej:
```
→ "Na jakich danych oparł Pan/Pani swoje wnioski?"
→ "Czy wziął Pan/Pani pod uwagę [alternatywną metodologię]?"
→ "Czy znane są Panu/Pani przypadki gdy ta metoda daje błędne wyniki?"
→ "Jakie jest standardowe odchylenie w tej metodzie pomiaru?"
```

## IMPEACHMENT ŚWIADKA (podważenie wiarygodności)
```
Technika: Dokument PRZED pytaniem o zaprzeczenie
1. Uzyskaj przyznanie że zeznaje prawdziwie
2. Pokaż dokument / wcześniejsze zeznanie
3. Zadaj pytanie o sprzeczność
→ Nie komentuj — pytaj, czy widzi sprzeczność
```

## PODSTAWY PRAWNE
- Zakaz pytań sugerujących własnym świadkom: art. 171 §4 KPK / art. 259 §2 KPC
- Konfrontacja z wcześniejszymi zeznaniami: art. 272 KPC / art. 391 KPK
- Prawo do odmowy odpowiedzi: art. 183 KPK / art. 261 KPC
- Świadek koronny / tajny: odrębne przepisy — weryfikuj KPK

## MOWA KOŃCOWA — CO ZABRAĆ Z PRZESŁUCHANIA
- Przyznania wrogich świadków (mały krok)
- Wyeksponowane sprzeczności
- Ujawnione motywy / stronniczość
- Potwierdzenia własnych dowodów przez świadków neutralnych

## FAZA 0 — INTAKE
```
□ Typ świadka: własny naoczny / własny ekspert / wrogi naoczny / biegły?
□ Cel przesłuchania: narracja faktów / ograniczenie szkód / podważenie?
□ Czy masz wcześniejsze zeznania / oświadczenia świadka? → impeachment
□ Jakie fakty chcesz uzyskać jako przyznane?
□ Jakie fakty świadek może zaprzeczyć? → mały krok przed kluczowym
□ Czas na przygotowanie sekwencji pytań: [min]
```

## PREDYKCJA WYNIKU PRZESŁUCHANIA
```
Siła świadka dla sprawy: [0–100%]
IN PLUS: naoczny, bezstronny, spójna relacja, brak motywu
IN MINUS: zainteresowany, sprzeczność z dokumentami, wcześniej zmienił zeznania
TECHNIKI DO ZASTOSOWANIA: [mały krok / kotwica czasowa / impeachment]
```
