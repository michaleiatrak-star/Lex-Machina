# Prawo Budowlane — Szczegółowy Framework

## TERMINY — SPRAWDŹ JAKO PIERWSZE
```
Odwołanie od decyzji o pozwoleniu na budowę:        14 dni (art. 129 §2 KPA)
Odwołanie od nakazu rozbiórki (PINB):               14 dni
Skarga do WSA:                                      30 dni od decyzji II inst.
Legalizacja samowoli budowlanej (art. 49 PrBud):    na wezwanie PINB — termin wyznaczony
Uproszczona legalizacja (art. 49f PrBud):           gdy budowa zakończona ≥ 20 lat temu
Rękojmia za wady (umowa z wykonawcą — art. 568 KC): 5 lat (nieruchomości)
Rękojmia (umowa deweloperska):                      5 lat od dnia przeniesienia własności
Roszczenia odszkodowawcze (art. 4421 KC):           3 lata od wykrycia szkody / wady
```

## FAZA 0 — INTAKE
```
□ Czego dotyczy sprawa?
  → samowola budowlana / uzyskanie pozwolenia / odwołanie od decyzji PINB
  → spór z wykonawcą (wady, opóźnienie, rozliczenie)
  → warunki zabudowy / MPZP / zmiana przeznaczenia
  → legalizacja lub rozbiórka

□ Jaki obiekt?
  → dom jednorodzinny / budynek wielorodzinny / usługowy / garaż / altana / rozbudowa

□ Jaki etap?
  → planowanie / budowa w toku / wybudowany / użytkowany bez odbioru

□ Czy jest pozwolenie na budowę lub zgłoszenie?
□ Czy obiekt jest już wybudowany / użytkowany?
□ Jaka data decyzji — czy termin odwołania nie minął?
□ Czy są wady wykonawcy? (kiedy ujawnione, zakres, kto potwierdza)
□ [SAMOWOLA] Kiedy zakończono budowę? (kryterium 20 lat dla art. 49f)
□ [SAMOWOLA] Czy przed 19.09.2020 r. wydano nakaz rozbiórki? (blokuje art. 49f)
```

## MAPA PROCEDUR BUDOWLANYCH

### LEGALIZACJA SAMOWOLI
```
TRYB KLASYCZNY (art. 48–49e PrBud) — gdy budowa BEZ pozwolenia:
  PINB wszczyna postępowanie → wstrzymanie / nakaz rozbiórki
  ↓ możliwość legalizacji — PINB wyznacza termin na:
    □ Przedłożenie zaświadczenia o zgodności z MPZP (lub WZ)
    □ Cztery dokumenty legalizacyjne (projekt, oświadczenia, etc.)
    □ Opłata legalizacyjna = 50-krotność opłaty skarbowej za pozwolenie
  Po spełnieniu → decyzja o legalizacji
  Niespełnienie → nakaz rozbiórki

UPROSZCZONA LEGALIZACJA (art. 49f PrBud) — OD 19.09.2020:
  WARUNKI ŁĄCZNIE:
    □ Budowa (lub wybudowanie) zakończona co najmniej 20 lat temu
    □ Brak nakazu rozbiórki wydanego przed 19.09.2020
    □ Brak zagrożenia dla życia, zdrowia, bezpieczeństwa

  PROCEDURA:
    Właściciel składa wniosek do PINB
    PINB żąda: ekspertyzy technicznej + dokumentów potwierdzających datę budowy
    BRAK opłaty legalizacyjnej (!)
    Decyzja o legalizacji lub nakaz rozbiórki (jeśli zagrożenie)

  ⚠️ WERYFIKUJ: aktualny stan art. 49f w ISAP — przepis jest stosunkowo nowy,
  weryfikuj jak PINB stosuje go w praktyce (orzecznictwo WSA)
```

### POZWOLENIE NA BUDOWĘ
```
ORGANY:
  → Starosta / Prezydent (I inst.) — pozwolenie na budowę (art. 35–36a PrBud)
  → Wojewoda (II inst.) — odwołanie
  → WSA → NSA (skarga)

WYMOGI DO POZWOLENIA (weryfikuj w aktualnym PrBud):
  □ Projekt budowlany (projekt zagospodarowania, architektoniczno-budowlany, techniczny)
  □ Oświadczenie o prawie do dysponowania nieruchomością
  □ Decyzja o warunkach zabudowy lub wypis z MPZP
  □ Uzgodnienia (jeśli wymagane — konserwator, środowisko, zarządca drogi)

TERMIN NA WYDANIE:
  → 65 dni (standardowo; art. 35 ust. 6 PrBud)
  ⚠️ Brak ogólnego trybu 45-dniowego — art. 35a PrBud dotyczy elektronicznego dziennika budowy, nie przyspieszenia decyzji. Weryfikuj aktualnie isap.sejm.gov.pl
  → Bezczynność → ponaglenie (art. 37 KPA) → skarga do WSA na bezczynność

ZGŁOSZENIE ZAMIAST POZWOLENIA (art. 30 PrBud):
  → Prostsze obiekty i roboty — lista w art. 29–30 PrBud (weryfikuj aktualnie)
  → Dom jednorodzinny (do 70 m² — uproszczona procedura od 2022 r.): weryfikuj zakres
  → Termin na sprzeciw organu: 21 dni → milczenie = zgoda
```

### SAMODZIELNA FUNKCJA TECHNICZNA — NADZÓR
```
KIEROWNIK BUDOWY: obowiązek na budowach wymagających pozwolenia
INSPEKTOR NADZORU INWESTORSKIEGO: obowiązkowy przy określonych inwestycjach
  → Lista obowiązkowych: rozporządzenie MiT — weryfikuj w isap

DZIENNIK BUDOWY:
  → Obowiązkowy → brak = utrudniony dowód w sporze z wykonawcą
  → Wpisy kierownika budowy, inspektora, organów kontrolnych
```

## SPORY Z WYKONAWCĄ — QUICK CHECK
```
PODSTAWY PRAWNE:
  Art. 647 KC (umowa o roboty budowlane) — gdy przedmiotem jest obiekt budowlany (szeroki zakres); stosuje się niezależnie od tego czy inwestor jest przedsiębiorcą
  Art. 627 KC (umowa o dzieło) — drobne prace, remont

RĘKOJMIA (art. 556–576 KC + art. 568 KC):
  → 5 lat (nieruchomości) / 2 lata (rzeczy ruchome)
  → Liczy się od dnia wydania (odbioru) obiektu
  → Uprawnienia: naprawa / obniżenie ceny / odstąpienie (wada istotna)
  → UWAGA: odbiór z zastrzeżeniami = zachowanie roszczeń z rękojmi

GWARANCJA (art. 577 KC):
  → Termin z umowy (najczęściej 2–5 lat)
  → Niezależna od rękojmi (można korzystać z obu)
  → Wymogi: pisemne zgłoszenie wady w terminie z umowy gwarancyjnej

WSTRZYMANIE WYNAGRODZENIA / KARY UMOWNE:
  → Kary umowne za opóźnienie (art. 483 KC): możliwe miarkowanie przez sąd (art. 484 §2)
  → Solidarna odpowiedzialność inwestora za wynagrodzenie podwykonawcy (art. 647¹ KC)
```

## DOKUMENTY KLUCZOWE W SPORZE Z WYKONAWCĄ
```
□ Umowa o roboty budowlane (lub o dzieło) — pełna treść
□ Protokoły odbioru (etapowe i końcowy)
□ Dziennik budowy
□ Kosztorys i zmiany do niego
□ Dokumentacja fotograficzna z datą (kolejne etapy + ujawnione wady)
□ Korespondencja z wykonawcą (e-mail / SMS / pisemna)
□ Opinie biegłego / rzeczoznawcy budowlanego (prywatna lub sądowa)
□ Faktury i dowody płatności
```

## WARUNKI ZABUDOWY I MPZP
```
MPZP (Miejscowy Plan Zagospodarowania Przestrzennego):
  → Uchwalany przez Radę Gminy
  → Wiążący bezpośrednio — można budować bez WZ jeśli jest MPZP

WZ (Warunki Zabudowy):
  → Gdy brak MPZP → decyzja Wójta/Burmistrza/Prezydenta
  → Wymagania: działka sąsiednia zabudowana, dostęp do drogi, media
  → Odwołanie: SKO → WSA → NSA

ZASADA DOBREGO SĄSIEDZTWA (art. 61 ustawy o planowaniu):
  → WZ wydawane gdy nowa zabudowa kontynuuje cechy zabudowy działek sąsiednich
  → Obszar analizowany: typowo 3-krotność frontu działki
```
