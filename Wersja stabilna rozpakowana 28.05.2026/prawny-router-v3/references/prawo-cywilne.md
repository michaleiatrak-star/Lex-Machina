# Prawo Cywilne — Szczegółowy Framework

## DWA REŻIMY ODPOWIEDZIALNOŚCI — ROZRÓŻNIENIE KRYTYCZNE
```
DELIKT (art. 415 KC): brak umowy, ciężar na powodzie, przedaw. 3/10/20 lat
KONTRAKT (art. 471 KC): istnieje umowa, domniemanie winy dłużnika, 3/6 lat
```

## TRZY PRZESŁANKI ODSZKODOWANIA — ZAWSZE KOMPLET
1. Zdarzenie sprawcze (czyn / naruszenie umowy)
2. Szkoda (damnum emergens + lucrum cessans)
3. Adekwatny związek przyczynowy (art. 361 §1 KC — tylko normalne następstwa)

## SZCZEGÓLNE PODSTAWY ODPOWIEDZIALNOŚCI (zasada ryzyka, bez winy)
- Art. 435 KC: Przedsiębiorstwo wprawiane siłami przyrody
- Art. 436 KC: Posiadacz pojazdu (§1: ryzyko; §2: przy zderzeniu pojazdów — wina)
- Art. 433 KC: Wyrzucenie/wylanie z pomieszczenia
- Art. 430 KC: Zwierzchnik za podwładnego (błąd lekarski, wypadek)

## KARA UMOWNA (art. 483–484 KC)
Tylko przy zobowiązaniach NIEPIENIĘŻNYCH (za pieniężne = odsetki, nie kara).
**Miarkowanie (art. 484 §2):** dwie podstawy:
- Zobowiązanie w znacznej części wykonane
- Kara rażąco wygórowana (stosunek do wartości umowy i szkody)
Żądanie miarkowania = wniosek strony (sąd nie z urzędu).

## BEZPODSTAWNE WZBOGACENIE (art. 405 KC)
Cztery przesłanki: wzbogacenie + zubożenie + związek + brak podstawy prawnej.
Nie wymaga winy ani bezprawności. Granica: aktualne wzbogacenie (art. 409).

## SKARGA PAULIAŃSKA (art. 527 KC)
Cel: ochrona wierzyciela przed uszczupleniem majątku przez dłużnika.
Termin: **5 lat od czynności** (art. 534 KC — biegnie od czynności, nie od dowiedzenia!).
Ułatwienie: darowizna → brak wymogu złej wiary os. trzeciej.

## WŁAŚCIWOŚĆ SĄDU
SR: do 100.000 zł | SO: powyżej.
Właściwość przemienna (wybór powoda): miejsce wykonania zobowiązania / miejsce szkody.

*KC (Dz.U. 2025.1071) + KPC (Dz.U. 2026.468)*

## FAZA 0 — INTAKE
```
□ Czy istnieje umowa między stronami? → kontrakt (471 KC) vs delikt (415 KC)
□ Jaka jest wartość roszczenia? → SR do 100.000 zł / SO powyżej
□ Kiedy powstała szkoda / naruszono umowę? → sprawdź przedawnienie!
□ Trzy przesłanki: zdarzenie + szkoda + związek przyczynowy — wszystkie?
□ Czy dłużnik ma majątek? → sprawdź KW, KRS przed procesem
□ Czy warto wnosić o zabezpieczenie roszczenia?
```
