# Prawo Podatkowe — Szczegółowy Framework

## TERMINY ZAWITE — SPRAWDŹ JAKO PIERWSZE
```
Odwołanie od decyzji US/UCS:      14 dni od doręczenia (art. 223 §1 Op)
Skarga do WSA:                    30 dni od decyzji org. odwoł. (art. 53 §1 PPSA)
Skarga kasacyjna NSA:             30 dni od wyroku WSA (art. 177 §1 PPSA)
Korekta deklaracji:               zawieszona po wszczęciu postępowania (art. 81b Op)
Przedawnienie zobowiązania podat.: 5 lat od końca roku cal. (art. 70 §1 Op)
Przedawnienie prawa do korekty:   5 lat od końca roku cal. (art. 81 §1 Op)
Wniosek o interpretację indyw.:   3 miesiące na wydanie (art. 14d Op)
Czynny żal (KKS):                 PRZED wszczęciem postępowania przez organ
```

## FAZA 0 — INTAKE
```
□ Jakiego podatku dotyczy? (PIT / VAT / CIT / SD / PCC / inne / KKS)
□ Którego roku podatkowego / okresu rozliczeniowego?
□ Jaki etap?
  → decyzja US / postępowanie podatkowe / kontrola podatkowa KAS
  → brak decyzji (milczenie organu) / interpretacja / korekta / KKS
□ Data decyzji / doręczenia — czy 14-dniowy termin odwołania nie upłynął?
□ Osoba fizyczna / JDG / spółka / inna jednostka?
□ Kwota sporu / zaległości? (decyduje o sądzie i opłacie)
□ Czy organ powołał GAAR (art. 119a Op) lub klauzulę szczegółową?
□ Czy grozi odpowiedzialność KKS? (art. 54–107 KKS)
□ Czy złożono korektę deklaracji?
□ Czy był pełnomocnik w postępowaniu?
```

## ŚCIEŻKI ODWOŁAWCZE
```
A. POSTĘPOWANIE PODATKOWE (Op)
   Decyzja US / UCS (I instancja)
       ↓ [14 dni] odwołanie do DIAS (przez organ I inst.)
   Dyrektor Izby Administracji Skarbowej (DIAS)
       ↓ [30 dni] skarga do WSA
   WSA
       ↓ [30 dni] skarga kasacyjna NSA
   NSA

B. KONTROLA CELNO-SKARBOWA (KAS)
   Wynik kontroli celno-skarbowej
       ↓ [14 dni] korekta deklaracji  LUB  brak korekty
   Decyzja Naczelnika UCS
       ↓ [14 dni] odwołanie do DIAS
   DIAS → WSA → NSA

C. SAMODZIELNA KOREKTA DEKLARACJI (art. 81 Op)
   → Możliwa PRZED wszczęciem kontroli/postępowania
   → Zawieszona w trakcie — ale możliwa po zakończeniu kontroli
   → Odsetki: standardowe / obniżone do 50% gdy korekta w 6 m-cy od końca roku i zap. ≤ 25% w ciągu 7 dni
```

## MAPA KLUCZOWYCH INSTYTUCJI

### PIT — najczęstsze spory
| Zagadnienie | Uwaga |
|---|---|
| Ryczałt ewidencjonowany | Stawki 2–17% zależnie od PKD — weryfikuj isap |
| IP Box (art. 30ca–30cb) | Wymagana ewidencja kwalifikowanego IP; 5% PIT/CIT |
| Ulga B+R (art. 26e–26f) | Odliczenie kosztów kwalifikowanych od dochodu |
| Ulga dla młodych (art. 21 ust. 1 pkt 148) | Do 26. r.ż.; limity roczne; weryfikuj |
| Najem prywatny | Od 2023: wyłącznie ryczałt 8,5% / 12,5% |
| Odszkodowania i zadośćuczynienia | Zwolnienie art. 21 ust. 1 pkt 3 — zależne od podstawy |

### VAT — najczęstsze spory
| Zagadnienie | Uwaga |
|---|---|
| Odmowa odliczenia VAT | Weryfikuj: dobra wiara, należyta staranność kupującego |
| JPK_VAT błędy | Korekta; sankcja za niekorektę: weryfikuj Op |
| Split payment (MPP) | Obowiązkowy przy fakturach >15 000 zł z zał. 15 ustawy VAT |
| WIS — stawka na towar | Wniosek do KIS; wiążąca; 3 miesiące na wydanie |
| Zwrot VAT | 60 dni (podstawowy), 25 dni (przyspieszony), 15 dni (r-k VAT) |

### CIT — najczęstsze spory
| Zagadnienie | Uwaga |
|---|---|
| Estoński CIT | Opodatkowanie przy dystrybucji zysku; warunki: brak udz. w spółkach etc. |
| Ceny transferowe (TP) | Dokumentacja przy transakcjach z podmiotami powiązanymi |
| Cienka kapitalizacja | Limit: 30% EBITDA lub 3 mln PLN kosztów finansowania dłużnego |
| Podatek minimalny | Weryfikuj zakres po nowelizacjach 2024/2025 w isap |

### GAAR i klauzule antyabuzywne
```
GAAR (art. 119a Op):
  → Organ pomija korzyść podatkową gdy:
    □ Uzyskanie korzyści było głównym lub jednym z głównych celów czynności
    □ Sposób działania był sztuczny (brak uzasadnienia ekonomicznego)
  → Obrona: cel ekonomiczny > cel podatkowy; mała klauzula ogólna (art. 119f Op)
  → Opinia zabezpieczająca — złożona do Szefa KAS, chroni przed GAAR
```

## ODPOWIEDZIALNOŚĆ ZARZĄDU (art. 116 Op)
```
WARUNKI:
  → Spółka ma zaległości podatkowe
  → Egzekucja z majątku spółki okazała się bezskuteczna
  → Zarząd NIE wykazał:
    □ że we właściwym czasie złożono wniosek o upadłość lub wszczęto układ
    □ niezłożenie wniosku nie nastąpiło z winy członka zarządu
    □ wierzyciel nie doznał uszczerbku mimo niezłożenia

OBRONA:
  → Wykazanie, że wniosek o upadłość był złożony w terminie
  → Wykazanie obiektywnej niemożności działania (np. choroba, wyłączenie)
  → Wykazanie, że mienie spółki nie pozwalało na zaspokojenie
```

## KKS — CZYNNY ŻAL I KOREKTA (art. 16 KKS)
```
CZYNNY ŻAL (art. 16 KKS):
  → Złożyć PRZED wszczęciem przez organ postępowania przygotowawczego
  → Zawiadomić organ + uiścić uszczuploną należność wraz z odsetkami
  → Skutek: bezkarność za przestępstwo/wykroczenie skarbowe
  → UWAGA: nieskuteczny gdy organ już wiedział o naruszeniu z innego źródła

KOREKTA DEKLARACJI jako forma czynnego żalu — dopuszczalna gdy:
  → Złożona przed wszczęciem kontroli / postępowania
  → Jednoczesne uiszczenie zaległości
```

## PRZEDAWNIENIE — PRZERWY I ZAWIESZENIA (art. 70 Op)
```
ZASADA: 5 lat od końca roku kalendarzowego, w którym upłynął termin płatności

ZAWIESZENIE (termin nie biegnie):
  → Wniesienie skargi do WSA (art. 70 §6 pkt 1)
  → Wniesienie skargi kasacyjnej (art. 70 §6 pkt 2)
  → Inne przesłanki — weryfikuj aktualny art. 70 §6 w isap.sejm.gov.pl

PRZERWANIE (termin biegnie od nowa):
  → Zastosowanie środka egzekucyjnego o którym podatnik wiedział
  ⚠️ WYROK TK SK 40/12 (2013): zawiadomienie podatnika = warunek skuteczności przerwania
  → Wyszukaj aktualną linię NSA przed powołaniem: orzeczenia-sadowe-v2
```
