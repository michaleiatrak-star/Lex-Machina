## 1. CORE

### Zakres modułu
UOKiK (Urząd Ochrony Konkurencji i Konsumentów), praktyki naruszające zbiorowe interesy konsumentów, decyzje UOKiK i kary, nieuczciwe praktyki rynkowe, dark patterns, ceny referencyjne (Omnibus), platformy cyfrowe (DSA/P2B), klauzule abuzywne (rejestr UOKiK), Rzecznik Konsumentów, Inspekcja Handlowa, skargi i postępowania.

## 2. INTAKE

```
□ Jaki status stron: konsument / przedsiębiorca / prosument* / platforma?
  * prosument = os. fizyczna prowadząca JDG, gdy umowa niezwiązana bezpośrednio z dz. gosp.
□ Jaka data czynności: przed czy po 01.01.2023 (reforma konsumencka)?
□ Naruszenie indywidualne (roszczenie cywilne) czy zbiorowe (UOKiK)?
□ Jaki charakter naruszenia:
  niezgodność towaru / klauzula abuzywna / nieuczciwa praktyka rynkowa / dark pattern /
  cena referencyjna bez najniższej ceny z 30 dni / fałszywe opinie?
□ Dowody: regulamin, screenshoty oferty, korespondencja, potwierdzenie zamówienia,
  historia płatności, reklamacja i odpowiedź
□ Wybór ścieżki: reklamacja / Rzecznik Konsumentów / UOKiK / Inspekcja Handlowa / sąd?
```

---

## 3. PROCEDURA

### Sekwencja analizy

```
K1. Ustal status stron (konsument / przedsiębiorca / prosument / platforma)
K2. Ustal datę czynności — przed czy po 01.01.2023 (reforma niezgodności z umową)
K3. Zidentyfikuj naruszenie: indywidualne roszczenie czy zbiorowy interes konsumentów
K4. Sprawdź podstawę: niezgodność, odstąpienie, klauzula abuzywna, nieuczciwa praktyka
K5. Ustal dowody minimalne (patrz INTAKE)
K6. Wybierz ścieżkę procesową (patrz niżej)
```

| Ścieżka | Organ | Kiedy |

| Powiatowy / Miejski Rzecznik Konsumentów | PMRK | BEZPŁATNA pomoc indywidualna |
| Inspekcja Handlowa | IH | Konkretny sklep, BHP produktu |

## 4. CZERWONE FLAGI — KWALIFIKATOR NARUSZEŃ

| Sygnał | Naruszone prawo | Skutek |
|---|---|---|
| Ukryte koszty przed finalizacją zamówienia | Obowiązek informacyjny ustawy o PK | Klauzula bezskuteczna; żądanie zwrotu |
| Fałszywy licznik promocji (odliczanie) | Nieuczciwa praktyka rynkowa | Decyzja UOKiK; roszczenie cywilne |
| Cena „przed obniżką" bez najniższej ceny z 30 dni | Zasady Omnibus (art. 4a ustawy o PK) | Naruszenie ustawy; kara UOKiK |
| Automatyczne przedłużanie subskrypcji bez jasnej zgody | Abuzywność; praktyka naruszająca zbiorowe interesy | Decyzja UOKiK; klauzula nie wiąże |
| Regulamin wyłącza ustawową odpowiedzialność sprzedawcy | Klauzula abuzywna (art. 385¹ KC) | Klauzula nie wiąże — ex lege |
| Fałszywe opinie konsumentów (kupione recenzje) | Nieuczciwa praktyka rynkowa (Omnibus) | Kara UOKiK; roszczenie cywilne |
| Dark patterns — manipulacyjny interfejs | Sprzeczność z dobrymi obyczajami | Potencjalnie UOKIK / TSUE |

## 5. KLAUZULE ABUZYWNE — QUICK CHECK

> **Podstawa:** art. 385¹–385³ KC (Dz.U. 2025 poz. 1071) | weryfikuj w ISAP.

```
Przesłanki abuzywności (art. 385¹ §1 KC):
  □ Postanowienie nie zostało indywidualnie uzgodnione
  □ Kształtuje prawa/obowiązki konsumenta sprzecznie z dobrymi obyczajami
  □ Rażące naruszenie interesów konsumenta
Skutek: postanowienie nie wiąże ex lege — umowa wiąże w pozostałej części

Szara lista (art. 385³ KC): katalog postanowień domniemanych jako abuzywne
  → weryfikuj aktualny art. 385³ KC w ISAP

Rejestr klauzul niedozwolonych UOKiK:
  → https://rejestr.uokik.gov.pl
```

## 6. OMNIBUS — CENY REFERENCYJNE (od 01.01.2023)

```
Obowiązek: przy promocji podawać najniższą cenę z 30 dni poprzedzających obniżkę
  → art. 4a ustawy o prawach konsumenta (Dz.U. 2023 poz. 2759 t.j.)
  ⚠️ Weryfikuj aktualne brzmienie w ISAP

Naruszenie: kara UOKiK + roszczenie cywilne konsumenta
  Brak najniższej ceny z 30 dni → informacja wprowadzająca w błąd

Platformy online: obowiązek ujawniania parametrów rankingów (P2B/DSA)
  → weryfikuj aktualne rozporządzenia UE stosowane bezpośrednio
```

## 7. DECYZJE UOKIK I KARY

```
UOKiK wszczyna postępowanie w sprawach zbiorowych (nie indywidualnych)
  → Praktyka naruszająca zbiorowe interesy konsumentów
  → Niedozwolone porozumienia / nadużywanie pozycji dominującej

Kary:
  → Do 10% obrotu w roku poprzednim (naruszenia zbiorowych interesów)
  → Do 10 000 EUR dziennie (naruszenia DSA — platformy)
  ⚠️ Aktualne stawki — weryfikuj UOKIK w ISAP

Decyzja UOKiK:
  → Zaskarżalna do SOKiK (Sąd Okręgowy w Warszawie — wydział antymonopolowy)
  → Termin: 1 miesiąc od doręczenia (weryfikuj w ustawie)
```

## 9. STRATEGIA

### Perspektywa konsumenta

1. Dokumentuj naruszenie natychmiast (screenshoty z datą, archiwum www).
2. Złóż reklamację pisemnie — zachowaj dowód doręczenia.
3. Brak odpowiedzi w 14 dniach = reklamacja uznana (art. 7a ustawy o PK — weryfikuj w ISAP).
4. Powiatowy Rzecznik Konsumentów — bezpłatna pomoc; warto użyć przed sądem.
5. UOKiK nie rozstrzyga sporów indywidualnych — tylko zbiorowe.

### Perspektywa przedsiębiorcy

1. Regulamin i wzorce umowne — weryfikuj pod kątem klauzul abuzywnych przed wdrożeniem.
2. Ceny promocyjne — zachowaj i wyświetlaj najniższą cenę z 30 dni (Omnibus).
3. Dark patterns w UX — ryzyko decyzji UOKiK nawet bez skargi indywidualnej.
4. Platforma cyfrowa — sprawdź obowiązki z DSA i P2B.
