# Checklist — sprawa pracownicza

- [ ] Stosunek pracy / forma współpracy.
- [ ] Daty zatrudnienia i rozwiązania.
- [ ] Wynagrodzenie zasadnicze i należności uboczne.
- [ ] Urlop i ekwiwalent.
- [ ] Świadectwo pracy.
- [ ] Dokumentacja pracownicza.
- [ ] Mobbing / dyskryminacja / nierówne traktowanie.
- [ ] Dobra osobiste.
- [ ] Dowody komunikacji.
- [ ] Świadkowie.
- [ ] Terminy procesowe.
- [ ] Ciężar dowodu.
- [ ] Skutki zdrowotne i związek przyczynowy.
- [ ] Wysokość roszczenia i sposób wyliczenia.
