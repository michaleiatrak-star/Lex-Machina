# ROSZCZENIA — konstrukcja roszczeń i podstaw alternatywnych

## 1. Zasada

Każde roszczenie musi mieć osobno: żądanie, podstawę faktyczną, podstawę prawną, dowody, wyliczenie i ocenę przedawnienia.

## 2. Matryca roszczenia

```text
ROSZCZENIE: ...
Żądanie główne: ...
Żądanie ewentualne: ...
Podstawa prawna główna: ...
Podstawa alternatywna: ...
Fakty konieczne: ...
Dowody: ...
Wysokość: ...
Przedawnienie: ...
Ryzyko: ...
```

## 3. Kontrola konstrukcji

- czy roszczenia można kumulować,
- czy nie są wzajemnie sprzeczne,
- czy żądanie ewentualne jest logicznie podporządkowane,
- czy podstawa deliktowa, kontraktowa i pracownicza nie są pomieszane,
- czy wysokość szkody jest wykazana,
- czy żądanie odsetek ma datę początkową.
