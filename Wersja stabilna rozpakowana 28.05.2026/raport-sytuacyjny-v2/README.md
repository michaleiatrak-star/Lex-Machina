# raport-sytuacyjny-v2-min85

Osobna iteracja modernizacyjna modułu raportu sytuacyjnego.

Nie nadpisuje `raport-sytuacyjny-v2-min8` ani wersji bazowej.

## Cel

Podniesienie modułu z poziomu raportowego do poziomu procesowo-analitycznego przez dodanie:

- chronologii źródłowej,
- statusu źródeł,
- mapy ryzyk,
- detekcji sprzeczności i luk,
- rekomendacji procesowych,
- twardych bramek gotowości.
