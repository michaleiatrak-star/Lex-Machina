# CONTRADICTION-AND-GAP-GATE

Raport musi wskazać sprzeczności i luki, gdy materiał na to pozwala.

## Wykrywaj

- sprzeczne daty,
- sprzeczne kwoty,
- sprzeczne wersje zdarzeń,
- brak dowodu doręczenia,
- brak dokumentu źródłowego,
- niejasny termin,
- nieustaloną właściwość sądu/organu,
- niejasny status procesowy.

## Output

Dla każdej sprzeczności podaj: punkt, wersję 1, wersję 2, znaczenie i rekomendację usunięcia sprzeczności.
