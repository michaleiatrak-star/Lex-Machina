# MODUŁ J3 — UMOWY DYSTRYBUCYJNE: FRANCZYZA I AGENCYJNA
## Analizator Umów v2 · Moduł J3

> Wczytaj dla: umowa franczyzy (franczyzodawca/biorca), umowa agencyjna,
> umowa dystrybucyjna, umowa przedstawicielstwa handlowego.

## J.5 UMOWA FRANCZYZY

**Podstawa: KC art. 353¹ (umowa nienazwana)**
```
UWAGA: Brak osobnej regulacji franczyzy w prawie polskim
  Stosuje się KC + dyrektywy UE (jeśli dotyczy) + kodeksy etyki branżowych
  Weryfikuj: Europejski Kodeks Etyki Franczyzy (jeśli franczyzodawca jest jego sygnatariuszem)
```

### Pułapki:

**FR-1 — Terytorialność bez exclusivity (HIGH RISK)**
```
PUŁAPKA: "Franczyzodawca przyznaje Franczyzobiorcy prawo do prowadzenia
  działalności w [mieście]" — BEZ słowa "wyłącznie"
  → Franczyzodawca może otworzyć kolejny punkt w tym samym mieście
  → "Kanibalizm" sprzedaży

REKOMENDACJA:
  „§X. Franczyzodawca przyznaje Franczyzobiorcy WYŁĄCZNE prawo prowadzenia
  działalności franczyzowej w obszarze: [mapa / opis / kod pocztowy].
  W tym obszarze Franczyzodawca nie otworzy własnych punktów ani nie
  udzieli licencji innym podmiotom na czas trwania Umowy."
```

**FR-2 — Opłaty ukryte poza royalties (HIGH RISK)**
```
TYPOWY SCHEMAT UKRYTYCH OPŁAT:
  → Opłata wstępna (wejściowa): jednorazowa, często bezzwrotna
  → Royalties: % obrotu miesięcznie
  → Opłata marketingowa: % obrotu na "wspólny fundusz reklamowy"
  → Obowiązkowe zakupy: produkty/usługi WYŁĄCZNIE od franczyzodawcy lub wskazanych dostawców
  → Opłaty za szkolenia: każdy kurs = oddzielna faktura
  → Opłata za audyt: franczyzodawca audytuje na koszt franczyzobiorcy
  → Kary za naruszenie standardów: uznaniowe

REKOMENDACJA:
  → Żądaj pełnego schedule opłat na etapie Due Diligence
  → Sprawdź: czy obowiązkowi dostawcy mają ceny rynkowe?
  → Negocjuj cap na opłaty marketingowe + audyt sprawozdania z wydatkowania funduszu
```

**FR-3 — Zakaz konkurencji bez odszkodowania po zakończeniu (CRITICAL)**
```
→ Patrz Moduł I (references/zakaz-konkurencji.md) — sekcja I.3 ZK-1 do ZK-8
  W umowach franczyzowych zakaz jest szczególnie dotkliwy:
  → Franczyzobiorca inwestuje kapitał w markę, której potem nie może używać
  → Zakaz może uniemożliwić wykorzystanie nabytego doświadczenia branżowego

PUŁAPKA SPECYFICZNA: "Franczyzobiorca nie może prowadzić działalności
  konkurencyjnej przez 3 lata po rozwiązaniu umowy na obszarze [cały kraj]"
  → Szerokość + czas + brak odszkodowania = potencjalnie art. 5 KC lub art. 58 KC
```

**FR-4 — Rozwiązanie natychmiastowe za naruszenie standardów (HIGH RISK)**
```
PUŁAPKA: "Franczyzodawca może rozwiązać Umowę ze skutkiem natychmiastowym
  w przypadku niespełnienia standardów jakości przez [X] kolejnych kontroli"
  → "Standardy jakości" definiowane przez franczyzodawcę jednostronnie
  → Kontrole przeprowadzane przez pracowników franczyzodawcy
  → Brak niezależnej procedury odwoławczej

REKOMENDACJA:
  „§X. Rozwiązanie Umowy z powodu niespełnienia standardów jest dopuszczalne gdy:
  a) niespełnienie potwierdzone przez [2] niezależne audyty w odstępie [30] dni;
  b) Franczyzobiorca pisemnie wezwany do usunięcia naruszeń z terminem [60] dni;
  c) naruszenia nie usunięte po upływie terminu.
  Franczyzobiorca ma prawo do zewnętrznej niezależnej ekspertyzy kwestionowanego standardu."
```

## J.6 UMOWA AGENCYJNA

**Podstawa: KC art. 758–764⁹ — WERYFIKUJ BEZWZGLĘDNIE W ISAP**
```
SZCZEGÓLNA UWAGA: Przepisy o umowie agencyjnej zawierają liczne normy bezwzględnie
  wiążące (ius cogens) — strony nie mogą ich wyłączyć ani ograniczyć na niekorzyść agenta.
```

### Pułapki:

**AG-1 — Świadczenie wyrównawcze pominięte (CRITICAL)**
```
PRAWO: KC art. 764³ — świadczenie wyrównawcze (del credere):
  Po rozwiązaniu umowy agent może żądać świadczenia wyrównawczego jeśli:
  (1) pozyskał nowych klientów lub rozwinął istniejących
  (2) dający zlecenie czerpie znaczne korzyści z klientów po rozwiązaniu
  (3) wypłata jest słuszna

UWAGA: Art. 764³ §4 — agent TRACI prawo do wyrównawczego jeśli:
  → Sam wypowiedział umowę (wyjątek: wypowiedzenie z winy dającego zlecenie)
  → Przeniósł prawa i obowiązki na inną osobę bez zgody zlecającego

PUŁAPKA: "Agent zrzeka się wszelkich roszczeń z tytułu rozwiązania umowy"
  → Art. 764⁹ KC: zrzeczenie PRZED rozwiązaniem → bezskuteczne!
  → Zrzeczenie jest możliwe dopiero PO rozwiązaniu umowy

REKOMENDACJA DLA AGENTA:
  → Dokumentuj aktywnie pozyskanych klientów (listy, raporty, bazy danych)
  → Przechowuj dane o generowanych obrotach przez min. 3 lata po rozwiązaniu
  → Żądaj wyraźnego zapisu o zachowaniu prawa do świadczenia wyrównawczego
```

**AG-2 — Prowizja del credere bez limitu (HIGH RISK)**
```
PRAWO: KC art. 761⁶ — agent przejmujący ryzyko wypłacalności klienta (del credere)
  musi za to otrzymać dodatkowe wynagrodzenie; ryzyko del credere ograniczone do
  konkretnej transakcji (nie ogólne)

PUŁAPKA: "Agent odpowiada za wypłacalność klientów pozyskanych przez agenta"
  → Bez dodatkowego wynagrodzenia del credere → naruszenie art. 761⁶ KC
  → Agent przejmuje ryzyko kredytowe przedsiębiorcy bez zapłaty
```
