## Zakres
Reguluje swiadczenie uslug droga elektroniczna (e-uslugi) przez dostawcow
majacych siedzibe w Polsce (lub swiadczacych uslugi dla polskich uzytkownikow).

## Definicje i obowiazki informacyjne
```
Usluga droga elektroniczna: Realizowana bez jednoczesnej obecnosci stron przez systemy IT
Obowiazkowe informacje (art. 5): Nazwa, adres, kontakt, NIP, KRS, UODO
Regulamin: Art. 8 - obowiazkowy regulamin uslug elektronicznych
Zgoda na marketing: Art. 10 - zakaz spamu bez wyraznej zgody (opt-in)
```

## Odpowiedzialnosc dostawcy - safe harbour
```
Cache hosting (art. 12):  Brak odpowiedz. za transmisje
Access hosting (art. 13): Brak odpowiedz. za przechowywanie
Hosting (art. 14):        Brak odpowiedz. jesli nie wiedzial o nielegalnosci
                          Odpow. gdy dowiedzial sie + nie usunal promptly (notice-and-take-down)
```

## Relacja z DSA (od 17.02.2024)
```
DSA (Rozp. UE 2022/2065): Zastapil Dyrektywe e-Commerce dla posrednikow internetowych
Zakres DSA: Platformy internetowe, bardzo duze platformy (VLOP), VLOSE
Polska implementacja DSA: Koordynator uslugi cyfrowej (DSC) - KNF lub UKE (w toku)
UUDE: Polska ustawa uzupelniajaca DSA - weryfikuj aktualny stan
```
