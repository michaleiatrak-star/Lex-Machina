# ADVERSARIAL LITIGATION ANALYSIS V9 — v6

## Miejsce w modelu czteroprzebiegowym

Ten silnik jest integralną częścią Przejścia III (sekcja 3A — trzy perspektywy).
Wyniki sekcji A, B, C wchodzą do Raportu Adversarialnego Przejścia III.
Macierz strategiczna stanowi podstawę §10 (Rekomendacje procesowe) w raporcie końcowym.

## Cel

Analiza sprawy z trzech perspektyw jednocześnie — bez mieszania perspektyw.
Kolejność jest obowiązkowa: najpierw B (sąd), potem A (przeciwnik), potem C (własny).
Analiza własna (C) jako ostatnia — po poznaniu pełnego obrazu zewnętrznego.

## Etap A — analiza jako pełnomocnik przeciwnika

Wyszukaj systematycznie:
- najmocniejsze argumenty przeciwko naszej stronie (top 5 z uzasadnieniem)
- najsłabsze dowody naszej strony (hierarchia A/B/C/D z Filtru #5)
- luki chronologiczne (zestawić z Mapą Faktyczną z Przejścia I)
- twierdzenia bez dowodu (każde z osobna — nie zbiorczo)
- przesadzone żądania (wartość vs. podstawa vs. dowód)
- niespójności narracji (zmiana między pismami — V10-4)
- argumenty, które można obrócić przeciwko naszej stronie (V10-5)

## Etap B — analiza jako sąd

Oceń z perspektywy bezstronnego arbitra:
- co jest udowodnione (poziom A lub B z Filtru #5)
- co jest tylko twierdzeniem (poziom D bez dokumentu)
- co jest zbędne dla rozstrzygnięcia (fakty poza przedmiotem sporu)
- co jest decydujące (bez czego wynik byłby inny)
- które argumenty są wiarygodne (spójność wewnętrzna i zewnętrzna)
- które wyglądają na polemikę (emocjonalne zamiast merytoryczne)
- jak sąd oceni wiarygodność stron (V10-6)

## Etap C — analiza jako pełnomocnik własny

Zbuduj:
- linię główną (najsilniejszy argument + najsilniejszy dowód)
- linię pomocniczą (alternatywna podstawa na wypadek oddalenia linii głównej)
- odpowiedź na kontrargumenty zidentyfikowane w Etapie A
- plan dowodowy (jakich dowodów brakuje, jak je pozyskać)
- plan kolejnego pisma (co ująć, czego nie ujawniać — MOD-A + MOD-F)

## Zasada priorytetu ataku

Najpierw uderzaj w elementy DECYDUJĄCE dla rozstrzygnięcia,
nie w elementy najbardziej irytujące emocjonalnie.

Test decydującości: „Czy bez tego elementu wynik sprawy byłby inny?"
TAK → element decydujący → atakuj w pierwszej kolejności
NIE → element pomocniczy → atakuj dopiero po zabezpieczeniu linii głównej
