# Moduł B — Groźba bezprawna / presja ekonomiczna

**Temat:** Kwalifikacja presji prawno-ekonomicznej. **Nie każda presja jest groźbą bezprawną.**

**Uruchom:** gdy porozumienie zostało zawarte pod presją wypowiedzenia lub innej groźby ekonomiczno-prawnej.

## Drzewo decyzyjne

```
KROK 1: Czy działanie stanowiło groźbę wypowiedzenia zgodną z prawem?
  TAK → groźba zgodna z prawem → nie art. 87 KC
        Badaj: zgodność porozumienia z art. 8 KP i rażącą niekorzystność warunków
  NIE  → groźba bezprawna → art. 87 KC, możliwe uchylenie oświadczenia woli
  WĄTPLIWE → Filtr #8 obowiązkowy

KROK 2: Odstęp czasowy między groźbą a podpisaniem?
  Dni → podejrzana presja czasowa
  Tygodnie → trudniej wykazać bezpośredni związek przyczynowy

KROK 3: Dostęp do pomocy prawnej przed podpisaniem?
  NIE → wzmacnia argument o przymusie
  TAK lub rezygnacja z własnej woli → osłabia argument

KROK 4: Zapisy porozumienia — rażąco niekorzystne dla pracownika?
  TAK → argument z art. 8 KP
  NIE → trudno kwestionować porozumienie
```

## Prawidłowe sformułowanie argumentu

Gdy podstawy groźby są wątpliwe:

*Nie:* „Porozumienie nieważne — bezprawna groźba."

*Tak:* „Porozumienie podpisane w warunkach presji ekonomiczno-prawnej, co winno być uwzględnione przy ocenie zgodności z zasadami współżycia społecznego (art. 8 KP) oraz dobrowolności oświadczenia woli pracownika."
