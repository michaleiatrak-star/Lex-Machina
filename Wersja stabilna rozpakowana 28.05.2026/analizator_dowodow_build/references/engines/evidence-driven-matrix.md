# EVIDENCE-DRIVEN MATRIX

## Dowód → Fakt → Norma → Roszczenie

Każdy dowód musi zostać przypisany do:
1. konkretnego faktu,
2. przesłanki prawnej,
3. roszczenia albo zarzutu,
4. przewidywanej reakcji przeciwnika,
5. ryzyka sądowej oceny.

## Klasyfikacja dowodów

- dokument urzędowy,
- dokument prywatny,
- korespondencja,
- nagranie,
- zeznania świadka,
- przesłuchanie strony,
- opinia biegłego,
- wydruk/screenshot,
- dowód pośredni,
- dowód pomocniczy.

## Analiza sprzeczności

Dla każdej sprzeczności:
- kto mówi co,
- kiedy,
- w jakim dokumencie,
- czy sprzeczność dotyczy faktu kluczowego,
- czy wpływa na wiarygodność,
- czy można ją wyjaśnić.
