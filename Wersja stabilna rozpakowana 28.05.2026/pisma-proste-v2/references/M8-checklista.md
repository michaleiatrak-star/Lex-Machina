# M8 — Lista Kontrolna Przed Wydaniem

*Wczytaj ZAWSZE przed oddaniem gotowego pisma.*
*Nie wydawaj pisma jeśli którykolwiek element nie jest spełniony.*

---

## CHECKLISTA FINALNA (pisma proste)

```
□ TERMIN ZAWITY
  Czy nie upłynął?
  Alert terminowy wyświetlony przed treścią pisma?
  Data graniczna obliczona poprawnie?

□ DANE STRON
  Wnioskodawca: imię i nazwisko / firma, adres, PESEL lub NIP?
  Pozwany/dłużnik: imię i nazwisko / firma, adres?
  Dane kompletne lub oznaczone ⬛ [UZUPEŁNIJ]?

□ SYGNATURA AKT
  Wpisana poprawnie? (lub „—" dla nowego postępowania)

□ PODSTAWA PRAWNA
  Każdy przepis zweryfikowany online (isap.sejm.gov.pl)?
  Pełne oznaczenie: ustawa + artykuł + ustęp + punkt?
  Przepis w wersji obowiązującej na datę zdarzenia?

□ OPŁATA SĄDOWA
  Podana z artykułem KSCU?
  Kwota obliczona poprawnie?
  Sposób uiszczenia opisany?
  Jeśli brak opłaty — podana podstawa zwolnienia?

□ ŻĄDANIE
  Jedno żądanie procesowe?
  Jasne, w trybie „Wnoszę o…" / „Żądam…"?
  Brak sformułowań warunkowych?

□ ODPIS
  Czy wymagany odpis dla strony przeciwnej?
  Wymieniony w załącznikach?

□ TYTUŁ PISMA
  Spójny z treścią?
  WERSALIKAMI?

□ UZASADNIENIE
  Obecne (jeśli wymagane)?
  Struktura: stan faktyczny → ocena prawna → konkluzja?
  Brak faktów fikcyjnych (⛔ = błąd krytyczny)?

□ PODPIS
  Miejsce na podpis?
  Dane pełnomocnika (jeśli reprezentuje)?

□ ZAŁĄCZNIKI
  Wymienione i numerowane?
  Liczba egzemplarzy zgodna z wymaganiami?

□ HYBRID-VALIDATION
  Uruchomiony (wczytaj references/HYBRID-VALIDATION.md)?
  Raport braków wyświetlony?
  Licznik ⬛ podany?

□ M-FAKTY
  Weryfikacja faktyczna wykonana?
  Raport M-FAKTY wyświetlony?
  Brak fikcji faktycznej potwierdzone?
```

---

## BŁĘDY KRYTYCZNE — BLOKUJĄ WYDANIE PISMA

```
⛔ Termin zawity upłynął bez wyświetlenia alertu
⛔ Fikcja faktyczna w treści pisma (fakt bez oparcia w źródle)
⛔ Przepis nieobowiązujący lub błędnie oznaczony
⛔ Więcej niż jedno żądanie procesowe (→ eskalacja)
⛔ Brak danych obowiązkowych stron bez oznaczenia ⬛
```

---

## OSTRZEŻENIA — WYŚWIETL, NIE BLOKUJĄ

```
⚠ Opłata nieznana — podaj orientacyjnie z zastrzeżeniem weryfikacji
⚠ Sygnatura niezweryfikowana — wpisz jako ⬛ [UZUPEŁNIJ]
⚠ Data doręczenia nieznana — termin zawity nie może być obliczony
⚠ Przepis podejrzany o nowelizację — weryfikacja w ISAP zalecana
```
