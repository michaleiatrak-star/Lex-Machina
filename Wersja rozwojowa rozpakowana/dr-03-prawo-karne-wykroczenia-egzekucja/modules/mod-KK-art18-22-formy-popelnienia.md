# Formy popełnienia przestępstwa: sprawstwo, współsprawstwo, podżeganie, pomocnictwo (art. 18-22 KK) v1.0.0 (dodany 2026-07-16)

> ⛔ Do 2026-07-16 brak dedykowanego modułu o formach zjawiskowych —
> pojęcia "współsprawstwo"/"pomocnictwo" pojawiały się incydentalnie
> (np. przy art. 258) bez systematycznego wykładu art. 18-22 KK. Ten
> moduł naprawia lukę, na wyraźne żądanie użytkownika ("pomocnictwo
> i współudział").
> ⚠️ Zweryfikowane 2026-07-16 — do potwierdzenia na isap.sejm.gov.pl
> przed użyciem w piśmie (PRAWO-HARDGATE).

## 0. MAPA POJĘĆ (art. 18 KK) — cztery formy SPRAWSTWA + dwie formy NIESPRAWCZE

```
FORMY SPRAWCZE (art. 18 §1 KK) — sprawca odpowiada jak za czyn własny:
  1. SPRAWSTWO POJEDYNCZE — sam wykonuje czyn zabroniony
  2. WSPÓŁSPRAWSTWO — wykonuje czyn WSPÓLNIE I W POROZUMIENIU z inną osobą
  3. SPRAWSTWO KIEROWNICZE — kieruje wykonaniem czynu przez inną osobę
  4. SPRAWSTWO POLECAJĄCE — wykorzystując uzależnienie innej osoby od
     siebie, poleca jej wykonanie czynu zabronionego

FORMY NIESPRAWCZE (art. 18 §2-3 KK) — odpowiedzialność akcesoryjna:
  5. PODŻEGANIE (art. 18 §2) — nakłania inną osobę do popełnienia czynu
  6. POMOCNICTWO (art. 18 §3) — ułatwia popełnienie czynu innej osobie
```

---

## DRZEWO F.1 — WSPÓŁSPRAWSTWO (art. 18 §1 KK, człon 2)

```
START: Czy co najmniej dwie osoby uczestniczyły w realizacji znamion?
│
├─ Test POROZUMIENIA — czy istniało (choćby dorozumiane, konkludentne)
│  porozumienie co do wspólnego wykonania czynu?
│  │
│  ├─ NIE → brak współsprawstwa — każdy odpowiada indywidualnie za to,
│  │        co sam zrealizował (jeśli w ogóle wypełnia znamiona)
│  │
│  └─ TAK → test WSPÓLNEGO WYKONANIA (teoria materialno-obiektywna,
│           dominująca w orzecznictwie SN): czy wkład danej osoby był
│           NA TYLE ISTOTNY, że bez niego czyn nie zostałby popełniony
│           w tym kształcie? (nie wymaga realizacji WSZYSTKICH znamion
│           osobiście — wystarczy "podział ról")
│           │
│           ├─ Wkład istotny (np. jeden trzyma ofiarę, drugi zabiera
│           │  mienie — rozbój) → WSPÓŁSPRAWSTWO — obaj odpowiadają
│           │  za CAŁOŚĆ czynu (art. 280), nie tylko za swój fragment
│           │
│           └─ Wkład marginalny (np. tylko "trzymał straż" bez
│              wcześniejszego porozumienia co do roli) → rozważ
│              POMOCNICTWO zamiast współsprawstwa (DRZEWO F.3)
```

⚠️ **Ekscesy współsprawcy** — jeśli jeden ze współsprawców wykracza poza
uzgodniony zakres (np. uzgodniona kradzież, ale jeden ze sprawców
niespodziewanie używa przemocy) — pozostali odpowiadają TYLKO za
uzgodniony zakres, chyba że eksces był OBJĘTY ICH ŚWIADOMOŚCIĄ jako
możliwy wariant przebiegu zdarzenia (dolus eventualis co do ekscesu).

## DRZEWO F.2 — PODŻEGANIE (art. 18 §2 KK)

```
START: Czy sprawca NAKŁANIAŁ inną osobę do popełnienia czynu zabronionego?
│
├─ Forma nakłaniania — dowolna (namowa, prośba, polecenie niewynikające
│  z zależności służbowej, płatne zlecenie, groźba niewystarczająca do
│  wyłączenia winy nakłanianego)
│
├─ Wymagany ZAMIAR BEZPOŚREDNI podżegacza (chce, aby inna osoba
│  dokonała czynu zabronionego) — ⚠️ SPÓR DOKTRYNALNY co do możliwości
│  podżegania z zamiarem ewentualnym — dominujące stanowisko: TYLKO
│  zamiar bezpośredni (odróżnia od pomocnictwa, które dopuszcza dolus
│  eventualis)
│
├─ Czy nakłaniany DOKONAŁ czynu (chociażby w formie usiłowania)?
│  │
│  ├─ TAK → podżegacz odpowiada wg SWOJEGO zamiaru, niezależnie od tego,
│  │  czy nakłaniany w ogóle wiedział o roli podżegacza (odpowiedzialność
│  │  akcesoryjna ograniczona — art. 20 KK: każdy współdziałający
│  │  odpowiada w granicach swojej UMYŚLNOŚCI/NIEUMYŚLNOŚCI, niezależnie
│  │  od odpowiedzialności pozostałych)
│  │
│  └─ NIE (nakłaniany odmówił / czyn nie doszedł do skutku) →
│     art. 22 §2 KK: sąd MOŻE zastosować nadzwyczajne złagodzenie kary
│     wobec podżegacza (bezskuteczne podżeganie — karalne, ale
│     fakultatywnie łagodniej)
```

---

## DRZEWO F.3 — POMOCNICTWO (art. 18 §3 KK)

```
START: Czy sprawca UŁATWIŁ innej osobie popełnienie czynu zabronionego?
│
├─ Formy pomocnictwa (katalog przykładowy, nie zamknięty):
│  ├─ FIZYCZNE — dostarczenie narzędzia, środka przewozu, informacji
│  ├─ PSYCHICZNE — udzielenie rady, wzmocnienie zamiaru już powziętego
│  │  przez sprawcę (⚠️ różnica od podżegania: pomocnictwo psychiczne
│  │  WZMACNIA już istniejący zamiar, podżeganie GO WYWOŁUJE)
│  └─ PRZEZ ZANIECHANIE — gdy na sprawcy ciążył szczególny prawny
│     obowiązek niedopuszczenia do czynu (rzadkie, wymaga odrębnej
│     podstawy obowiązku gwaranta)
│
├─ Wymagany zamiar — pomocnictwo MOŻLIWE również z zamiarem EWENTUALNYM
│  (art. 18 §3 — "chcąc albo godząc się") — szerszy zakres niż
│  podżeganie
│
└─ Czyn główny musi zostać CHOCIAŻ USIŁOWANY (pomocnictwo do czynu,
   który w ogóle nie doszedł do usiłowania, nie jest karalne — inaczej
   niż bezskuteczne podżeganie, które jest karalne z nadzwyczajnym
   złagodzeniem, patrz DRZEWO F.2)
```

⚠️ **Pomocnictwo a poplecznictwo (art. 239)** — kluczowa granica
czasowa: pomocnictwo z art. 18 §3 musi nastąpić PRZED lub W TRAKCIE
czynu głównego. Pomoc udzielona DOPIERO PO jego dokonaniu (np. pomoc
w ukryciu sprawcy) to poplecznictwo — odrębne przestępstwo, patrz
`mod-KK-art233-244b-przeciwko-wymiarowi-sprawiedliwosci.md` DRZEWO S.3.

## 4. WYMIAR KARY WOBEC PODŻEGACZA I POMOCNIKA (art. 19 KK)

Sąd wymierza karę w granicach zagrożenia przewidzianego za sprawstwo
(art. 19 §1). Fakultatywnie, wobec pomocnika, sąd MOŻE zastosować
nadzwyczajne złagodzenie kary (art. 19 §2) — instytucja niedostępna
wprost wobec podżegacza (poza przypadkiem bezskutecznego podżegania
z art. 22 §2).

| Współsprawstwo | art. 18 §1 (człon 2) | wg typu czynu + porozumienie | — |
| Sprawstwo kierownicze | art. 18 §1 (człon 3) | wg typu czynu | — |
| Sprawstwo polecające | art. 18 §1 (człon 4) | wg typu czynu + zależność ofiary polecenia | — |

| Pomocnictwo | art. 18 §3 | bezpośredni lub ewentualny | wymaga co najmniej usiłowania czynu głównego |
