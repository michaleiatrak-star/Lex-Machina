# mod-ISO-37001-antykorupcja.md — ISO 37001: System Zarządzania Antykorupcyjnego (ABMS)

Status: moduł norm ISO klasy wzorcowej. Stan metodyczny: 2026-06-07.
Normy ISO są standardami dobrowolnymi — nie aktami prawnymi. Weryfikuj ich aktualność
na iso.org lub PKN (pkn.pl). Powiązane akty prawne weryfikuj w isap.sejm.gov.pl.

## 2. Zakres spraw

- Budowanie i wdrażanie systemu zarządzania antykorupcyjnego (ABMS) zgodnego z ISO 37001
- Certyfikacja ISO 37001 — przygotowanie, audyt certyfikacyjny, nadzory, recertyfikacja
- Ocena ryzyka łapówkarstwa (bribery risk assessment) w organizacji
- Due diligence kontrahentów i partnerów biznesowych (third-party due diligence)
- Polityki, procedury i kontrole antykorupcyjne
- Anonimowy kanał zgłoszeń naruszeń (whistleblowing) — element ABMS + ustawa o sygnalistach
- Dochodzenia wewnętrzne (internal investigations) w przypadku podejrzenia korupcji
- Obrona w postępowaniu karnym lub regulacyjnym — ABMS jako evidence of compliance
- Wymagania przetargowe / inwestorskie dotyczące certyfikacji ISO 37001
- Konflikty interesów, prezenty i gościnność (gifts & hospitality)
- Odpowiedzialność podmiotów zbiorowych za korupcję

## 3. Intake — ustal obowiązkowo

1. Jaki jest cel zaangażowania: certyfikacja ISO 37001 / gap analysis / obrona w postępowaniu
   / dochodzenie wewnętrzne / due diligence kontrahenta / szkolenie?
2. Jaki jest sektor i wielkość organizacji? Czy działa transgranicznie (ryzyka międzynarodowe)?
3. Czy organizacja jest już certyfikowana ISO 37001 lub ISO 37301?
4. Czy posiada dział compliance / CCO / Anti-Bribery Compliance Officer?
5. Czy istnieje lub jest wymagany kanał zgłoszeń (ustawa o sygnalistach ≥ 50 pracowników)?
6. Czy toczy się lub grozi postępowanie karne / regulacyjne za korupcję?
7. Czy kontrahent lub partner wymaga certyfikacji ISO 37001 (PZP, przetarg, umowa)?
8. Czy ABMS jest certifikowany przez jednostkę akredytowaną przez PCA?
9. Czy organizacja korzysta z pośredników, agentów, dystrybutorów — obszary wysokiego ryzyka?

## 4. Struktura ISO 37001 — elementy obowiązkowe

### Kontekst organizacji (kl. 4)
```
Analiza ryzyka łapówkarstwa: kontekst wewnętrzny i zewnętrzny
Identyfikacja interesariuszy i ich wymagań
Zakres systemu ABMS
Ocena ryzyka łapówkarstwa (bribery risk assessment):
  → zidentyfikuj przypadki, w których mogłoby dojść do łapówkarstwa
  → oceń prawdopodobieństwo i skutki
  → określ priorytety kontroli
```

### Przywództwo (kl. 5)
```
Zaangażowanie zarządu (tone from the top) — norma wymaga dowodów
Polityka antykorupcyjna — zatwierdzona przez najwyższe kierownictwo
Anti-Bribery Compliance Function (ABCF) — rola i niezależność
Kultura antykorupcyjna — pomiar i monitorowanie
```

### Planowanie (kl. 6)
```
Cele antykorupcyjne i plany działania
Działania adresujące ryzyka (controls)
```

### Wsparcie (kl. 7)
```
Zasoby, kompetencje, szkolenia
Świadomość antykorupcyjna (awareness)
Komunikacja wewnętrzna i zewnętrzna
Dokumentacja systemu ABMS
```

### Działania operacyjne (kl. 8)
```
Due diligence kontrahentów i partnerów biznesowych
Kontrole finansowe (przegląd transakcji, limity zatwierdzeń)
Prezenty, gościnność, darowizny, sponsoring — polityki i rejestry
Konflikty interesów — deklaracje i zarządzanie
Kanał zgłoszeń (whistleblowing) — anonimowy, chroniony
Dochodzenia wewnętrzne — procedura i niezależność
Działania naprawcze i korygujące
```

### Ocena wyników (kl. 9) i Doskonalenie (kl. 10)
```
Monitorowanie, pomiary, audyty wewnętrzne ABMS
Przegląd zarządzania
Niezgodności i działania korygujące
```

## 5. Mapa proceduralna

### Ścieżka — wdrożenie i certyfikacja
```
Gap analysis (ISO 37001 vs. stan obecny) →
→ Ocena ryzyka łapówkarstwa →
→ Opracowanie polityki antykorupcyjnej i procedur →
→ Due diligence kontrahentów (wdrożenie procesu) →
→ Uruchomienie kanału zgłoszeń (+ wymagania ustawy o sygnalistach) →
→ Szkolenia dla pracowników i zarządu →
→ Audyt wewnętrzny ABMS →
→ Wybór jednostki certyfikującej (akredytowanej przez PCA) →
→ Audyt certyfikacyjny (etap 1: dokumentacja; etap 2: wdrożenie) →
→ Certyfikat ISO 37001 (ważność 3 lata + nadzory roczne)
```

### Ścieżka — podejrzenie korupcji / dochodzenie wewnętrzne
```
Zgłoszenie (kanał whistleblowing / bezpośrednie) →
→ Wstępna ocena istotności przez ABCF →
→ Decyzja o wszczęciu dochodzenia wewnętrznego →
→ Dochodzenie (niezależne, poufne, udokumentowane) →
→ Raport do zarządu i rady nadzorczej →
→ [jeśli uzasadnione] Dobrowolne zawiadomienie prokuratury →
→ Działania dyscyplinarne / naprawcze →
→ Aktualizacja oceny ryzyka i kontroli
```

### Ścieżka — postępowanie karne za korupcję
```
Zarzuty → ABMS jako dowód należytej staranności (due diligence defence) →
→ Wykazanie: polityka + szkolenia + kontrole + dochodzenie →
→ Ocena odpowiedzialności podmiotów zbiorowych →
→ Obrona / negocjacja porozumienia (jeśli prawo krajowe dopuszcza)
```

## 6. Warunki skuteczności ABMS jako obrony

Sprawdź:
- czy ABMS był faktycznie wdrożony (nie tylko dokumentacyjny),
- czy certyfikacja jest aktualna (PCA-akredytowana jednostka),
- czy ocena ryzyka obejmowała sektor i geografię sprawy,
- czy pracownicy przeszli szkolenia (dokumentacja),
- czy kanał zgłoszeń był dostępny i funkcjonował,
- czy dochodzenie wewnętrzne było niezależne i udokumentowane,
- czy zarząd był zaangażowany (tone from the top — dokumenty z posiedzeń).

## 7. Matryca dowodowa

| Fakt | Dowód | Źródło | Siła | Luka | Ryzyko |
|---|---|---|---|---|---|
| Polityka antykorupcyjna wdrożona | Polityka zatwierdzona przez zarząd + data | Dokumenty wewnętrzne | Wysoka | Brak podpisu zarządu | CMS papierowy |
| Ocena ryzyka przeprowadzona | Raport bribery risk assessment | Wewnętrzny / zewnętrzny | Wysoka | Niepokrycie sektorów wysokiego ryzyka | Luka w defensie |
| Due diligence kontrahentów | Raporty DD, rejestry weryfikacji | Dział compliance | Średnia | Brak DD dla starszych kontrahentów | Ryzyko retroaktywne |
| Szkolenia przeprowadzone | Rejestry szkoleń + testy | HR / Compliance | Wysoka | Brak szkolenia zarządu | Luka tone-from-top |
| Kanał zgłoszeń funkcjonuje | Procedura + rejestr zgłoszeń | Compliance | Wysoka | Brak anonimowości | Naruszenie ustawy sygnaliści |
| Certyfikat ISO 37001 | Certyfikat + raporty nadzorcze | Jednostka certyfikująca | Wysoka | Certyfikat wygasły | Brak obrony |

## 8. Typowe zarzuty i kontrargumenty

| Zarzut | Kontrargument |
|---|---|
| ABMS to „papierowy compliance" | Wykazać dowody rzeczywistego wdrożenia: szkolenia, incydenty, dochodzenia |
| Certyfikacja nie chroni przed odpowiedzialnością | ABMS nie jest tarczą absolutną — redukuje ryzyko i może wpłynąć na wymiar kary |
| Due diligence niepełny | Wykazać proporcjonalność do ryzyka danego kontrahenta (risk-based approach) |
| Kanał zgłoszeń fasadowy | Wykazać obsłużone zgłoszenia i podjęte działania (zanonimizowane) |
| Brak niezależności ABCF | ABCF raportuje do rady nadzorczej — wykazać dokumentację linii raportowania |

## 9. Strategia

Priorytety:

1. Ustal cel: certyfikacja / gap analysis / obrona / dochodzenie — strategia różna dla każdego,
2. Przy wdrożeniu — zacznij od oceny ryzyka łapówkarstwa (bribery risk assessment),
3. Przy certyfikacji — wybierz jednostkę akredytowaną przez PCA; sprawdź harmonogram audytów,
4. Przy obronie w postępowaniu — zgromadź dowody rzeczywistego wdrożenia, nie tylko dokumentów,
5. Zintegruj z ISO 37301 (CMS) jeśli organizacja buduje szerszy system compliance (IMS),
6. Kanał zgłoszeń musi spełniać jednocześnie wymagania ISO 37001 i ustawy o sygnalistach,
7. Due diligence kontrahentów dokumentuj proporcjonalnie do ryzyka — unikaj podejścia „wszystko albo nic".

## 10. Orzecznictwo i praktyka

Brak orzecznictwa sądowego wprost do ISO 37001. Szukaj w:
- Orzeczeniach sądów karnych dot. łapówkarstwa i odpowiedzialności podmiotów zbiorowych
  (Portal Orzeczeń Sądów Powszechnych, SN — sn.pl),
- Decyzjach CBA i prokuratury dostępnych publicznie,
- Wytycznych OECD dot. zwalczania przekupstwa (oecd.org),
- Praktyce certyfikacyjnej PCA i jednostek certyfikujących.

**NIGDY nie cytuj sygnatur bez uprzedniej weryfikacji w oficjalnej bazie.**

## 11. Ryzyka

| Ryzyko | Opis | Mitygacja |
|---|---|---|
| CMS papierowy bez wdrożenia | Brak obrony w postępowaniu; ryzyko reputacyjne | Dowody rzeczywistego funkcjonowania |
| Niepokrycie pośredników/agentów | Korupcja przez third-party; odpowiedzialność | Due diligence dla wszystkich pośredników |
| Brak niezależności ABCF | Konflikt interesów; zakwestionowanie przez organ | Linia raportowania do RN, nie do zarządu |
| Nieaktualna ocena ryzyka | Nowe obszary ryzyka niepokryte | Przegląd co 12–18 miesięcy lub przy zmianie modelu biznesowego |
| Niespójność ISO 37001 i ustawy sygnalistów | Luki w kanale zgłoszeń | Audyt wspólny obu wymogów |
| Certyfikat wygasły | Brak ochrony w przetargu lub postępowaniu | Kalendarz nadzorów i recertyfikacji |
