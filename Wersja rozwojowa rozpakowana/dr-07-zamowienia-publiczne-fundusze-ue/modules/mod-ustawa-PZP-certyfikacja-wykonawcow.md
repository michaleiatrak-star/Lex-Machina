## 1. CORE

### Zakres
Certyfikacja wykonawców zamówień publicznych jako dobrowolny instrument potwierdzający spełnienie przesłanek udziału w postępowaniach o zamówienie publiczne; podmioty certyfikujące; zakres certyfikatu; skutki wobec zamawiającego; akredytacja PCA.

## 2. CO TO JEST CERTYFIKAT WYKONAWCY

```
CERTYFIKAT = Urzędowe potwierdzenie przez akredytowany podmiot certyfikujący że:
  □ Nie zachodzą wobec wykonawcy obligatoryjne podstawy wykluczenia (art. 108 PZP)
  □ Nie zachodzą wskazane fakultatywne podstawy wykluczenia (art. 109 PZP)
  □ Wykonawca spełnia wskazane w certyfikacie warunki udziału w postępowaniu

CHARAKTER: FAKULTATYWNY (nie obowiązkowy dla wykonawców)

SKUTEK PRAWNY:
  Zamawiający akceptuje certyfikat zamiast:
  □ JEDZ (Jednolity Europejski Dokument Zamówienia)
  □ Dokumentów podmiotowych potwierdzających brak podstaw wykluczenia
  □ Dokumentów podmiotowych potwierdzających spełnienie warunków udziału
    (wyłącznie w zakresie objętym certyfikatem)

WIELOKROTNE UŻYCIE: jeden certyfikat ważny w wielu postępowaniach
```

---

## 3. PROCEDURA CERTYFIKACJI

```
PODMIOTY CERTYFIKUJĄCE:
  → Akredytowane przez PCA (Polskie Centrum Akredytacji)
  → Rejestr: pca.gov.pl / uzp.gov.pl
  → Działają na zasadach komercyjnych (opłaty za certyfikację)

ZAKRES CERTYFIKATU:
  → Określony przez podmiot certyfikujący wg standardu uzgodnionego z UZP
  → Może obejmować: wszystkie obligatoryjne wyłączenia + wybrane fakultatywne
    + wybrane warunki udziału (zdolność techniczna, finansowa)

WAŻNOŚĆ: Określona w certyfikacie — weryfikuj przez podmiot certyfikujący

WYCOFANIE / ZAWIESZENIE CERTYFIKATU:
  → Podmiot certyfikujący może wycofać gdy przesłanki przestały być spełnione
  → Wykonawca zobowiązany do niezwłocznego informowania o zmianach
```

---

## 4. SKUTKI DLA ZAMAWIAJĄCEGO

```
Zamawiający:
  □ Musi akceptować certyfikat w zakresie nim objętym
  □ Nie może żądać ponownego złożenia dokumentów objętych certyfikatem
  □ Może żądać dokumentów nieobjętych certyfikatem (np. warunki spoza zakresu)

Wyjątek: Zamawiający może odstąpić od akceptacji gdy:
  → Certyfikat stracił ważność
  → Zamawiający ma uzasadnione wątpliwości co do aktualności danych
  → Weryfikuj aktualny art. ustawy w ISAP — przepisy szczegółowe
```

---

## 5. QUALITY GATE / OUTPUT

**Quality gate:** Certyfikat w zakresie postępowania? Ważność certyfikatu nie wygasła? Podmiot certyfikujący akredytowany przez PCA?

**Output:** Zakres certyfikatu → skutek wobec zamawiającego → dokumenty uzupełniające → rekomendacja.
