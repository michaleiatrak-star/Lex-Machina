## 2. TRANSPORT DROGOWY

### Akty i organy

```
Ustawa o transporcie drogowym (Dz.U. 2024 poz. 1539 t.j. ze zm. — weryfikuj w ISAP)
Organ kontrolny: ITD (Inspekcja Transportu Drogowego)
Organ licencyjny: starosta (licencja taxi, przewóz osób) / Główny Inspektor Transportu Drogowego

Licencje:
  → Przewóz osób i rzeczy: licencja od starosty lub GITD
  → Transport kabotażowy: licencja wspólnotowa (UE)
  → ADR: zezwolenie na przewóz materiałów niebezpiecznych
```

### Kary ITD (weryfikuj aktualne stawki w ustawie i rozporządzeniach w ISAP)

```
Kara administracyjna: decyzja → odwołanie do GITD (14 dni) → WSA (30 dni)
Typowe naruszenia: czas jazdy/odpoczynku (tachograf), przekroczenie masy, brak dokumentów
Kara pieniężna: do 15 000 PLN (weryfikuj aktualny taryfikator)
Mandat karny: do 5 000 PLN (w trybie wykroczeniowym)
```

## 3. DROGI PUBLICZNE (Dz.U. 2025 poz. 889)

### Kategorie dróg i zarządcy

```
Drogi krajowe:      GDDKiA + autostrady/ekspresowe (zarząd ministerialny)
Drogi województwa:  Zarząd Dróg Wojewódzkich
Drogi powiatowe:    Zarząd Dróg Powiatowych / Starosta
Drogi gminne:       Wójt / Burmistrz / Prezydent
```

⭐ Mechanizm SPECUSTAWY DROGOWEJ (ZRID — Zezwolenie na Realizację
Inwestycji Drogowej, decyzja łącząca pozwolenie na budowę/podział
nieruchomości/wywłaszczenie) — patrz DEDYKOWANY moduł `mod-GDDKiA-
specustawa-drogowa-ZRID.md` (dodany 2026-07-21), NIE duplikowany tutaj.

### Pas drogowy i odpowiedzialność odszkodowawcza

```
Pas drogowy: grunt + droga + pobocze + rowy + urządzenia
Zajęcie pasa: decyzja zarządcy drogi + opłata (zezwolenie indywidualne lub kategoryzowane)
Reklama w pasie: zakaz dla autostrad; ograniczenia dla innych — weryfikuj art. 43 w ISAP

Odpowiedzialność zarządcy za szkodę z nieutrzymania drogi:
  → Art. 20 ustawy o drogach + art. 415 KC / 429 KC
  → Orzecznictwo SN: zarządca odpowiada jak sprawca deliktu
  → Szkoda: przewrót pojazdu na dziurze, poślizgnięcie na lodzie, wpadnięcie w wyrwę
  → Przedawnienie: 3 lata od dowiedzenia się o szkodzie (art. 4421 KC) lub 10 lat od zdarzenia
  → Pozew do sądu cywilnego właściwego dla siedziby zarządcy
```

## 4. SPECUSTAWA DROGOWA — ZRID (Dz.U. 2024 poz. 1641)

### Decyzja ZRID — jedna decyzja zastępuje trzy

```
ZRID (Zezwolenie na Realizację Inwestycji Drogowej) = 
  pozwolenie na budowę + podział nieruchomości + wywłaszczenie (w jednej decyzji)

Organ I instancji:
  → Starosta: drogi powiatowe i gminne
  → Wojewoda: drogi krajowe, ekspresowe, autostrady

Termin: 90 dni od złożenia kompletnego wniosku
Rygor natychmiastowej wykonalności: DECYZJA NADAJE GO Z MOCY USTAWY
  → Odwołanie NIE wstrzymuje budowy!
  → Wniosek o wstrzymanie wykonania: WSA (art. 61 §3 PPSA) — rzadko skuteczny

Zaskarżenie: odwołanie do Wojewody / Ministra Infrastruktury (14 dni) → WSA (30 dni)
```

### Wywłaszczenie i odszkodowanie przy ZRID

```
Wywłaszczenie z mocy prawa: grunty pod drogę stają się własnością Skarbu Państwa / JST
  z dniem uprawomocnienia ZRID

Odszkodowanie:
  → Starosta ustala w terminie 30 dni od ZRID
  → Podstawa: 105% wartości rynkowej nieruchomości (premia za szybkość)
  → Wszczęcie postępowania o wyższe odszkodowanie: sąd cywilny
  → Termin na wniesienie pozwu o wyższe odszkodowanie: weryfikuj przepisy w ISAP

Prawo wejścia na teren: natychmiast po rygorzę natychmiastowej wykonalności
```

## 5. TRANSPORT KOLEJOWY (Dz.U. 2025 poz. 1234)

### Organy i licencje

```
Prezes UTK (Urząd Transportu Kolejowego):
  → Nadzór nad bezpieczeństwem i regulacja rynku kolejowego
  → Certyfikaty bezpieczeństwa dla przewoźników
  → Decyzje w sprawach dostępu do infrastruktury
  → Licencje przewoźnika kolejowego

PLK (PKP Polskie Linie Kolejowe S.A.): Zarządca infrastruktury

Licencja przewoźnika: UTK na podstawie kryteriów wiarygodności i zasobów
Certyfikat bezpieczeństwa:
  Część A (SMS — System Zarządzania Bezpieczeństwem)
  Część B (konkretna sieć/infrastruktura)
```

### Dostęp do infrastruktury

```
Zasada otwartego dostępu: każdy uprawniony przewoźnik ma prawo dostępu do infrastruktury PLK
Harmonogram (rozkład jazdy): negocjowany z PLK (aktualizowany co grudzień)
Opłaty dostępowe (unit charges): stawki zatwierdzone przez UTK
Odmowa dostępu: decyzja PLK → odwołanie do Prezesa UTK → WSA
```

## 6. PRAWO LOTNICZE (Dz.U. 2023 poz. 2110 ze zm.)

### Organy i certyfikacja

```
ULC (Urząd Lotnictwa Cywilnego): nadzór, certyfikacja, licencje, rejestry
PANSA (Polska Agencja Żeglugi Powietrznej): ATM — kontrola ruchu lotniczego
EASA (Agencja UE): certyfikacja typów statków powietrznych, regulacje UE
```

### Drony (UAV/UAS) — regulacje UE i polskie

```
Kategorie unijne (Rozp. UE 2019/945 + 2019/947):
  Otwarta (A1–A3): loty do 120 m, bez specjalnego zezwolenia — warunki operacyjne
  Szczególna (STS): deklaracja / autoryzacja ULC
  Certyfikowana: dla dużych dronów — pełna certyfikacja

Rejestracja: drony > 250 g lub z kamerą → obowiązkowa w ewidencji ULC
Strefy zakazu lotów: lotniska, granice państwa, drogi — mapa ULC/PANSA (app PansaUTM)
Certyfikat operatora (UAVO): wymagany przy komercyjnym lataniu dronem
Naruszenie stref zakazu: mandat + przepadek drona
```

### Prawa pasażera lotniczego

```
Rozporządzenie UE 261/2004 (odszkodowania za opóźnienia/odwołania):
  → Opóźnienie ≥ 3 h lub odwołanie → odszkodowanie 250/400/600 EUR (zależnie od dystansu)
  → Wyłączenie: nadzwyczajne okoliczności (weryfikuj orzecznictwo TSUE)
  → Skarga do ULC lub pozew do sądu cywilnego
```

## 7. ELEKTROMOBILNOŚĆ (Dz.U. 2024 poz. 1634)

### Strefy Czystego Transportu (SCT)

```
Tworzy: Rada gminy (na wniosek wójta/burmistrza/prezydenta)
Wjazd tylko: pojazdy elektryczne, wodorowe, CNG/LNG, hybrydowe (warunki)
Wyłączenia: pojazdy niepełnosprawnych, służby ratunkowe, mieszkańcy na wniosek
Opłaty: gmina może pobierać opłaty za wjazd (weryfikuj przepisy w ISAP)
```

### Infrastruktura ładowania — AFIR

```
Rozp. UE 2023/1804 (AFIR): sieci ładowania co 60 km wzdłuż sieci TEN-T
Obowiązek budynków: biurowce i obiekty handlowe → min. % miejsc z LPE (weryfikuj EPBD)
Operatorzy OC-P: wymagana rejestracja w URE + interoperacyjność
```

## 8. KODEKS MORSKI (Dz.U. 2023 poz. 1523)

```
Armator: właściciel lub użytkownik statku we własnym imieniu i na własny rachunek
Bandera polska: statek wpisany do rejestru okrętowego (Izba Morska)
Konosament (bill of lading): dokument przewozu morskiego — papier wartościowy
Odpowiedzialność armatora: wg Reguł Haga-Wisby (ograniczona do SDR)
Termin dochodzenia: 1 rok od wydania ładunku lub daty przewozu (Reguły Haga)
Konwencje: SOLAS, MARPOL, MLC 2006, UNCLOS — stosowane bezpośrednio
```

---

## 9. MAPA PROCEDURALNA — KTÓRY TRYB

```
Problem → kwalifikacja trybu:
  Kara ITD → decyzja administracyjna → GITD → WSA
  Odmowa licencji transportowej → decyzja → odwołanie → WSA
  Szkoda z dziury w drodze → pozew do sądu cywilnego (art. 415 KC)
  ZRID (budowa drogi) → odwołanie → Wojewoda/Minister → WSA (nie wstrzymuje!)
  Odmowa dostępu do torów → Prezes UTK → WSA
  Dron w zakazanej strefie → mandat ITD lub sprawa karna
  Opóźnienie lotu → skarga do ULC lub pozew (sąd cywilny)
```
