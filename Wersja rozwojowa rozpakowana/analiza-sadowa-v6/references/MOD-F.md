# Moduł F — Szkoda wyrządzona klientowi przez własnego pełnomocnika
**v6:** Uruchamiany po Przejściu IV (Weryfikacja Ostateczna). Wyniki checklisty poniżej
wchodzą do §8 raportu. Każdy punkt checklisty weryfikuj protokołem O2 z WERYFIKACJA-DOWODOW.md
(pełne przeczytanie pisma, nie tylko cytowany fragment).

**Uruchom gdy:** analizujesz pisma procesowe klienta pod kątem błędów jego własnego pełnomocnika.
**Cel:** Nie krytyka adwokata, lecz ochrona klienta i przygotowanie naprawczej strategii procesowej.

## Checklist audytu pism własnego pełnomocnika

```
□ Czy pismo zawiera twierdzenie, które strona nie musiała ujawniać?
  → Kategoria I (przyznanie niekorzystne)

□ Czy powołano przepis prawny bez jego faktycznego zastosowania?
  → Sygnał słabości dowodowej (np. groźba art. 52 KP bez dyscyplinarki)

□ Czy pismo odpowiada na zarzuty, które nie zostały jeszcze postawione?
  → Kategoria III (ujawnienie zbędnych faktów)

□ Czy złożono wniosek dowodowy bez weryfikacji dostępności świadka/dokumentu?
  → Kategoria IV (błąd strategiczny)

□ Czy narracja pisma atakuje osobę zamiast argumentować prawnie?
  → Ryzyko odbioru przez Sąd jako zastępowania argumentów emocjami

□ Czy pismo zawiera sprzeczność z wcześniejszym pismem tej samej strony?
  → Filtr #10 (sprzeczności między-pismowe)

□ Czy pełnomocnik ujawnił informację o progach właściwości sądu?
  → Kategoria IV (dostarcza strategii stronie przeciwnej)
```

## Format raportu audytu własnych pism

```
AUDYT PISM PROCESOWYCH — [strona / pełnomocnik]
Data audytu: [data]

SZKODA [nr]:
  KATEGORIA: I / II / III / IV
  DOKUMENT: [pismo z datą]
  FRAGMENT: [opis problematycznej treści]
  RYZYKO: [co może zrobić strona przeciwna]
  REKOMENDACJA: [jak zneutralizować w kolejnym piśmie]
  PILNOŚĆ: NATYCHMIASTOWA / WYSOKA / STANDARDOWA

PODSUMOWANIE AUDYTU:
  Krytyczne: [n] | Wysokie: [n] | Standardowe: [n]
  Rekomendowana akcja: [co zrobić w pierwszej kolejności]
```
